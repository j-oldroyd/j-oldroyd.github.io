Definition 3.2.2. Unit Tangent Vectors.

Let $\vb{r}(t)$ be a smooth vector function.

A vector function is smooth on an ineterval $I$ if it's continuously differentiable on $I$ and $\vb{r}^\prime\neq\vb{0}$ on $I\text{.}$ The graph would be a space curve with no cusps.

The unit tangent to $\vb{r}(t)$ is the vector function

\begin{equation*} \vb{T}(t) = \frac{\vb{r}^\prime(t)}{\norm{\vb{r}^\prime(t)}}\text{.} \end{equation*}
