Theorem 6.1.1. Properties of matrix multiplication.

Let \(A, B\) and \(C\) be matrices of compatible sizes and let \(c\in\RR\text{.}\) Then
  1. \(A(BC) = (AB)C\text{.}\)
  2. \(A(B+C) = AB + AC\text{.}\)
  3. \((B+C)A = BA + CA\text{.}\)
  4. \(c(AB) = (cA)B = A(cB)\text{.}\)
  5. \((AB)^T = B^T A^T\text{.}\)
