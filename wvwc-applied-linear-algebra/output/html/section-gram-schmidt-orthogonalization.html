    Section 2.4 Gram-Schmidt Orthogonalization

    Given a linearly independent set \(\{\vecm{v}_i\}\text{,}\) we often want to orthogonalize it to make certain computations easier to perform. This can be done (at least theoretically) by using the Gram-Schmidt Algorithm. The algorithm works be successively subtracting out parallel components of vectors and normalizing what remains to produce an orthonormal set. The parallel components are themselves determined by the inner product: given two nonzero vectors \(\vecm{u},\vecm{v}\) in \(\RR^n\text{,}\) we can always write
    \begin{equation*} \vecm{u} = \vecm{v}_{\parallel} + \vecm{v}_{\perp} \end{equation*}
    where
    \begin{align*} \vecm{v}_{\parallel} \amp = \frac{\vecm{u}^{T}\vecm{v}}{\norm{\vecm{v}}^2}\vecm{v} \\ \vecm{v}_{\perp} \amp = \vecm{u} - \vecm{v}_{\parallel} \text{.} \end{align*}
    Given \(k\) vectors \(\{\vecm{v}_1,\ldots, \vecm{v}_k\}\text{,}\) the algorithm will either produce \(k\) nonzero orthogonal vectors or it will terminate before the \(k\)th step and produce the zero vector as one of its outputs. This second option occurs precisely when the original collection is linearly dependent. Hence, the Gram-Schmidt algorithm will always turn a collection of linearly independent vectors into an orthogonal (and therefore linearly independent) collection of the same size. In particular, the algorithm converts bases into ONBs.

    Example 2.4.1. Using the Gram-Schmidt algorithm to construct an ONB.

    Apply the Gram-Schmidt algorithm to the vectors
    \begin{equation*} \vecm{v}_1 = \mqty[1\\-1\\0], \vecm{v}_2 = \mqty[0 \\ 0 \\ 2],\vecm{v}_3 = \mqty[1\\0\\1]\text{.} \end{equation*}
    Solution.
    We start by setting \(\vecm{q}_1 = \frac{\vecm{v}_1}{\norm{\vecm{v}_1}}\text{:}\)
    \begin{equation*} \vecm{q}_1 = \frac{1}{\sqrt{2}}\mqty[1\\-1\\0]\text{.} \end{equation*}
    Now we orthogonalize the set \(\{\vecm{q}_1, \vecm{v}_2\}\) by replacing \(\vecm{v}_2\) with \(\vecm{\tilde{q}}_2=\vecm{v}_2 - (\vecm{q}_1^{T}\vecm{v}_2)\vecm{u}_2\) and then normalizing the result:
    \begin{align*} \vecm{\tilde{q}}_2 = \vecm{v}_2 - (\vecm{q}_1^{T}\vecm{v}_2)\vecm{q}_1 \amp = \mqty[0\\0\\1] \\ \vecm{q}_2 \amp = \mqty[0\\0\\1] \text{.} \end{align*}
    Finally, we orthogonalize the set \(\{\vecm{q}_1, \vecm{q}_2, \vecm{v}_3\}\text{:}\)
    \begin{align*} \vecm{\tilde{q}}_3 = \vecm{v}_3 - (\vecm{q}_1^{T}\vecm{v}_3)\vecm{q}_1 - (\vecm{q}_2^{T}\vecm{v}_3)\vecm{q}_2 \amp = \mqty[\frac{1}{2} \\ \frac{1}{2} \\ 0]\\ \vecm{q}_3 \amp = \frac{1}{\sqrt{2}}\mqty[1 \\ 1 \\ 0] \text{.} \end{align*}
    The resulting set \(\{\vecm{q}_1, \vecm{q}_2, \vecm{q}_3\}\) is an orthonormal set (in fact, an ONB) in \(\RR^3\text{.}\)

    Example 2.4.2. Gram-Schmidt and linear dependence.

    Apply the Gram-Schmidt algorithm to the vectors
    \begin{equation*} \mqty[1 \\ -1 \\ 0], \mqty[0\\0\\1], \mqty[2\\-2\\1], \mqty[1\\0\\2]\text{.} \end{equation*}
    Solution.
    If we apply the Gram-Schmidt process to this collection of vectors we will eventually get \(\vecm{v}_3 = \vecm{0}\text{.}\) This is because the first three vectors in our collection are linearly dependent. At this point the Gram-Schmidt produces no more useful results. If we want to obtain a linearly independent orthonormal set, we will need to remove \(\vecm{b}_3\) from our collection.

    Subsection The modified Gram-Schmidt algorithm.

    Theoretically, the Gram-Schmidt algorithm is guaranteed to produce an orthonormal set as long as the input is a linearly independent set. In practice, however, roundoff errors in computer computations can produce vectors that are no longer orthogonal. To combat this, the modified Gram-Schmidt algorithm (MGS) can be used. The MGS process differs from the typical Gram-Schmidt process as follows: once the vector \(\vecm{q}_j\) is produced in step \(j\) of the algorithm, all subsequent vectors \(\vecm{v}_k, j+1\leq k\leq n\) are replaced with \(\vecm{v}_k - (\vecm{q}_j^{T}\vecm{v}_k)\vecm{q}_j\text{.}\) That is, we continually update the remaining vectors in our collection to be orthogonal to our current collection of orthonormal vectors.

    Example 2.4.3. Using the modified Gram-Schmidt algorithm to construct an ONB.

    Apply the modified Gram-Schmidt algorithm to the vectors
    \begin{equation*} \vecm{v}_1 = \mqty[1\\-1\\0], \vecm{v}_2 = \mqty[0 \\ 0 \\ 2],\vecm{v}_3 = \mqty[1\\0\\1]\text{.} \end{equation*}
    Solution.
    As before, we start by setting \(\vecm{q}_1 = \frac{\vecm{v}_1}{\norm{\vecm{v}_1}}\text{:}\)
    \begin{equation*} \vecm{q}_1 = \frac{1}{\sqrt{2}}\mqty[1\\-1\\0]\text{.} \end{equation*}
    Now we orthogonalize the set \(\{\vecm{q}_1, \vecm{v}_2, \vecm{v}_3\}\) with respect to \(\vecm{q}_1\text{:}\)
    \begin{align*} \vecm{\tilde{q}}_2 = \vecm{v}_2 - (\vecm{q}_1^{T}\vecm{v}_2)\vecm{q}_1 \amp = \mqty[0\\0\\1] \\ \vecm{\tilde{v}}_3 = \vecm{v}_3 - (\vecm{q}_1^{T}\vecm{v}_3)\vecm{q}_2 \amp = \mqty[\frac{1}{2}\\\frac{1}{2}\\1] \text{.} \end{align*}
    This produces \(\vecm{q}_2 = \smqty[0\\0\\1]\) as before, but now \(\vecm{v}_3\) has been replaced with a vector that is also orthogonal to \(\vecm{q}_1\text{.}\) The final step is to orthogonalize the set \(\{\vecm{q}_1, \vecm{q}_2, \vecm{\tilde{v}}_3\}\text{:}\)
    \begin{align*} \vecm{\tilde{q}}_3 = \vecm{\tilde{v}}_3 - (\vecm{q}_2^{T}\vecm{\tilde{v}}_3)\vecm{q}_2 \amp = \mqty[\frac{1}{2} \\ \frac{1}{2} \\ 0]\\ \vecm{q}_3 \amp = \frac{1}{\sqrt{2}}\mqty[1 \\ 1 \\ 0] \text{.} \end{align*}
    Therefore, we have once again obtained an ONB from the MGS algorithm.
