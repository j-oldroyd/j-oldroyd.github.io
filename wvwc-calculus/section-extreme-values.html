Section 10.7 Extreme Values

¶

Subsection 10.7.1 Local Maxima and Minima

¶

One of the most important applications of the derivative in Calculus I is in finding local maxima and minima of functions \(y = f(x)\text{.}\) We'd like to extend this concept to functions of multiple variables, but first we need to define what local maxima and minima actually are in this new setting.

Definition 10.7.1 Local Maxima and Minima

We say that a function \(f(x,y)\) has a local maximum (respectively, local minimum) at \((x_{0},y_{0})\) if \(f(x,y)\leq f(x_{0},y_{0})\) (respectively, \(f(x,y)\geq f(x_{0},y_{0})\)) for all points \((x,y)\) contained in some disk centered at \((x_{0},y_{0})\text{.}\)

Just as we used derivatives in Calculus I to find maxima and minima, we can use partial derivatives for that purpose here.

Theorem 10.7.2 Derivatives at a Local Maximum or Minimum

Suppose that \(f(x,y)\) is differentiable, and has a local maximum or minimum at \((x_{0},y_{0})\text{.}\) Then

\begin{equation*} f_{x}(x_{0},y_{0}) = f_{y}(x_{0},y_{0}) = 0. \end{equation*}

Theorem 10.7.2 does not tell us if a differentiable function \(f(x,y)\) has a local maximum or minimum at \((x_{0},y_{0})\text{,}\) but it does tell us where to look. We call \((x_{0},y_{0})\) a critical point of \(f(x,y)\) if \(f_{x}(x_{0},y_{0}) = f_{y}(x_{0},y_{0}) = 0\) or if these partial derivatives fail to exist at \((x_{0},y_{0})\text{.}\)

Example 10.7.3 Critical points of an exponential

Let \(f(x,y) = xe^{-x^{2} - y^{2}}\text{.}\) Find its critical points.

Solution

Find the critical points by solving \(f_{x} = 0\) and \(f_{y} = 0\text{.}\) Since

\begin{equation*} f_{x} = e^{-x^{2} - y^{2}} -2x^{2}e^{-x^{2} - y^{2}} = e^{-x^{2} - y^{2}}(1 - 2x^{2}) \end{equation*}

then \(f_{x} = 0\) forces \(x = \pm\sqrt{\frac{1}{2}}\text{.}\) Similarly, since

\begin{equation*} f_{y} = -2xye^{-x^{2} - y^{2}} \end{equation*}

then \(f_{y} = 0\) forces \(x = 0\) or \(y = 0\text{.}\) So the only points that satisfy both \(f_{x} = 0\) and \(f_{y} = 0\) are \((\pm\sqrt{\frac{1}{2}}, 0).\) So these must be our critical points.

We found the critical points in Example 10.7.3 without too much trouble, but these are still not guaranteed to be local maxima or minima. To determine if they are, we need to use the second derivative test.

Theorem 10.7.4 Second Derivative Test

Let \(f(x,y)\) be a differentiable function with continuous second partial derivatives, and let \((x_{0},y_{0})\) be a critical point of \(f\text{.}\) Define \(D\) to be

\begin{equation*} D(x,y) = f_{xx}f_{yy} - (f_{xy})^{2} \end{equation*}

Then we have the following possibilities:

  • If \(D(x_{0},y_{0}) > 0 \) and \(f_{xx}(x_{0},y_{0}) > 0\text{,}\) then \((x_{0},y_{0})\) is a local minimum.
  • If \(D(x_{0},y_{0}) > 0 \) and \(f_{xx}(x_{0},y_{0}) < 0\text{,}\) then \((x_{0},y_{0})\) is a local maximum.
  • If \(D(x_{0},y_{0}) < 0 \text{,}\) then \((x_{0},y_{0})\) is a neither a local minimum nor local maximum. In this case we call \((x_{0},y_{0})\) a saddle point.
  • If \(D(x_{0},y_{0}) = 0 \text{,}\) the test is inconclusive.

The quantity \(D\) from Theorem 10.7.4 is actually a determinant:

\begin{equation*} D(x,y) = \left|\begin{array}{cc} f_{xx} & f_{xy} \\ f_{yx} & f_{yy}\end{array}\right| \end{equation*}
Example 10.7.5 Local extrema of an exponential

Classify the critical points of the function from Example 10.7.3.

Solution

We have \(f(x,y) = xe^{-x^{2} - y^{2}}\text{,}\) and the critical points are \((-\sqrt{\frac{1}{2}},0)\) and \((\sqrt{\frac{1}{2}},0)\text{.}\) We need to find \(D(x,y)\text{.}\) Since \(f_{x} = (1 - 2x^{2})e^{-x^{2} - y^{2}}\text{,}\) we get

\begin{align*} f_{xx} & = -4xe^{-x^{2} - y^{2}} - 2x(1 - 2x^{2})e^{-x^{2} - y^{2}} \\ f_{xy} & = -2y(1-2x^{2})e^{-x^{2} - y^{2}} = f_{yx} \end{align*}

and similarly

\begin{equation*} f_{yy} = -2xe^{-x^{2} - y^{2}} + 4xy^{2}e^{-x^{2} - y^{2}} \end{equation*}

At \((-\sqrt{\frac{1}{2}},0)\text{,}\) we have

\begin{align*} D\left(-\sqrt{\frac{1}{2}},0\right) & = 4\sqrt{\frac{1}{2}}e^{-\frac{1}{2}}\cdot2\sqrt{\frac{1}{2}}e^{-\frac{1}{2}}\\ & > 0 \end{align*}

and \(f_{xx}(-\sqrt{\frac{1}{2}},0) >0\) as well. So this is a local minimum. At \((\sqrt{\frac{1}{2}},0)\text{,}\) we still have \(D>0\text{,}\) but now \(f_{xx}<0\text{.}\) So this is a local maximum.

Subsection 10.7.2 Absolute Maxima and Minima

¶

Even if a function \(f(x,y)\) has a local minimum at \((x_{0},y_{0})\text{,}\) there's no guarantee that this is the smallest value the function can take on its domain. A similar statement is also true for local maxima, which leads to the following definition.

Definition 10.7.6 Absolute Maxima and Minima

Let \(f(x,y)\) be a function defined on the set \(D\text{.}\) We call the point \((x_{0},y_{0})\) an absolute minimum (respectively, absolute maximum) if \(f(x,y)\geq f(x_{0},y_{0})\) (respectively, \(f(x,y)\leq f(x_{0},y_{0})\)) for all points \((x,y)\) in \(R\text{.}\)

As a first example, consider \(f(x,y) = \sqrt{1 - x^{2} - y^{2}}\text{,}\) which gives the top half of the unit sphere. Using critical points and the second derivative test, we can show that the origin is a local maximum. However, this method is blind to the absolute minimum values of \(f(x,y)\text{,}\) which occur at the boundary \(x^{2} + y^{2} = 1\text{,}\) where the sphere intersects the \(xy\)-plane. So this means we can't use critical points alone to find absolute maxima and minima, in general. However, the following theorem says that they'll get us most of the way there.

Theorem 10.7.7 Extreme Value Theorem

Let \(f(x,y)\) be a continuous function defined on a closed and bounded set \(D\text{.}\)  1 This means that \(D\) is finite in size and that it includes its boundary. Then \(f\) is guaranteed to have an absolute maximum and absolute minimum. Furthermore, these must occur either at a critical point or on the boundary of \(D\text{.}\)

Example 10.7.8 Absolute values on a triangle

Let \(f(x,y) = x + y - xy\text{.}\) Find the absolute maxima and minima of \(f\) on the triangle with vertices \((0,0), (0,2)\) and \((4,0)\text{.}\)

Solution

We'll start off by finding any critical points. Since \(\grad f = \dotprod{1-y, 1-x}\text{,}\) we see that the only critical point is \((1,1)\text{.}\) Now we need to see what happens to \(f(x,y)\) on the edges of the triangle, and to do that we need to find equations for the edges. Let \(l_{1}\) denote the edge from \((0,0)\) to \((0,2)\text{,}\) \(l_{2}\) the edge from \((0,2)\) to \((4,0)\) and \(l_{3}\) the edge from \((0,0)\) to \((4,0)\text{.}\) Then \(l_{1}\) is given by the equation \(x = 0\text{,}\) \(l_{2}\) is given by \(y = -\frac{1}{2}(x-4)\) and \(l_{3}\) is given by \(y = 0\text{.}\) So on \(l_{1}\) we have \(f(x,y) = f(0,y) = y\text{,}\) on \(l_{2}\) we have

\begin{equation*} f(x,y) = f\left(x,-\frac{1}{2}(x-4)\right) = \frac{1}{2}\left[\left(x-\frac{3}{2}\right)^{2} + \frac{7}{4}\right] \end{equation*}

and on \(l_{3}\) we have \(f(x,y) = f(x,0) = x\text{.}\) We can then set up the following table:

Note that in Example 10.7.8, we don't actually determine if the critical point is a local maximum or minimum. We just compare the value \(f\) reaches at the critical point with its maximum and minimum values on the boundary.

Example 10.7.10 Absolute maxima and minima on a circular sector

Let \(f(x,y) = xy^{2}\) and let

\begin{equation*} D = \{(x,y) : x,y \geq 0, x^{2} + y^{2} \leq 3\} \end{equation*}

Find the absolute maxima and minima of \(f\) on \(D\text{.}\)

Solution

First we'll find the critical point(s). If we compute \(\grad f\text{,}\) we get \(\grad f = \dotprod{y^{2}, 2xy}\text{.}\) This is \(0\) if and only if \(y=0\text{,}\) and if \(y=0\) then this forces \(f(x,y) = f(x,0) = 0\text{.}\)

Now we need to determine the maximum and minimum values of \(f(x,y)\) on the boundary of \(D\text{.}\) Let \(C_{1}\) denote the line segment from \((0,0)\) to \((\sqrt{3},0)\text{,}\) \(C_{2}\) the circular arc from \((\sqrt{3},0)\) to \((0,\sqrt{3})\) and \(C_{3}\) the line segment from \((0,\sqrt{3})\) to \((0,0)\text{.}\) Then on \(C_{1}\) we have \(f(x,y) = 0\text{,}\) on \(C_{2}\) we have

\begin{equation*} f(x,y) = xy^{2} = x(3-x^{2}) \end{equation*}

and on \(C_{3}\) we have \(f(x,y) = 0\text{.}\) So the smallest value that \(f\) takes on the boundary is \(0\text{,}\) and the largest value is \(2\text{.}\) So the absolute maximum is \(2\) and the absolute minimum is \(0\text{.}\)
