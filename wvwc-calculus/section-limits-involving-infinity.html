Section 2.4 Limits Involving Infinity

Subsection 2.4.1 Limits Involving Vertical Asymptotes

Consider the function \(f(x)=\frac{1}{x}\text{.}\) We know from algebra that this function has a vertical asymptote at \(x=0\text{,}\) and so in particular is undefined there. However, just because it's undefined at \(x=0\) doesn't mean that we can't gather important information about the function near \(0\text{.}\) This is because the function behaves in a very specific way as we let \(x\) approach \(0\text{.}\) For example, if we let \(x\) approach \(0\) from the right, then \(f(x)\) increases without bound. Similarly, \(f(x)\) decreases without bound as \(x\) approaches \(0\) from the left.

Even though \(f(x)\) is not approaching a specific value as \(x\) approaches \(0\) from either direction, this behavior shows up often enough and is important enough that we want to introduce notation to describe it. For this function, we would say \(\lim_{x\to0^{+}}\frac{1}{x}=\infty\) and \(\lim_{x\to0^{-}}\frac{1}{x}=-\infty\text{.}\)

Now consider \(g(x) = \frac{1}{(x-3)^{2}}\text{.}\) Then \(\lim_{x\to3^{-}}g(x) = \lim_{x\to3^{+}}g(x) = \infty\) since the function increases without bound when \(x\) approaches \(3\) from both directions. In this case, we say that \(\lim_{x\to3}g(x)=\infty\text{.}\)

It's extremely important to remember that the symbol \(\infty\) is not being used to represent a number or variable that we can perform algebra on. It's a symbol indicating how a particular function is behaving at a certain point.

If \(f(x)\) is a function and \(\lim_{x\to a}f(x)=\pm\infty\text{,}\) \(\lim_{x\to a^{-}}f(x)=\pm\infty\) or \(\lim_{x\to a^{+}}f(x)=\pm\infty\text{,}\) then this means that the function has a vertical asymptote at \(x=a\text{.}\) In this course, this basically corresponds to a divide-by-zero problem.

Example 2.4.1. Infinite limit involving a rational function.

Determine \(\lim_{x\to4^{-}}\frac{x-3}{2x-8}\text{.}\)

Solution.

If we try to plug in \(x=4\) into \(\frac{x-3}{2x-8}\) we get \(\frac{1}{0}\text{,}\) which means we have run into a divide-by-zero problem. This is a good hint that the limit should be \(\pm\infty\text{,}\) we just need to figure out the correct sign. There are a couple ways we can do this. First, we could set up a sign chart for this function to see where it's positive and negative and then use that to see if it's increasing or decreasing without bound as \(x\to4^{-}\text{.}\) Second, we could just plug in values of \(x\) that are closer and closer to \(4\) and see how the function behaves. Either way, we see that it's negative for values of \(x\) that are close to (but less than) \(4\text{.}\) Hence \(\lim_{x\to4^{-}}\frac{x-3}{2x-8}=-\infty\text{.}\)

Subsection 2.4.2 Limits at Infinity

Subsection 2.4.1 involved limits of functions whose values approached \(\pm\infty\text{.}\) Now we look at what can happen to a function if its input values approach \(\pm\infty\text{.}\) First, a definition of sorts.

Definition 2.4.2. Limit at Infinity.

Let \(f(x)\) be a function. We say that \(\lim_{x\to\infty}f(x) = L\) if \(f(x)\) gets (and stays) arbitrarily close to \(L\) as \(x\) is made arbitrarily large. Similarly, we say that \(\lim_{x\to-\infty}f(x) = L\) if \(f(x)\) gets (and stays) arbitrarily close to \(L\) as \(x\) is made arbitrarily small.

Example 2.4.3. An important limit at infinity.

Let \(f(x) = \frac{1}{x}\text{.}\) Determine \(\lim_{x\to\infty}f(x)\text{.}\)

Solution.

As \(x\) gets arbitrarily large, \(\frac{1}{x}\) gets arbitrarily close to \(0\text{.}\) Therefore \(\lim_{x\to\infty}f(x) = 0\text{.}\)

Example 2.4.3 holds for many other reciprocal powers of \(x\text{.}\) In particular, if \(n>0\) then \(\lim_{x\to\infty}\frac{1}{x^{n}} = \lim_{x\to-\infty}\frac{1}{x^{n}} = 0\text{.}\)

Example 2.4.4. A limit at infinity involving cosine.

Let \(g(s) = \cos s^{-3}\text{.}\) Compute \(\lim_{s\to-\infty}g(s)\text{.}\)

Solution.

First, note that \(g(s) = \cos\frac{1}{s^{3}}\text{.}\) By the previous remark, we know that \(\lim_{s\to-\infty}\frac{1}{s^{3}} = 0\text{.}\) Therefore \(\lim_{s\to-\infty}g(s) = \cos0 = 1\text{.}\)

The reason we were able to find the limit in Example 2.4.4 was because of the following fact: if \(\lim_{x\to a}g(x)\) exists and if \(f(x)\) is continuous at \(\lim_{x\to a}g(x)\text{,}\) then \(\lim_{x\to a}f(g(x)) = f(\lim_{x\to a}g(x))\text{.}\) Basically, we can swap continuous functions with limits without causing any harm.

For limits at infinity involving powers of a variable, it is the highest power variables that determine the outcome.

Example 2.4.5. Limit at infinity of a rational function.

Let

\begin{equation*} f(t) = \frac{100t - 3 + t^{3}}{5t^{2}+1}\text{ and } g(t) = \frac{1-2t^{4}}{5t^{4}+3000t}. \end{equation*}

Find \(\lim_{t\to\infty} f(t)\) and \(\lim_{t\to-\infty}g(t)\text{.}\)

Solution.

Let's start with \(f(t)\text{.}\) To figure out what this limit should be, we could try the following. As \(t\) gets very large the \(t^{3}\) term in the numerator should drown out everything else in the numerator. Similarly, the \(5t^{2}\) term in the denominator should drown out everything else in the denominator. So for \(t\) very large, \(f(t)\approx\frac{t^{3}}{5t^{2}} = \frac{1}{5}t\text{.}\) Hence \(f(t)\) should probably go to \(\infty\) as \(t\) goes to \(\infty\text{.}\) To make this more precise, we'll just divide the numerator and denominator by the largest power of the denominator, \(t^{2}\text{,}\) and then take the limit:

\begin{align*} \lim_{t\to\infty}f(t) & = \lim_{t\to\infty}\frac{100t-3+t^{3}}{5t^{2}+1} \\ & = \lim_{t\to\infty}\frac{\frac{100}{t} - \frac{3}{t^{2}}+t}{5+\frac{1}{t^{2}}} \\ & = \infty. \end{align*}

We can find \(\lim_{t\to-\infty}g(t)\) using the same idea. Just divide by the highest power in the denominator and then take the limit:

\begin{align*} \lim_{t\to-\infty}g(t) & = \lim_{t\to-\infty}\frac{\frac{1}{t^{4}}-2}{5+\frac{3000}{t^{3}}} \\ & = -\frac{2}{5}. \end{align*}

These limits at infinity have a graphical meaning as well. If \(\lim_{x\to\infty}f(x)\) or \(\lim_{x\to-\infty}f(x)\) exists and is equal to \(L\text{,}\) then the line \(y=L\) is a horizontal asymptote of the graph of \(y=f(x)\text{.}\)

Example 2.4.6. Asymptotic equivalence.

Two functions \(f(x)\) and \(g(x)\) are said to be asymptotically equivalent, written \(f\sim g\text{,}\) if the following is true:

\begin{equation*} \lim_{x\to\infty}\frac{f(x)}{g(x)} = 1. \end{equation*}

Show that \(\sin\frac{1}{x}\sim\frac{1}{x}\text{.}\)

Solution.

All we need to do is compute \(\lim_{x\to\infty}\frac{\sin\frac{1}{x}}{\frac{1}{x}}:\)

\begin{align*} \lim_{x\to\infty}\frac{\sin\frac{1}{x}}{\frac{1}{x}} & = \lim_{\theta\to0^{+}}\frac{\sin\theta}{\theta} \\ & = 1. \end{align*}

Therefore \(\sin\frac{1}{x}\sim\frac{1}{x}\text{.}\)
