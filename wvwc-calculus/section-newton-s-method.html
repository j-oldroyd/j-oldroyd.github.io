Section 4.5 Newton's Method

¶

Consider a differentiable function \(f(x)\text{,}\) and suppose we want to find a root of \(f(x)\text{,}\) which is a number \(x_{0}\) such that \(f(x_{0}) = 0\text{.}\) In some cases this is very easy (like \(f(x) = x^{2} + 2x + 1\)), but in others this may be more complicated (such as \(f(x) = x^{5} - x - 1\)). So we'd like to find a way to approximate the root \(x_{0}\text{.}\)

Newton's Method begins as follows: pick some starting point, or guess, \(x_{1}\text{.}\) Then draw the tangent line \(L\) to the graph of \(f(x)\) at \(x_{1}\text{.}\) Now, the \(x\)-intercept of this line may not be a root, but hopefully it'll be closer to the root. So call this point \(x_{2}\text{.}\) We can solve for \(x_{2}\) (if \(f'(x_{1})\neq0\)) to get

\begin{equation*} x_{2} = x_{1} - \frac{f(x_{1})}{f'(x_{1})}. \end{equation*}

Now, at this point \(x_{2}\) is (hopefully!) a better approximation to the root \(x_{0}\) than \(x_{1}\text{,}\) and we can run through the same procedure to get a new point \(x_{3}\text{.}\)

In general, we can get a sequence of approximations \(x_{1},x_{2},\ldots,x_{n}\) to the root \(x_{0}\) by using the formula

\begin{equation*} x_{n+1} = x_{n} - \frac{f(x_{n})}{f'(x_{n})}, \end{equation*}

as long as \(f'(x_{n}) \neq0\text{.}\)

Example 4.5.1. Approximating a root.

Find the third approximation given by Newton's Method for the root of \(f(x) = x^{5} - x - 1\text{,}\) using \(x_{1} = -1\text{.}\)

Solution

First, note that \(f'(x) = 5x^{4} - 1\text{.}\) Then

\begin{align*} x_{2} & = x_{1} - \frac{f(x_{1})}{f'(x_{1})} \\ & = -1 - \frac{-1}{4} \\ & = 3, \end{align*}

so \(x_{2} = 3\text{.}\) Now we run through this method again to get \(x_{3}\text{:}\)

\begin{align*} x_{3} & = x_{2} - \frac{f(x_{2})}{f'(x_{2})} \\ & = 3 - \frac{239}{404} \\ & = \frac{973}{404}. \end{align*}

So \(x_{3} = \frac{973}{404}.\)

Example 4.5.2. Approximating \(\pi\).

Approximate \(\pi\) using Newton's method.

Solution

We need to find a function \(f(x)\) for which \(f(\pi) = 0\text{.}\) Perhaps the easiest is \(f(x) = \sin x\text{.}\) Now, to use Newton's method we also need a starting guess. We'll pick \(x_{1} = 3\) since this is close to \(\pi\text{.}\) Then

\begin{equation*} x_{2} = x_{1} - \frac{\sin x_{1}}{\cos x_{1}} = 3 - \tan 3 \end{equation*}

so \(x_{2} \approx 3.142546543\text{.}\) Similarly,

\begin{align*} x_{3} & = x_{2} - \tan x_{2} \approx 3.141592653 \\ x_{4} & \approx 3.141592654 \end{align*}

Since these approximations are so close, we estimate that \(\pi\approx 3.14159265\text{.}\)

Example 4.5.3. A Babylonian problem.

Use Newton's method to find an algorithm for computing \(\sqrt{a}\text{.}\)

Solution

To use Newton's method, we need to come up with a function \(f(x)\) whose root is \(\sqrt{a}\text{.}\) A simple choice for this is \(f(x) = x^{2} - a\text{,}\) since \(f(\sqrt{a}) = 0\text{.}\) Now, if we're getting some sequence \(x_{1},\ldots,x_{n}\) from Newton's method then the next term in the sequence is

\begin{equation*} x_{n+1} = x_{n} - \frac{f(x_{n})}{f'(x_{n})} = x_{n} - \frac{x_{n}^{2} - a}{2x_{n}}. \end{equation*}

After some algebra, we can rewrite this as

\begin{equation*} x_{n+1} = \frac{1}{2}\left(x_{n} + \frac{a}{x_{n}}\right). \end{equation*}

We can actually test this algorithm out. Say we want to approximate \(\sqrt{5}\text{.}\) A reasonable guess would be \(x_{1} = 2\text{,}\) since this should be close to \(\sqrt{5}\text{.}\) Then we get

\begin{align*} x_{2} & = \frac{1}{2}\left(2 + \frac{5}{2}\right) = 2.25 \\ x_{3} & = \frac{1}{2}\left(2.25 + \frac{5}{2.25}\right) \approx 2.236111111 \\ x_{4} & \approx 2.236067978 \\ x_{5} & \approx 2.236067977 \end{align*}

So it looks like \(\sqrt{5}\approx 2.23606797\text{,}\) and indeed this is quite close to the actual value of \(\sqrt{5} = 2.236067977\ldots\text{.}\)

We can go one step further for this algorithm. If \(x_{1},x_{2},\ldots,x_{n},\ldots\) is the sequence we obtain using Newton's method for approximating \(\sqrt{a}\text{,}\) then as we saw earlier

\begin{equation*} x_{n+1} = \frac{1}{2}\left(x_{n} + \frac{a}{x_{n}}\right). \end{equation*}

Now, assume that \(\lim_{n\to\infty}x_{n}\) exists and is nonzero, and call it \(s\text{.}\) Then

\begin{equation*} \lim_{n\to\infty}x_{n+1} = \frac{1}{2}\left(\lim_{n\to\infty}x_{n} + \frac{a}{\lim_{n\to\infty}x_{n}}\right) \end{equation*}

becomes

\begin{equation*} s = \frac{1}{2}\left(s + \frac{a}{s}\right) \end{equation*}

which we can rearrange to get

\begin{equation*} s^{2} = \frac{1}{2}(s^{2} + a), \end{equation*}

or just \(s^{2} = a\text{.}\) Hence \(s = \pm\sqrt{a}\text{.}\)
