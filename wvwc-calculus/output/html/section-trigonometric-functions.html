Section 1.3 Trigonometric Functions

Subsection Angles and Terminal Points

Consider the unit circle in \(\RR^2\text{,}\) which is given by \(x^2 + y^2 = 1\text{.}\) Each point \(P\) on this circle makes an angle \(\theta\) with the positive \(x\)-axis, and therefore \(\theta\) must also determine the point completely. The angle \(\theta\) is typically specified using either radians or degrees. Converting from one unit to the other can be done by noting that \(\pi\) radians is precisely equal to \(180\) degrees.

Unless otherwise specified, we will use radians for angle measures in this course.

Example 1.3.1. Converting from Degrees to Radians.

Convert \(270\) degrees to radians and \(\frac{\pi}{6}\) radians to degrees.

Just as points on the unit circle determine angles, angles also determine points. A point \(P\) determined by an angle \(\theta\) is also known as the terminal point for the angle. Terminal points for certain reference angles are useful to remember.

Example 1.3.2. Finding Terminal Points.

Find the terminal points for the angles \(210^{\circ}\) and \(\frac{5\pi}{3}\text{.}\)

Solution.

First, note that \(210^{\circ} = \frac{7\pi}{6}\text{.}\) We therefore choose \(\frac{\pi}{6}\) as our reference angle and obtain the terminal point \((-\frac{\sqrt{3}}{2}, -\frac{1}{2})\text{.}\) The terminal point for \(\frac{5\pi}{3}\) is likewise \((\frac{1}{2}, -\frac{\sqrt{3}}{2})\text{.}\)

Subsection The Trigonometric Functions

Since angles determine terminal points on circles, the coordinates \((x, y)\) of each point on the circle can be viewed as functions of the angle \(\theta\text{.}\) These coordinate functions, \(x(\theta)\) and \(y(\theta)\text{,}\) are the fundamental trigonometric functions sine and cosine, and can be used to define the other four trigonometric functions commonly used.

Definition 1.3.3. Trigonometric Functions.

Let \(\theta\in\RR\) and let \(P = (x,y)\) denote the corresponding terminal point. The cosine function is the function \(\cos\theta = x\text{,}\) and the sine function is the function \(\sin\theta = y\text{.}\) The secant, cosecant, tangent and cotangent functions are defined as follows:

\begin{equation*} \sec\theta = \frac{1}{\cos\theta}, \csc\theta = \frac{1}{\sin\theta}, \tan\theta = \frac{\sin\theta}{\cos\theta}\text{ and }\cot\theta = \frac{\cos\theta}{\sin\theta}\text{.} \end{equation*}

The trigonometric functions satisfy important equalities known as Pythagorean identities.

Theorem 1.3.4. Pythagorean Identities.

Let \(\theta\in\RR\text{.}\) Then

\begin{equation*} \sin^2\theta + \cos^2\theta = 1\text{.} \end{equation*}

If \(\theta\neq\frac{\pi}{2}+k\pi\) for some \(k\in\ZZ\text{,}\) then

\begin{equation*} \tan^2\theta + 1 = \sec^2\theta\text{.} \end{equation*}

If \(\theta\neq k\pi\) for some \(k\in\ZZ\text{,}\) then

\begin{equation*} \cot^2\theta + 1 = \csc^2\theta\text{.} \end{equation*}
