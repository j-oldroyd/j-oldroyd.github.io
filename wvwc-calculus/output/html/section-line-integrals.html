    Section 14.2 Line Integrals

    In this section we move on to computing integrals over curves in \(\RR^{2}\) and \(\RR^{3}\text{.}\)

    Subsection Scalar Line Integrals

    Suppose we wish to find the mass of a wire given by a curve \(C\) in the \(xy\)-plane. If the density is a constant \(\rho\) then this is simple: the mass is just \(\rho L\) where \(L\) denotes the length of the wire.

    Now suppose that the density varies, and is now a function \(\rho(x,y)\text{.}\) Then we can estimate the mass of the wire by chopping the wire into \(k\) smaller segments, each of length \(\Delta s\text{,}\) choosing points \((x_{i},y_{i})\) from each segment, and then computing

    \begin{equation*} \sum_{j=1}^{k}\rho(x_{j}, y_{j})\Delta s\text{.} \end{equation*}

    If we send \(\Delta s\to 0\)the approximation becomes exact, and defines the scalar line integral of \(\rho\) along \(C\text{.}\)

    Definition 14.2.1. Scalar Line Integral.

    Suppose that \(C\) is a smooth curve in \(\RR^{2}\) and that the function \(f(x,y)\) is continuous on \(C\text{.}\) The scalar line integral of \(f\) along \(C\) is the number \(\int_{C}f(x,y)\,ds\) defined to be

    \begin{equation*} \int_{C}f(x,y)\,ds = \lim_{\Delta s\to 0}\sum_{j=1}^{k}f(x_{j},y_{j})\Delta s\text{.} \end{equation*}

    An analogous definition holds for curves in \(\RR^{3}\text{.}\) We can also replace the assumption that \(C\) is smooth with the assumption that \(C\) is rectifiable, which (roughly) means it has a well-defined length.

    Geometrically, \(\int_{C}f(x,y)\,ds\) represents the area under \(z = f(x,y)\) and above \(C\text{.}\)

    If \(C\) is nice enough (i.e., piecewise smooth), we can avoid using Riemann sums to compute the scalar line integral. Note that \(ds\) can be viewed as representing an infinitesimal length along the curve \(C\text{.}\) If \(C\) is traced out by the parametric equations

    \begin{align*} x \amp = x(t) \\ y \amp = y(t) \\ a \amp \leq t\leq b \text{,} \end{align*}

    then the length of \(C\) from \(a\) to \(t\) is given by the arc length function

    \begin{equation*} s(t) = \int_{a}^{t}\sqrt{x'(u)^{2} + y'(u)^{2}}\,du\text{.} \end{equation*}

    See Arc Length of Parametric Curves and Arc Length. Thus

    \begin{equation*} \dv{s}{t} = \sqrt{x'(t)^{2} + y'(t)^{2}}\text{,} \end{equation*}

    and we can write

    \begin{equation*} \int_{C}f(x,y)\,ds = \int_{a}^{b}f(x(t),y(t))\sqrt{x'(t)^{2} + y'(t)^{2}}\,dt\text{.} \end{equation*}

    If we view \(C\) as being traced out by a vector function \(\vb{r}(t), a\leq t\leq b\text{,}\) then we can write this formula even more compactly:

    \begin{equation*} \int_{C}f(x,y)\,ds = \int_{a}^{b}f(\vb{r}(t))\norm{\vb{r}'(t)}\,dt\text{.} \end{equation*}

    Example 14.2.2. Computing a Scalar Line Integral.

    Compute \(\int_{C}(xy)^{1/3}\,ds\text{,}\) where \(C\) is the segment of the parabola \(y = x^{2}\) from \(x = 0\) to \(x = 1\text{.}\)

    Solution.

    The curve \(C\) is traced out by the vector function \(\vb{r}(t) = \dotprod{t,t^{2}}\) for \(0\leq t\leq 1\text{.}\) Therefore

    \begin{align*} \int_{C}(xy)^{1/3}\,ds \amp = \int_{0}^{1} t\sqrt{1 + 4t^{2}}\,dt \\ \amp = \frac{1}{8}\int_{1}^{5}\sqrt{u}\,du \\ \amp = \frac{5^{3/2} - 1}{12} \end{align*}

    Line integrals along piecewise smooth curves can be found without too much trouble.

    Example 14.2.3. Integrating Along Line Segments.

    Find \(\int_{C}(3x - 3y)\,ds\) where \(C\) denotes the line segment from \((-1,0)\) to \((0,1)\) followed by the segment from \((0,1)\) to \((1,0)\text{.}\)

    Solution.

    We find this by breaking \(C\) down as the union of two smooth curves \(C_{1}\) and \(C_{2}\text{,}\) where

    \begin{align*} C_{1}:\vb{r}_{1}(t) \amp = (1-t)\dotprod{-1,0} + t\dotprod{0,1}, 0\leq t\leq 1 \\ C_{2}:\vb{r}_{2}(t) \amp = (1-t)\dotprod{0,1} + t\dotprod{1,0} \end{align*}

    Then

    \begin{align*} \int_{C}(2x - 3y)\,ds \amp = \int_{C_{1}}(2x - 3y)\,ds + \int_{C_{2}}(2x - 3y)\,ds \\ \amp = \int_{0}^{1}\left[(2t - 2 - 3t) + (2t - 3 + 3t)\right]\sqrt{2}\,dt \text{.} \end{align*}

    We can also compute scalar line integrals in \(\RR^{3}\text{.}\)

    Example 14.2.4. Average Value on a Circle.

    Let \(f(x,y,z) = x^{2} + y^{2} - 2z^{2}\) and let

    \begin{equation*} C:\vb{r}(t) = \dotprod{5\sin t,3\cos t, 4\cos t}, 0\leq t\leq 2\pi\text{.} \end{equation*}

    Find the average value of \(f\) on \(C\text{.}\)

    Solution.

    The average value of \(f\) on \(C\) is given by

    \begin{equation*} \frac{1}{\operatorname{length}(C)}\int_{C}f(x,y,z)\,ds \end{equation*}

    where

    \begin{align*} \operatorname{length}(C) \amp = \int_{C}\,ds \\ \amp = 10\pi \end{align*}

    and

    \begin{align*} \int_{C}f(x,y,z)\,ds \amp = \int_{0}^{2\pi}f(\vb{r}(t))\norm{\vb{r}'(t)}\,dt \\ \amp = 5\int_{0}^{2\pi}(25\sin^{2}t + 9\cos^{2}t - 32\cos^{2}t)\,dt \\ \amp = 5\int_{0}^{2\pi}(25 - 48\cos^{2}t)\,dt\\ \amp = 10\pi \text{.} \end{align*}

    Therefore \(f(x,y,z)\) has an average value of \(1\) on \(C\text{.}\)

    Subsection Vector Field Line Integrals

    Suppose we have a force field \(\vb{F}\) in \(\RR^{2}\) and some particle that is acted upon by the force. The particle has a trajectory given by a curve \(C: \vb{r}(t), 0\leq t\leq 1\text{.}\)

    We want to determine how the trajectory of the particle aligns with the forcefield \(\vb{F}\text{.}\) We can do this by picking an arbitrary point along \(C\text{,}\) finding the unit tangent \(\vb{T}\) at this point, and then comparing the direction of \(\vb{T}\) and that of \(\vb{F}\) by computing the dot product \(\vb{F}\cdot\vb{T}\text{.}\) This is a measure of how \(\vb{F}\) and \(C\) align at a specific point, and integrating this along \(C\) should tell us how \(C\) and \(\vb{F}\) align overall.

    Definition 14.2.6. Line Integral of a Vector Field.

    Let \(C\) denote a smooth curve and let \(\vb{F}\) be a vector field continuous on \(C\text{.}\) Then we define the line integral of \(\vb{F}\) over \(C\) to be \(\int_{C}\vb{F}\cdot\vb{T}\,ds\text{.}\)

    This integral is also sometimes called the circulation integral of \(\vb{F}\) along \(C\text{.}\)

    To find line integrals of vector fields, we proceed as follows. Suppose that \(C:\vb{r}(t), a\leq t\leq b\text{.}\) Then

    \begin{align*} \int_{C}\vb{F}\cdot\vb{T}\,ds \amp = \int_{a}^{b}\vb{F}(\vb{r}(t))\cdot\frac{\vb{r}'(t)}{\norm{\vb{r}'(t)}}\norm{\vb{r}'(t)}\,dt \\ \amp = \int_{a}^{b}\vb{F}(\vb{r}(t))\cdot\vb{r}^{\prime}(t)\,dt \text{.} \end{align*}

    We can also write this as

    \begin{equation*} \int_{a}^{b}\vb{F}\cdot d\vb{r} \qq{or} \int_{a}^{b} P\,dx + Q\,dy\text{,} \end{equation*}

    assuming that \(\vb{F} = \dotprod{P,Q}\text{.}\)

    In addition to measuring how well \(C\) and \(\vb{F}\) align, vector field line integrals can also represent work done.

    Example 14.2.7. Flow Along a Circle.

    Let \(C\) denote the segment of the parabola \(y = \cos x\) traversed once from \(x = -\frac{\pi}{2}\) to \(x = \frac{\pi}{2}\text{,}\) and let

    \begin{equation*} \vb{F} = \dotprod{y^{2}, -x}\text{.} \end{equation*}

    Does \(\vb{F}\) tend to flow with or against \(C\text{?}\)

    Solution.

    If we graph \(\vb{F}\) and \(C\text{,}\) we get the following:

    Figure¬†14.2.8 suggests that \(\vb{F}\) flows with \(C\text{,}\) and we can verify this by computing \(\int_{C}\vb{F}\cdot d\vb{r}\text{:}\)

    \begin{align*} \int_{C}\vb{F}\cdot d\vb{r} \amp = \int_{-\pi/2}^{\pi/2} \dotprod{\cos^{2}t, -t}\cdot\dotprod{1,-\sin t}\,dt\\ \amp = \int_{-\pi/2}^{\pi/2} \brackets{\cos^{2}t + t\sin t}\,dt\\ \amp = \int_{-\pi/2}^{\pi/2} \brackets{ \frac{1}{2}\parens{1 + \cos 2t} + t\sin t}\,dt \\ \amp = \brackets{\frac{1}{2}\parens{ t + \frac{1}{2}\sin 2t} - t\cos t + \sin t}_{-\pi/2}^{\pi/2} \\ \amp = \frac{\pi}{2} + 2 \text{.} \end{align*}

    Since the result is positive, this tells us that \(\vb{F}\) tends to flow with \(C\text{.}\)

    Example 14.2.9. A Nonconservative Force.

    Let \(\vb{F} = \dotprod{-y, x}\) denote a force field. Is the force \(\vb{F}\) conservative?

    Solution.

    Here we are using the physical definition of a conservative force, namely that the work done must be path independent. So we'll choose two paths between two points and find the work done on each path. Let the first path, \(C_{1}\text{,}\) denote the top half of the unit circle traversed counterclockwise. Similarly, let the second path, \(C_{2}\text{,}\) denote the bottom half of the unit circle traversed clockwise. Then both paths have the same initial and terminal points.

    From Figure¬†14.2.10, it appears as though \(\int_{C_{2}}\vb{F}\cdot d\vb{r} \lt 0 \lt \int_{C_{1}}\vb{F}\cdot d\vb{r}\text{,}\) which suggests that the vector field is path dependent.

    To verify this, we'll compute the appropriate line integrals.

    For \(C_{1}:\vb{r}_{1}(t) = \dotprod{\cos t,\sin t}, 0\leq t\leq \pi\text{,}\) the total circulation is

    \begin{align*} \int_{C_{1}}\vb{F}\cdot\vb{T}\,ds \amp = \int_{0}^{\pi}P\,dx + Q\,dy\\ \amp = \int_{0}^{\pi}\,dt \\ \amp = \pi \text{.} \end{align*}

    Likewise, the circulation of \(\vb{F}\) along \(C_{2}\) is

    \begin{equation*} \int_{C_{2}}\vb{F}\cdot\vb{T}\,ds = -\pi\text{.} \end{equation*}

    Since the work done by \(\vb{F}\) between the points \((1,0)\) and \((-1,0)\) clearly depends on the path taken between the points, this means that \(\vb{F}\) is path dependent and, therefore, not conservative.

    We can also compute vector field line integrals in \(\RR^{3}\) with essentially the same formula.

    Example 14.2.11. Circulation in \(\RR^{3}\).

    Find the circulation of \(\vb{F} = x^{2}\vb{i} + \vb{k}\) along

    \begin{equation*} C:\vb{r}(t) = 3t\vb{i} + 4t\vb{j}, 0\leq t\leq 2\text{.} \end{equation*}
    Solution.

    We have

    \begin{align*} \int_{C}\vb{F}\cdot\vb{T}\,ds \amp = \int_{0}^{2} P\,dx + Q\,dy + R\,dz \\ \amp = \int_{0}^{2}\brackets{9t^{2}(3) + 0 + 0}\,dt \\ \amp = 72 \text{.} \end{align*}

    Since the circulation is positive, this also shows that \(\vb{F}\) tends to flow with \(C\text{.}\)

    Subsection Flux Integrals

    The circulation integral in Definition¬†14.2.6 is useful for measuring how much a vector field flows along a given curve. Now, we want to measure how a vector field flows across a curve, at least in \(\RR^{2}\text{.}\)

    Given a smooth curve \(C\) with unit tangent \(\vb{T}\text{,}\) recall that \(\vb{F}\cdot\vb{T}\) measures how well a vector field \(\vb{F}\) and \(\vb{T}\) align at a point on \(C\text{.}\) By integrating this, we get a measure along the entire curve. So if we want to get a sense of how \(\vb{F}\) flows across \(C\text{,}\) we can do so by looking at a single point and then integrating along the curve again.

    To do this, let \(\vb{N}\) be the unit normal vector to \(C\text{,}\) given in Definition¬†11.8.8. Then \(\vb{F}\cdot\vb{N}\) determines how \(\vb{F}\) flows across \(C\) at a specific point, and integrating \(\vb{F}\cdot\vb{N}\) provides a measure along the entire curve. But there's one slight issue: if \(C\) is a closed curve then it's possible that \(\vb{N}\) points into the region enclosed by \(C\text{.}\) We would like to measure how \(\vb{F}\) flows out of \(C\) instead, so we'll replace \(\vb{N}\) with \(\vb{n}\text{,}\) the outward unit normal. This leads us to the flux integral.

    Definition 14.2.12. Flux Integral.

    Let \(C\) denote a smooth curve and let \(\vb{F}\) be a vector field continuous on \(C\text{.}\) Then we define the flux integral of \(\vb{F}\) across \(C\) to be \(\int_{C}\vb{F}\cdot\vb{n}\,ds\text{,}\) where \(\vb{n}\) is the outward unit normal vector to \(C\text{.}\)

    At this point it may be helpful to introduce some new notation. If \(C\) is a closed curve, then we often denote line integrals involving \(C\) with \(\oint_{C}\) instead of \(\int_{C}\text{.}\)

    Example 14.2.13. Flux Across the Unit Circle.

    Let \(\vb{F} = \dotprod{-y, x}\) and let \(C\) denote the unit circle, traversed exactly once counterclockwise. What is \(\oint_{C}\vb{F}\cdot\vb{n}\,ds\text{?}\)

    Solution.

    We should have \(\oint_{C}\vb{F}\cdot\vb{n}\,ds = 0\text{.}\)

    In order to actually compute flux integrals, we need to write \(\vb{n}\) using \(C\text{.}\) So suppose that \(C\) is traced out by the vector function \(\vb{r}(t) = \dotprod{x(t), y(t)}\text{.}\) Then \(\vb{n}\) must be perpendicular to the unit tangent

    \begin{equation*} \vb{T} = \frac{1}{\sqrt{x^{2} + y^{2}}}\dotprod{x', y'} \end{equation*}

    or equivalently \(\dotprod{x',y'}\text{.}\) Our primary tool for finding perpendicular vectors, the cross product, only applies in \(\RR^{3}\text{.}\) So to find \(\vb{n}\) we'll (temporarily) move everything into \(\RR^{3}\text{.}\) Now we need to find a vector orthogonal to \(\dotprod{x', y', 0}\) that also lies in the \(xy\)-plane. This can be done by computing

    \begin{equation*} \dotprod{x',y',0}\times\vb{k} \qq{or} \vb{k}\times \dotprod{x',y',0}\text{.} \end{equation*}

    At this point we need to decide which direction we want our normal vector to go. We'll usually choose the first option if \(C\) is traversed counterclockwise and the second option otherwise. Assuming counterclockwise orientation, we have

    \begin{equation*} \dotprod{x', y', 0}\times \vb{k} = \dotprod{y', -x', 0}\text{.} \end{equation*}

    Now moving back down to \(\RR^{3}\text{,}\) we can write the outward unit normal as

    \begin{equation*} \vb{n} = \frac{1}{\sqrt{(x')^{2} + (y')^{2}}}\dotprod{y', -x'} = \frac{1}{\norm{\vb{r}'(t)}}\dotprod{y', -x'}\text{.} \end{equation*}

    Therefore

    \begin{align*} \int_{C}\vb{F}\cdot\vb{n}\,ds \amp = \int_{C}\vb{F}\cdot\dotprod{y', -x'}\,dt\\ \amp = \int_{C} P\,dy - Q\,dx \text{.} \end{align*}

    Example 14.2.14. Verifying the Flux.

    Compute the flux integral in Example¬†14.2.13.

    Solution.

    We have

    \begin{align*} \oint_{C}\vb{F}\cdot\vb{n}\,ds \amp = \int_{0}^{2\pi}P\,dy - Q\,dx \\ \amp = \int_{0}^{2}\brackets{-\sin t\cos t - \cos t(-\sin t)}\,dt \\ \amp = 0 \text{.} \end{align*}
