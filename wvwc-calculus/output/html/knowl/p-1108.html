The sum \(\mathbf{x}+\mathbf{y}\) of vectors \(\mathbf{x},\mathbf{y}\) can be computed using either the triangle law, illustrated above in Figure 11.2.2, or the similar parallelogram law. We can also scale vectors using scalar multiplication: if \(\alpha\) is a scalar 4  then \(\alpha\mathbf{x}\) is defined to be the vector that has the same direction as \(\mathbf{x}\) if \(\alpha>0\) and the opposite direction if \(\alpha<0\text{,}\) but the magnitude is rescaled by the factor \(\alpha\text{.}\)
