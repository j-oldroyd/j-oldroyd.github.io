The process used in Section 6.1 can be generalized quite a bit. Start with a function \(f(x)\) defined on some interval \([a,b]\text{,}\) and divide \([a,b]\) into \(n\) subintervals
\begin{equation*} [x_{0},x_{1}], [x_{1},x_{2}],\ldots,[x_{n-1},x_{n}] \end{equation*}
where \(x_{0} = a, x_{n} = b\) and each subinterval has length \(\Delta x_{k}\) (i.e. \(\Delta x_{k} = x_{k} - x_{k-1}\)). This is known as a partition of \([a,b]\text{.}\) Then we can choose sample points \(x_{1}^{*}\) from \([x_{0},x_{1}]\text{,}\) \(x_{2}^{*}\) from \([x_{1},x_{2}]\text{,}\) all the way up to \(x_{n}^{*}\) from \([x_{n-1},x_{n}]\text{.}\) Note that the only restriction we’re placing on the sample point \(x_{k}^{*}\) is that is must come from the interval \([x_{k-1},x_{k}]\text{,}\) but it may be any point in this interval.
