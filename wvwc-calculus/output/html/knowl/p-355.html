If we imagine graphing some differentiable function \(f(x)\) on some interval \([a,b]\text{,}\) but \(f(a)\neq f(b)\text{,}\) then we can't apply Rolle's Theorem. But maybe we can adjust it just a little bit so that we can? In particular, the only reason we can't use Theorem 5.2.1 is that \(f(a)\neq f(b)\text{.}\) But if we subtract the line through these points from \(f(x)\text{,}\) we should get a new function for which Theorem 5.2.1 applies.
