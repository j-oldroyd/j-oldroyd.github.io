We can answer all of these questions by setting up a sign chart for
\begin{equation*} f^{\prime}(x) = x^{3}e^{-x}(4-x)\text{.} \end{equation*}
The critical points are \(x=0,4\text{,}\) and \(f^{\prime} < 0\) on \((-\infty,0)\cup(4,\infty)\) and \(f^{\prime} > 0\) on \((0,4)\text{.}\) So \(f\) is decreasing on the first set of intervals, increasing on \((0,4)\text{,}\) \(f\) has a local maximum at \(x=4\) and a local minimum at \(x=0\text{.}\)
