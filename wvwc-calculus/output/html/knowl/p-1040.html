Using Theorem 9.7.1, the Taylor series for \(f(x) = \cos x\) at \(a = 0\) is given by
\begin{equation*} \sum_{k=0}^{\infty}\frac{f^{(k)}(0)}{k!}x^{k}\text{.} \end{equation*}
So to find this series we need to find the derivatives of cosine at \(0\text{:}\)
\begin{align*} f(0) \amp = 1 \\ f^\prime(0) \amp = 0 \\ f''(0) \amp = -1 \\ f^{(3)}(0) \amp = 0 \end{align*}
and so on. So the Maclaurin series for \(\cos x\) is given by
\begin{equation*} 1 - \frac{1}{2!}x^{2} + \frac{1}{4!}x^{4} - \frac{1}{6!}x^{6} + \cdots\text{.} \end{equation*}
