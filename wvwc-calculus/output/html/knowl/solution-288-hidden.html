For any point \((x,y,z)\) on this surface, we know that \(x = 3u - v, y = u + v\) and \(z = 2u + 3v\text{.}\) We can use these conditions to find an equation in \(x, y\) and \(z\) that is equivalent to \(\vb{r}\text{.}\)

First, let's solve for \(u\) and \(v\text{,}\) giving

\begin{align*} u \amp = \frac{x + y}{4} \\ v \amp = \frac{3y - x}{4} \end{align*}

So

\begin{equation*} x + y + z = 6u + 3v = \frac{3}{4}x + \frac{15}{4}y\text{,} \end{equation*}

which implies that

\begin{equation*} \frac{1}{4}x - \frac{11}{4}y + z = 0\text{.} \end{equation*}
