Computing integrals in practice often involves setting up and approximating Riemann sums. These sums depend on how we choose the sample points \(x_{k}^{*}\) from each interval \([x_{k-1},x_{k}]\text{.}\) We've mostly worked with \(x_{k}^{*} = x_{k}\text{,}\) or right endpoints. We could also choose \(x_{k}^{*} = x_{k-1}\text{,}\) using left endpoints. However, it's often more accurate to use midpoints: \(x_{k}^{*} = \frac{x_{k-1}+x_{k}}{2}\text{.}\) The reason for this is that if we choose midpoints of intervals as our sample points, these tend to be less affected than either endpoint if the function is increasing or decreasing. In particular, we have the midpoint rule.
