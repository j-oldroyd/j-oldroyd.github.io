We need to check the sign of \(\int_{0}^{2\pi}x^{2}\sin x\,dx\text{.}\) Using the tabular method, we get

\begin{equation*} \int_{0}^{2\pi}x^{2}\sin x\,dx = [-x^{2}\cos x + 2x\sin x + 2\cos x]_{0}^{2\pi} = -4\pi^{2} + 4\pi - 2. \end{equation*}

Since this is negative, \(f(x)\) tends to be negative along \([0,2\pi]\text{.}\)
