Example 3.7.2. Linear approximation of sine.

Find the linear approximation of \(f(\theta) = \sin\theta\) at \(\theta=0\) and use this to estimate \(\sin(-.02)\text{.}\)
Solution.
The linear approximation is given by
\begin{equation*} L(\theta) = f(0) + f'(0)(\theta - 0) = \sin0 + \cos0 (\theta - 0), \end{equation*}
so \(L(\theta) = \theta\text{.}\) Therefore
\begin{equation*} \sin(-.02)\approx-.02 \end{equation*}
since \(-.02\) is very close to \(0\text{.}\)
