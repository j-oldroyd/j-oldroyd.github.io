Example 4.2.9. Derivative at a point.

Let \(f(x) = x^{3} + 3\sin x + 2\cos x\text{.}\) Find \((f^{-1})'(2)\)
Solution.
By TheoremÂ 4.2.7, we know that
\begin{equation*} (f^{-1})'(2) = \frac{1}{f'(f^{-1}(2))}. \end{equation*}
By inspection, \(f(0) = 2\) which means \(f^{-1}(2) = 0\text{.}\) Since
\begin{equation*} f'(x) = 3x^{2} + 3\cos x - 2\sin x, \end{equation*}
this gives
\begin{equation*} (f^{-1})'(2) = \frac{1}{f'(f^{-1}(2))} = \frac{1}{f'(0)} = \frac{1}{3}. \end{equation*}
