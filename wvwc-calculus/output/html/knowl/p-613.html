Since \(\frac{1}{(x^{2} - 1)^{2}} = \frac{1}{(x-1)^{2}(x+1)^{2}}\text{,}\) it’s tempting to write
\begin{equation*} \frac{1}{(x^{2}-1)^{2}} = \frac{A_{1}}{x-1} + \frac{A_{2}}{x+1}. \end{equation*}
Unfortunately, doing so gives \(A_{1} = A_{2} = 0\text{,}\) which obviously won’t work. If we instead use
\begin{equation*} \frac{1}{(x^{2}-1)^{2}} = \frac{A_{1}}{(x-1)^{2}} + \frac{A_{2}}{(x+1)^{2}}, \end{equation*}
we get \(A_{1} = A_{2} = \frac{1}{4}\text{.}\) However,
\begin{equation*} \frac{1}{(x^{2}-1)^{2}} \neq \frac{\frac{1}{4}}{(x-1)^{2}} + \frac{\frac{1}{4}}{(x+1)^{2}}. \end{equation*}
