Solution 9.5.2.1.

We'll try the ratio test to check convergence of this series. Doing so, we get
\begin{equation*} r = \lim_{k\to\infty}|x|\frac{k+1}{k+2} = |x|\text{.} \end{equation*}
So the series converges if \(|x| \lt 1\) and diverges if \(|x| > 1\text{.}\)
When \(r = |x| = 1\text{,}\) or \(x = \pm1\text{,}\) the test fails. So we need to use other methods to determine the convergence or the series at these points. At \(x = -1\text{,}\) the series becomes
\begin{equation*} \sum_{k=0}^{\infty}\frac{(-1)^{2k+1}}{k+1} = - \sum_{k=0}^{\infty}\frac{1}{k+1}\text{,} \end{equation*}
which diverges by the comparison test. At \(x = 1\text{,}\) the series reduces to the alternating harmonic series which converges by ExampleÂ 9.4.3.
Therefore this series converges for all \(x\) in the interval \((-1, 1]\) and diverges otherwise.
