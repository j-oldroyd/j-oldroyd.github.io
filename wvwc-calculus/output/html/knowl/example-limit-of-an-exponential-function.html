Example 4.1.2. Limit of an exponential function.

Let \(f(x) = 2^{-x}\text{.}\) Find \(\lim_{x\to\infty}f(x)\text{.}\)
Solution.
First, note that
\begin{equation*} f(x) = \frac{1}{2^{x}}. \end{equation*}
So as \(x\to\infty\text{,}\) the denominator \(2^{x}\to\infty\) as well. Hence the fraction as a whole should decrease to \(0\text{,}\) and so
\begin{equation*} \lim_{x\to\infty}2^{-x} = 0. \end{equation*}
