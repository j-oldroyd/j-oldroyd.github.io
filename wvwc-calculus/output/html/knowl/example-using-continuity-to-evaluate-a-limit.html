Example 2.3.9. Using continuity to evaluate a limit.

Evaluate \(\lim_{x\to\pi}\cos(x+\sin x)\text{.}\)
Solution.
Since \(x, \cos x\) and \(\sin x\) are all continuous, this means that \(\cos(x+\sin x)\) must be continuous as well. Therefore
\begin{equation*} \lim_{x\to\pi}\cos(x+\sin x) = \cos(\pi+\sin\pi) = -1. \end{equation*}
