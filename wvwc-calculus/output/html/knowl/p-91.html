If \(f(x)\) is a function and \(\lim_{x\to a}f(x)=\pm\infty\text{,}\) \(\lim_{x\to a^{-}}f(x)=\pm\infty\) or \(\lim_{x\to a^{+}}f(x)=\pm\infty\text{,}\) then this means that the function has a vertical asymptote at \(x=a\text{.}\) In this course, this basically corresponds to a divide-by-zero problem.
