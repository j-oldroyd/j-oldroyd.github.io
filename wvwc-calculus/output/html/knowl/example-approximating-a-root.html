Example 5.6.1. Approximating a root.

Find the third approximation given by Newtonâ€™s Method for the root of \(f(x) = x^{5} - x - 1\text{,}\) using \(x_{1} = -1\text{.}\)
Solution.
First, note that \(f^{\prime}(x) = 5x^{4} - 1\text{.}\) Then
\begin{align*} x_{2} & = x_{1} - \frac{f(x_{1})}{f^{\prime}(x_{1})} \\ & = -1 - \frac{-1}{4} \\ & = 3, \end{align*}
so \(x_{2} = 3\text{.}\) Now we run through this method again to get \(x_{3}\text{:}\)
\begin{align*} x_{3} & = x_{2} - \frac{f(x_{2})}{f^{\prime}(x_{2})} \\ & = 3 - \frac{239}{404} \\ & = \frac{973}{404}. \end{align*}
So \(x_{3} = \frac{973}{404}.\)
