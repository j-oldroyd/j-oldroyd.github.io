We can use vectors to describe curves and surfaces. For example, let \(\mathbf{r} = \dotprod{x,y,z}\) and \(\mathbf{r}_{0} = \dotprod{a,b,c}\text{.}\) Let \(r\geq0\text{.}\) Then the set of all points \((x,y,z)\) that satisfy the equation

\begin{equation*} \norm{\mathbf{r}-\mathbf{r}_{0}} = r \end{equation*}

has a very nice description: it's just the sphere of radius \(r\) centered at \((a,b,c)\text{.}\)
