Theorem 4.5.4. Derivative of Inverse Sine.

Let \(-1 < x < 1\text{.}\) Then \(\dv{}{x}\sin^{-1}x = \frac{1}{\sqrt{1-x^{2}}}\text{.}\)

Proof.

If we set \(y = \sin^{-1}x\text{,}\) then we get \(\sin y = x\text{.}\) Differentiating this gives \(y'\cos y = 1\) or just
\begin{equation*} y' = \frac{1}{\cos y} = \frac{1}{\sqrt{1-x^{2}}}. \end{equation*}
