Solution 12.8.5.1.

First, recall from Section 12.7 that the absolute maxima and minima of a differentiable function over a bounded set occur at either a critical point or somewhere on the boundary. So first, we need to find if \(f(x,y)\) has any critical points inside the disk. If we solve \(f_{x} = f_{y} = 0\text{,}\) then we quickly get that \((0,0)\) is a critical point, and hence a possible maximum or minimum value. The only other place we need to check is the boundary \(x^{2} + y^{2} = 1\text{,}\) and this is something we can use Lagrange multipliers for.
Set \(g(x,y) = x^{2} + y^{2}\text{.}\) Then we need to find the extreme values of \(f\) subject to the constraint \(g = 1\text{.}\) The system of equations we need to solve is then
\begin{align*} 2x & = \lambda(2x) & \Rightarrow x = \lambda x \\ 4y & = \lambda(2y) & \Rightarrow 2y = \lambda y\\ x^{2} + y^{2} & = 1 \end{align*}
The first equation is true if \(\lambda = 1\text{,}\) but then this forces \(y = 0\) in the second equation. If we then plug \(y = 0\) into the constraint, this forces \(x = \pm1\text{.}\) So the case \(\lambda=1\) gives us two points to test: \((-1,0)\) and \((1,0)\text{.}\) On the other hand, the second equation is true if \(\lambda = 2\text{.}\) This then forces \(x = 0\) and \(y = \pm 1\text{,}\) which gives us the points \((0,-1)\) and \((0,1)\text{.}\) Finally, if \(\lambda\neq1,2\text{,}\) then this forces \(x = y = 0\text{.}\) But there’s no way to satisfy the constraint if \(x = y = 0\text{,}\) so we discard this possibility.
So we have five points that we need to check, which we arrange in the following table: So the absolute minimum is \(0\text{,}\) which occurs at the origin. The absolute maximum is \(2\text{,}\) which occurs at the points \((0,\pm1)\text{.}\)
