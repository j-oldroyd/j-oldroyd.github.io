Solution 12.7.5.1.

We have \(f(x,y) = xe^{-x^{2} - y^{2}}\text{,}\) and the critical points are \((-\sqrt{\frac{1}{2}},0)\) and \((\sqrt{\frac{1}{2}},0)\text{.}\) We need to find \(D(x,y)\text{.}\) Since \(f_{x} = (1 - 2x^{2})e^{-x^{2} - y^{2}}\text{,}\) we get
\begin{align*} f_{xx} & = -4xe^{-x^{2} - y^{2}} - 2x(1 - 2x^{2})e^{-x^{2} - y^{2}} \\ f_{xy} & = -2y(1-2x^{2})e^{-x^{2} - y^{2}} = f_{yx} \end{align*}
and similarly
\begin{equation*} f_{yy} = -2xe^{-x^{2} - y^{2}} + 4xy^{2}e^{-x^{2} - y^{2}} \end{equation*}
At \((-\sqrt{\frac{1}{2}},0)\text{,}\) we have
\begin{align*} D\left(-\sqrt{\frac{1}{2}},0\right) & = 4\sqrt{\frac{1}{2}}e^{-\frac{1}{2}}\cdot2\sqrt{\frac{1}{2}}e^{-\frac{1}{2}}\\ & > 0 \end{align*}
and \(f_{xx}(-\sqrt{\frac{1}{2}},0) >0\) as well. So this is a local minimum. At \((\sqrt{\frac{1}{2}},0)\text{,}\) we still have \(D>0\text{,}\) but now \(f_{xx}<0\text{.}\) So this is a local maximum.
