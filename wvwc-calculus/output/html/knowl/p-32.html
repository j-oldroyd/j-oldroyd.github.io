Let \(\theta\in\RR\) and let \(P = (x,y)\) denote the corresponding terminal point. The cosine function is the function \(\cos\theta = x\text{,}\) and the sine function is the function \(\sin\theta = y\text{.}\) The secant, cosecant, tangent and cotangent functions are defined as follows:
\begin{equation*} \sec\theta = \frac{1}{\cos\theta}, \csc\theta = \frac{1}{\sin\theta}, \tan\theta = \frac{\sin\theta}{\cos\theta}\text{ and }\cot\theta = \frac{\cos\theta}{\sin\theta}\text{.} \end{equation*}
