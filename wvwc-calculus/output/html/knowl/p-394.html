Now, we have a constraint that \((x,y)\) must satisfy; namely, this must lie on the ellipse \(4x^{2} + y^{2} = 4\text{.}\) This means that \(y^{2} = 4 - 4x^{2}\text{,}\) and if we plug this into our distance function we get

\begin{equation*} d = \sqrt{(x-1)^{2} + (y-0)^{2}} = \sqrt{(x-1)^{2} + 4 - 4x^{2}}. \end{equation*}

Now here's a trick we can use: maximizing \(d\) is the same thing as maximizing \(d^{2}\text{,}\) but \(d^{2}\) is much nicer to work with algebraically. So instead of maximizing \(d\text{,}\) we'll maximize the function

\begin{equation*} f(x) = d^{2} = (x-1)^{2} + 4 - 4x^{2}. \end{equation*}
