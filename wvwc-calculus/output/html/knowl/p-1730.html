Theorem 14.3.4 shows that line integrals involving conservative vector fields are straightforward to evaluate if we know a corresponding potential function. So now we want to do two things:
  1. Given a vector field \(\vb{F}\text{,}\) determine if it’s conservative.
  1. Given a conservative vector field \(\vb{F}\text{,}\) determine a potential function \(\varphi\text{.}\)
