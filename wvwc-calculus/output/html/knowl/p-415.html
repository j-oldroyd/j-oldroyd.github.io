To use Newton’s method, we need to come up with a function \(f(x)\) whose root is \(\sqrt{a}\text{.}\) A simple choice for this is \(f(x) = x^{2} - a\text{,}\) since \(f(\sqrt{a}) = 0\text{.}\) Now, if we’re getting some sequence \(x_{1},\ldots,x_{n}\) from Newton’s method then the next term in the sequence is
\begin{equation*} x_{n+1} = x_{n} - \frac{f(x_{n})}{f^{\prime}(x_{n})} = x_{n} - \frac{x_{n}^{2} - a}{2x_{n}}. \end{equation*}
After some algebra, we can rewrite this as
\begin{equation*} x_{n+1} = \frac{1}{2}\left(x_{n} + \frac{a}{x_{n}}\right). \end{equation*}
