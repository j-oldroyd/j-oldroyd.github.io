Example 3.3.10. Derivative of tangent.

Let \(f(x) = \tan x\text{.}\) Find \(f'(x)\text{.}\)
Solution.
Since \(\tan x = \frac{\sin x}{\cos x}\text{,}\) then we can apply the quotient rule to get the derivative of \(\tan x\text{:}\)
\begin{equation*} \dv{}{x}\tan x = \frac{\cos^{2}x + \sin^{2}x}{\cos^{2}x} = \sec^{2}x. \end{equation*}
