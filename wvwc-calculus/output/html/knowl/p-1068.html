We can also rewrite these equations in terms of the starting and ending values of \(x\) and \(y\text{.}\) In particular, we have
\begin{align*} x \amp = 1(1 - t) + (-2)t \\ y \amp = -3(1 - t) + 4t \\ 0 \amp \leq t\leq 1 \text{.} \end{align*}
To see why this works, note that the second term in each equation vanishes at \(t = 0\text{,}\) leaving the starting value for each coordinate. Likewise, the first term in each equation vanishes at \(t = 1\text{,}\) leaving the terminal value for each coordinate. Expanding the equations gives the previous result. Note that this method only works if you select \([0,1]\) as your interval of values for \(t\text{.}\)
