Example 1.3.2. Finding Terminal Points.

Find the terminal points for the angles \(210^{\circ}\) and \(\frac{5\pi}{3}\text{.}\)
Solution.
First, note that \(210^{\circ} = \frac{7\pi}{6}\text{.}\) We therefore choose \(\frac{\pi}{6}\) as our reference angle and obtain the terminal point \((-\frac{\sqrt{3}}{2}, -\frac{1}{2})\text{.}\) The terminal point for \(\frac{5\pi}{3}\) is likewise \((\frac{1}{2}, -\frac{\sqrt{3}}{2})\text{.}\)
