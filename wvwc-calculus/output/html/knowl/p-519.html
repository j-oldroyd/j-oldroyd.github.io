First, note that we can compute \(\int_{1}^{x}f(t)\,dt\) by finding an antiderivative of \(f(t) = 2 - t^{2}\text{.}\) So
\begin{equation*} g(x) = \int_{1}^{x}(2-t^{2})\,dt = \left[2t-\frac{1}{3}t^{3}\right]_{t=1}^{x} \end{equation*}
or just
\begin{equation*} g(x) = 2x - \frac{1}{3}x^{3} - \frac{5}{3}. \end{equation*}
