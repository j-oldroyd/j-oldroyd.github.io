Example 11.7.1. Limit of a vector function.

Let
\begin{equation*} \mathbf{r}(t) = \frac{t^{2}-1}{t-1}\mathbf{i} + \sqrt{t+8}\mathbf{j} - \frac{\sin\pi t}{\ln t}\mathbf{k}. \end{equation*}
Suppose we want to find \(\lim_{t\to 1}\mathbf{r}(t)\text{.}\) Then we just need to take the limit of each component. So
\begin{equation*} \lim_{t\to1}\mathbf{r}(t) = 2\mathbf{i}+3\mathbf{j} -\pi\mathbf{k}. \end{equation*}
