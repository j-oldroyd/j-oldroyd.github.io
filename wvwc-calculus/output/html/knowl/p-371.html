The line through these points has equation
\begin{equation*} y = f(a) + \frac{f(b) - f(a)}{b-a}(x-a). \end{equation*}
So define
\begin{equation*} g(x) = f(x) - y = f(x) - f(a) - \frac{f(b) - f(a)}{b-a}(x-a). \end{equation*}
Then
\begin{equation*} g(a) = g(b) = 0, \end{equation*}
So Theorem 5.3.1 does apply to this function. Hence there exists some number \(c\) between \(a\) and \(b\) for which \(g'(c) = 0\text{.}\)
