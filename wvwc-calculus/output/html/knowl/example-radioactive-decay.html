Example 7.4.6. Radioactive Decay.

A radioactive substance undergoes decay, and has mass \(m(t)\) at time \(t\) given by
\begin{equation*} m(t) = m_{0}e^{-kt} \end{equation*}
for some \(k\gt0\text{.}\) The expected lifetime of a particle is given by
\begin{equation*} k\int_{0}^{\infty}te^{-kt}\,dt. \end{equation*}
Find this value.
Solution.
We can solve this using integration by parts. Doing so, we get
\begin{align*} k\int_{0}^{\infty}te^{-kt}\,dt & = -\lim_{b\to\infty}\left[te^{-kt} + \frac{1}{k}e^{-kt}\right]_{0}^{b} \\ & = \frac{1}{k} \end{align*}
