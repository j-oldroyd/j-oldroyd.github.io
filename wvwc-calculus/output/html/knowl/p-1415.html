Suppose that \(f(x,y)\) is defined on a disk \(D\) that contains the point \((a,b)\text{.}\) If \(f_{xy}\) and \(f_{yx}\) are both continuous on \(D\text{,}\) then
\begin{equation*} f_{xy}(a,b) = f_{yx}(a,b). \end{equation*}
In other words, the mixed partials are equal to each other wherever they happen to be continuous.
