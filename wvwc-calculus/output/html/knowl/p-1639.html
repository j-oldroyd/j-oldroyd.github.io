The region of integration is an ellipsoid, but if we can change this to a sphere then we can use spherical coordinates to help us. A quick fix is to set
\begin{equation*} u = x, v = \frac{y}{3} \qq{and} w = \frac{z}{2}\text{.} \end{equation*}
Then these new variables satisfy
\begin{equation*} u^{2} + v^{2} + w^{2} \leq 1 \end{equation*}
and so the corresponding region \(S\) in \(uvw\)-space is the unit sphere.
