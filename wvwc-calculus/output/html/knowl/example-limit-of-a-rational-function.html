Example 2.2.4. Limit of a rational function.

Let
\begin{equation*} f(x) = \frac{3 - x^{5} + 5x}{2x-\sqrt[4]{x}}. \end{equation*}
Evaluate \(\lim_{x\to 16}f(x)\text{.}\)
Solution.
