Example 10.1.5. Parametric Equations for an Ellipse.

Find parametric equations for the ellipse
\begin{equation*} \frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} = 1\text{.} \end{equation*}
Solution.
These equations will look much like the parametric equations for a circle:
\begin{align*} x \amp = a\cos t \\ y \amp = b\sin t \end{align*}
for \(0\leq t\leq 2\pi\text{.}\)
