First, we should find \(L(x)\text{,}\) which from Section 3.7 is just

\begin{equation*} L(x) = 3 + \frac{1}{6}(x-9). \end{equation*}

The error we need to estimate is \(|f(x) - L(x)|\text{,}\) for \(x\) in \([9,16]\text{.}\) So we'll set \(g(x) = f(x) - L(x)\) and let \(x\) be some number in \([9,16]\text{.}\) By Theorem 5.3.3, there exists some number \(c\) in \((9,x)\) such that

\begin{equation*} g(x) - g(9) = g'(c)(x-9). \end{equation*}

Now, \(g(9) = 0\) and \(g'(c) = \frac{1}{2\sqrt{c}} - \frac{1}{6}\text{.}\) So the above equation becomes

\begin{equation*} f(x) - L(x) = \left(\frac{1}{2\sqrt{c}} - \frac{1}{6}\right)(x-9). \end{equation*}
