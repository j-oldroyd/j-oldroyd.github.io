We want to estimate this using a linear approximation, and we want to make sure the linear approximation is easy to set up. This means we want to base the linear approximation at a value of \(a\) that both tangent and its derivative are relatively easy to compute at, and is also close to \(\pi+\frac{3}{49}\text{.}\) So weâ€™ll pick \(a=\pi\) and find the linear approximation to \(\tan x\) at \(a = \pi\text{:}\) Set \(f(x) = \tan x\text{.}\) Then
\begin{align*} L(x) & = f(a) + f^\prime(a)(x-a)\\ & = \tan\pi + \sec^{2}(\pi)(x-\pi) \\ & = x-\pi. \end{align*}
So
\begin{equation*} \tan(\pi+\frac{3}{49}) \approx (\pi+\frac{3}{49}) - \pi = \frac{3}{49}. \end{equation*}
