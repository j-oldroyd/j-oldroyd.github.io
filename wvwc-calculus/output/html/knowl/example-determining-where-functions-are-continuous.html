Example 2.3.8. Determining where functions are continuous.

Let \(h(t) = \sqrt{t} + \frac{t}{t-4} - \frac{t+1}{t^{2}-1}\text{.}\) On what intervals is \(h(t)\) continuous?
Solution.
By TheoremÂ 2.3.7, \(h(t)\) should be continuous wherever \(\sqrt{t}, \frac{t}{t-4}\) and \(\frac{t+1}{t^{2}-1}\) are all defined. Since \(\sqrt{t}\) is defined for \(t\geq0\text{,}\) \(\frac{t}{t-4}\) is defined for \(t\neq4\) and \(\frac{t+1}{t^{2}-1}\) is defined for \(t\neq\pm1\text{,}\) it follows that \(h(t)\) is continuous on \([0,1)\cup(1,4)\cup(4,\infty)\text{.}\)
