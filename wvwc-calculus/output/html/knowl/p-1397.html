We'd like to find the equation of this plane. First, recall that every plane can be described by an equation of the form

\begin{equation*} a(x-x_{0})+b(y-y_{0})+c(z-z_{0}) = 0. \end{equation*}

We can view \(x_{0},y_{0}\) and \(z_{0} = f(x_{0},y_{0})\) as given, so we just need to find \(a,b,c\text{.}\) If we assume that \(c\neq 0\text{,}\) then we can rewrite this equation to obtain

\begin{equation*} z-z_{0} = A(x-x_{0}) + B(y-y_{0}) \end{equation*}

where \(A = -\frac{a}{c}\) and \(B = -\frac{b}{c}\text{.}\) If we set \(y=y_{0}\text{,}\) then we have \(z-z_{0} = A(x-x_{0})\text{.}\) This is the equation of a line tangent to the surface and parallel to the \(x\)-axis, and so the slope of this line must be

\begin{equation*} A = f_{x}(x_{0},y_{0}) \end{equation*}

since the slope of a tangent line in the \(x\)-direction gives the rate of change in the \(x\)-direction. Similarly,

\begin{equation*} B = f_{y}(x_{0},y_{0}). \end{equation*}
