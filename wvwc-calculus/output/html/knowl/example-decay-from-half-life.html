Example 4.4.2. Decay from Half-Life.

A radioactive substance has a half-life of \(28\) days. If we start with a 50 mg sample, how much of the mass will remain after \(t\) days?
Solution.
If we let \(m(t)\) denote the mass of the sample at time \(t\text{,}\) and let \(t=0\) denote the first day we have the sample, then
\begin{equation*} m(t) = m_{0}e^{kt} = 50e^{kt}. \end{equation*}
We still need to find \(k\text{,}\) but we can do this using the fact that the half-life is \(28\) days. This means that
\begin{equation*} m(28) = \frac{m(0)}{2} = 25. \end{equation*}
So
\begin{equation*} 25 = 50e^{28k}\Rightarrow \ln\frac{1}{2} = 28k\Rightarrow k = -\frac{\ln2}{28}. \end{equation*}
Hence
\begin{equation*} m(t) = 50e^{-\frac{\ln2}{28}t} = 50\cdot2^{-\frac{t}{28}}. \end{equation*}
