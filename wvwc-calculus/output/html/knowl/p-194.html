Using the fact that the water is in a conical tank, we can use the formula for the volume of a cone to say that \(V = \frac{1}{3}\pi r^{2}h\text{.}\) If we differentiate both sides with respect to \(t\text{,}\) then we get

\begin{equation*} V' = \frac{1}{3}\pi (2rr'h + r^{2}h'). \end{equation*}

Unfortunately, we don't have any information on \(r\) or \(r'\) that we can use, just information on \(h\) and \(h'\text{.}\) So we need to get \(r\) in terms of \(h\text{.}\) By similar triangles, we can say that \(\frac{r}{h} = \frac{200}{600} = \frac{1}{3}\text{,}\) and so \(r = \frac{h}{3}\) and likewise \(r' = \frac{h'}{3}\text{.}\) Hence

\begin{equation*} V' = \frac{1}{3}\pi (2rr'h + r^{2}h') = \frac{1}{3}\pi \left(2\frac{h^{2}h'}{9} + \frac{h^{2}h'}{9}\right) \end{equation*}

which boils down to

\begin{equation*} V' = \frac{h^{2}h'}{9}\pi = \frac{800000\pi}{9}. \end{equation*}
