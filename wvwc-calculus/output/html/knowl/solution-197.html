Solution 9.4.5.1.

If we take the absolute value of each term, then we get
\begin{equation*} \frac{|\sin(3n - \sqrt{n})|}{3^{n}} \leq \frac{1}{3^{n}}\text{.} \end{equation*}
Since \(\sum_{n=4}^{\infty}3^{-n}\) is a geometric series with \(|r|\lt 1\text{,}\) then
\begin{equation*} \sum_{n=4}^{\infty}\left|\frac{|\sin(3n - \sqrt{n})|}{3^{n}}\right| \end{equation*}
must converge by TheoremÂ 9.3.6.
Hence the original series is absolutely convergent, and so also convergent.
