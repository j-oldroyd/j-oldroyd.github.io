Example 11.7.3. Finding vector functions.

Consider the cylinder \(x^{2}+y^{2}=4\) and the surface \(z=xy\text{,}\) and suppose we want to trace out there intersection with a vector function. Here's how we can do this. First, we'll come up with the \(x\) and \(y\) components of \(\mathbf{r}(t)\text{.}\) Since \(x^{2}+y^{2} = 4\text{,}\) this suggests that we should take

\begin{align*} x & = 2\cos t \\ y & = 2\sin t \end{align*}

So that's two down, one to go. To get \(z\text{,}\) we just need to use the equation \(z=xy\text{.}\) So

\begin{equation*} z = xy = 4\cos t\sin t. \end{equation*}

So our vector function is

\begin{equation*} \mathbf{r}(t) = \dotprod{2\cos t, 2\sin t, 2\sin 2t}. \end{equation*}
