Recall that if \(f(x)\) is a function and \(f^{\prime}(x)\) exists, then \(f^{\prime}(c) > 0\) for some \(c\) means that \(f\) is increasing at \(c\text{,}\) while \(f^{\prime}(c) < 0\) means that \(f\) is decreasing at \(c\text{.}\) Now, suppose that \(c\) is a critical point of \(f(x)\text{.}\) Then \(f^{\prime}(c) = 0\text{.}\) If \(f^{\prime}\) changes sign from negative to positive, then \(f^{\prime}(c)\) is a local minimum. Conversely, if \(f^{\prime}\) changes sign from positive to negative, then \(f^{\prime}(c)\) is a local maximum. This is the first derivative test.
