We'll start with the first coefficient \(c_{0}\text{.}\) We can find it very easily by setting \(x = a\) to get

\begin{equation*} f(a) = c_{0}\text{.} \end{equation*}

So the first coefficient must be equal to \(f(a)\text{.}\) Now we'll try to find \(c_{1}\text{,}\) which we can do by differentiating:

\begin{equation*} f^\prime(x) = c_{1} + 2c_{2}(x - a) + 3c_{3}(x - a)^{2} + \cdots\text{,} \end{equation*}

which is still valid for \((a - R, a + R)\text{.}\) Now set \(x = a\) again to obtain

\begin{equation*} f^\prime(a) = c_{1}\text{.} \end{equation*}
