We need to find \(\int\tan x\,dx\text{.}\) If we rewrite \(\tan x\) in terms of sine and cosine, we get
\begin{equation*} \int\tan x = \int\frac{\sin x}{\cos x}\,dx. \end{equation*}
It's not obvious, but we can solve this using \(u\)-substitution since \(\sin x\) looks a lot like the derivative of \(\cos x\text{.}\) So we'll let \(u = \cos x\text{,}\) giving \(du = -\sin x\,dx\) or just \(dx = -\frac{du}{\sin x}\text{.}\) This simplifies our integral to
\begin{equation*} \int\tan x\,dx = \int-\frac{du}{u} = -\ln|u| + C = -\ln|\cos x|+C. \end{equation*}
Since \(-\ln|\cos x| = \ln|\cos x|^{-1} = \ln|\sec x|\text{,}\) this shows that the general antiderivative of \(\tan x\) is
\begin{equation*} \int \tan x\,dx = \ln|\sec x| + C. \end{equation*}
Note that \(u = \sin x\) would not work in this example, at least not without some nasty algebra.
