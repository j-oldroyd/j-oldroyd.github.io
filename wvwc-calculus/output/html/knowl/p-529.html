To see how the chain rule leads to an integration rule, suppose we have an expression \(F(g(x))\) where \(F,g\) are differentiable. Then the chain rule says that \(\dv{}{x}F(g(x)) = F^{\prime}(g(x))g^{\prime}(x).\) Now, write \(F^{\prime} = f\) and make the substitution \(u = g(x)\text{.}\) Then we get
\begin{equation*} f(g(x))g^{\prime}(x) = f(u)\dv{u}{x}. \end{equation*}
Now we can integrate both sides of this equation with respect to \(x\) to get
\begin{equation*} \int f(g(x))g^{\prime}(x)\,dx = \int f(u)\dv{u}{x}\,dx = \int f(u)\,du. \end{equation*}
This method is known as \(u\)-substitution.
