Itâ€™s not obvious at all that the series is telescoping, even if we write out a few terms. However, if we try partial fractions on \(\frac{3}{n(n-1)}\) we obtain (see SageMath cell below)
\begin{equation*} \frac{3}{n(n-1)} = \frac{3}{n - 1} - \frac{3}{n}\text{.} \end{equation*}
So
\begin{equation*} \sum_{n=3}^{\infty}\frac{3}{n(n-1)} = \sum_{n=3}^{\infty}\left[\frac{3}{n-1} - \frac{3}{n}\right]\text{.} \end{equation*}
The \(k^{\text{th}}\) partial sum is
\begin{equation*} S_{k} = \sum_{n=3}^{k}\left[\frac{3}{n-1} - \frac{3}{n}\right] = \frac{3}{2} - \frac{3}{k}\text{,} \end{equation*}
and so \(S_{k}\to \frac{3}{2}\text{.}\) Hence \(\sum_{n=3}^{\infty}\frac{3}{n(n-1)} = \frac{3}{2}\text{.}\)
