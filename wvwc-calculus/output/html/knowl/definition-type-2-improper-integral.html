Definition 7.4.9. Type 2 Improper Integral.

If \(f(x)\) is continuous on \([a,b)\) and is discontinuous at \(b\text{,}\) then
\begin{equation*} \int_{a}^{b}f(x)\,dx = \lim_{c\to b^{-}}\int_{a}^{c}f(x)\,dx \end{equation*}
assuming this limit exists.
If \(f(x)\) is continuous on \((a,b]\) and is discontinuous at \(a\text{,}\) then
\begin{equation*} \int_{a}^{b}f(x)\,dx = \lim_{c\to a^{+}}\int_{c}^{b}f(x)\,dx \end{equation*}
assuming this limit exists.
If \(f(x)\) is at \(c\) where \(a \lt c\lt b\text{,}\) then
\begin{equation*} \int_{a}^{b}f(x)\,dx = \int_{a}^{c}f(x)\,dx + \int_{c}^{b}f(x)\,dx \end{equation*}
assuming these integrals are convergent.
