Although it’s not obvious, we can actually solve this integral with substitution. First, we’ll separate a power of \(\cos3x\) from the rest in order to act as our \(du\text{:}\)
\begin{equation*} \int_{0}^{\frac{\pi}{6}}\cos^{4}(3x)\cos3x\,dx. \end{equation*}
Now, if we want \(\cos3x\) to act as \(du\) in a substitution, then we need \(u\) to look like \(\sin3x\text{.}\) We can do this by using the Pythagorean Identity and writing \(\cos^{4}(3x) = (1 - \sin^{2}(3x))^{2}\text{.}\)
