Example 11.2.6. Finding components of vectors.

Consider the points \(A(4,0,-2)\) and \(B(4,2,1)\text{.}\) We want to find the components of the vector \(\overrightarrow{AB}\text{.}\) We can do this by translating \(\overrightarrow{AB}\) to the origin, which is done by subtracting from each coordinate of \(B\) the corresponding coordinate of \(A\text{.}\) So the vector \(\overrightarrow{AB}\) is given by
\begin{equation*} \overrightarrow{AB} = \dotprod{4-4,2-0,1+2} = \dotprod{0,2,3}. \end{equation*}
