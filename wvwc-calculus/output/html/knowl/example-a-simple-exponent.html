Example 4.3.7. A simple exponent.

Let \(h(z) = z^{z}\text{.}\) Find \(h'(z)\text{.}\)
Solution.
Weâ€™ll use logarithmic differentiation again to simplify \(h\) and remove the exponent. Set \(y = h(z)\text{,}\) which gives
\begin{equation*} \ln y = z\ln z. \end{equation*}
So
\begin{equation*} \frac{y'}{y} = \ln z + 1, \end{equation*}
which means that
\begin{equation*} y' = y(\ln z + 1) = z^{z}(\ln z + 1). \end{equation*}
