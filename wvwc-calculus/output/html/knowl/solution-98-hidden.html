We won’t be able to solve this integral algebraically until Calculus 2, but we can solve it geometrically instead. If we graph \(y = \sqrt{4-x^{2}}\text{,}\) we get the upper half of the circle of radius \(2\) centered at the origin, as seen in Figure 6.2.8. So the integral given must be exactly one-fourth the area of this circle, so
\begin{equation*} \int_{0}^{2}\sqrt{4-x^{2}}\,dx = \frac{1}{4}\pi(2^{2}) = \pi. \end{equation*}
