Example 3.3.2. Derivatives using the power rule.

Find the derivatives of the following functions:
  1. \(f(x) = x^{7}\text{.}\)
  2. \(g(x) = \frac{1}{\sqrt[5]{x^{11}}}\text{.}\)
  3. \(h(x) = x^{\pi}\text{.}\)
Solution.
The derivative of \(f(x)\) isnâ€™t too hard to find using the power rule, and we quickly get \(f^{\prime}(x) = 7x^{6}\text{.}\) For \(g(x)\text{,}\) first rewrite it as \(g(x) = x^{-\frac{11}{5}}\text{.}\) Then \(g^{\prime}(x) = -\frac{11}{5}x^{-16}{5}\text{.}\) Finally, \(h^\prime(x) = \pi x^{\pi-1}\text{.}\)
