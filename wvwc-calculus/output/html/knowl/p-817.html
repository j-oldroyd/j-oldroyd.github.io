It is occasionally difficult to determine if an ODE of the form
\begin{equation*} \dv{y}{x} = f(x, y) \end{equation*}
is separable or not. Thankfully, there is an algebraic method available that determines whether or not an ODE is separable and, if so, provides a factorization. This method works by first finding some point \((x_0, y_0)\) such that \(f(x_0,y_0) \neq 0\text{.}\) Then we define
\begin{equation} F(x) = \frac{f(x, y_0)}{f(x_0,y_0)}\text{ and }G(y) = f(x_0, y)\text{.}\tag{8.7.1} \end{equation}
If \(f(x,y) = F(x)G(y)\) then the ODE is separable and \(F(x)G(y)\) provides a factorization of \(f(x,y)\text{.}\) If \(f(x,y)\neq F(x)G(y)\) then the equation is not separable.
