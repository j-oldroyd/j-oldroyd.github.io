From the divergence we get another useful form of (second) derivative for scalar functions. First, let \(f(x,y,z)\) be a differentiable scalar function. Then we can compute its gradient \(\grad f\text{.}\) This is a vector field that represents how \(f\) changes. Now, since \(\grad f\) is a vector field we can also consider its curl and divergence. If \(f\) is “nice enough” we know that \(\curl(\grad f) = \vb{0}\text{,}\) which is not particularly useful in this case. But if we take the divergence, we get
\begin{equation*} \div(\grad f) = f_{xx} + f_{yy} + f_{zz}\text{.} \end{equation*}
