Example 14.8.3. Circulation Along a Paraboloid.

Let \(\vb{F} = \dotprod{xz, yz, xyz}\) and let \(S\) denote the paraboloid \(z = x^{2} + y^{2}\) that lies inside the cylinder \(x^{2} + y^{2} = 1\text{,}\) oriented upwards. Use Stokes' Theorem to set up the surface integral equal to \(\oint_{\partial S}\vb{F}\dr\text{.}\)
