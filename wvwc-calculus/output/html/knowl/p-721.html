If the curve were just a line we could easily find its length by using the Pythagorean theorem, but for a more general curve like this we’re stuck. So we’ll start by approximating the curve by using line segments, which we’ll call \(L_1\text{,}\) \(L_2\text{,}\) and so on up to \(L_{n}\text{,}\) then we’ll find the lengths of the line segments. Say the first line segment goes from \(x=x_0\) to \(x=x_1\text{,}\) the second from \(x=x_1\) to \(x=x_2\) and so on. To make life simpler, we’ll assume that all these intervals have equal length \(\Delta x\text{.}\) That is,
\begin{equation*} x_{i} - x_{i-1} = \Delta x \end{equation*}
for \(0 \leq i\leq n\text{.}\) Also, we’ll define \(\Delta y_{i} = f(x_{i}) - f(x_{i-1})\text{.}\)
