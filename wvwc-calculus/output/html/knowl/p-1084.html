We have an idea of how to represent curves in the \(xy\)-plane using parametric equations. What we’d like to do now is to figure out how to do calculus with parametric equations. To start, we’ll find the slope of a parametric curve at some value \(t\text{.}\) Suppose that
\begin{align*} x \amp = f(t) \\ y \amp = g(t) \text{.} \end{align*}
Then we can find the slope \(\dv{y}{x}\) using the chain rule:
\begin{equation*} \dv{y}{t} = \dv{y}{x}\dv{x}{t} \end{equation*}
and so \(\dv{y}{x} = \dv{y}{t}/\dv{x}{t} = g'(t)/f'(t)\text{,}\) assuming that \(g'(t)\neq0\text{.}\)
