Using Definition 6.2.4, we see that the work done should be
\begin{equation*} W = \int_{-1}^{3}3x\,dx. \end{equation*}
Since the definite integral represents area, and the definite integral of \(3x\) exists (since this is continuous), then we can see from Figure 6.2.6 that
\begin{equation*} W = \int_{-1}^{3}3x\,dx = \frac{1}{2}(1)(-3) + \frac{1}{2}(3)(9) = 12. \end{equation*}
Note that regions underneath the \(x\)-axis correspond to negative areas!
