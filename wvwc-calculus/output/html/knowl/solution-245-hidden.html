The constraint is represented as a contour plot of \(g(x,y)\text{.}\) We can compare this with contours of \(f(x,y)\text{,}\) and any point where there two contours intersect is a point for which the constraint is satisfied.
Using these contours, we see that the maximum value of \(f\) subject to \(g = 1\) appears to be \(2\text{,}\) while the minimum value of \(f\) subject to \(g=1\) appears to be \(1\text{.}\)
