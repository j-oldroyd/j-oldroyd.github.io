To minimize this, we find the critical points by solving \(f^{\prime}(x) = 0\text{.}\) This gives, after some algebra, \(x = 40\text{.}\) Now we need to be careful here! We only have a critical point at this step. We need to make an argument now as to why this critical point should be an absolute minimum of \(f(x)\text{.}\) One way to do this is by setting up a sign chart for \(f^{\prime}(x)\text{.}\) If we do this, then we see that \(f' < 0\) for \(x\) in \((0,40)\text{,}\) while \(f' > 0\) for \(x\) in \((40,\infty)\text{.}\) If we think about what this tells us of the graph of \(f(x)\text{,}\) we see that \(x=40\) must minimize \(f(x)\text{,}\) at least for \(x > 0\text{.}\) So the dimensions that minimize the cost of building this box are \(40\times40\times20\text{.}\)
