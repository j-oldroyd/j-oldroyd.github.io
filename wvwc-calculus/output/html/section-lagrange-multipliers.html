    Section 12.8 Lagrange Multipliers

    Theorem¬†12.7.4 is useful for finding local maxima and minima of a function \(f(x,y)\text{,}\) but it's often the case where we have an additional restriction on the variables \(x\) and \(y\text{,}\) which we interpret as an equality \(g(x,y) = k\) for some constant \(k\text{.}\) Geometrically, this reduces down to finding the maximum and minimum values of \(f(x,y)\) on the curve \(g(x,y) = k\text{.}\) This problem can be attacked algebraically using contours, as in the example below.

    Example 12.8.1. Constrained Optimization Using Contours.

    Estimate the minimum and maximum values of \(f(x,y) = x^2 + 2y^2\) subject to the constraint \(g(x,y) = x^2 + y^2 = 1\text{.}\)

    Solution.

    The constraint is represented as a contour plot of \(g(x,y)\text{.}\) We can compare this with contours of \(f(x,y)\text{,}\) and any point where there two contours intersect is a point for which the constraint is satisfied.

    Using these contours, we see that the maximum value of \(f\) subject to \(g = 1\) appears to be \(2\text{,}\) while the minimum value of \(f\) subject to \(g=1\) appears to be \(1\text{.}\)

    Although the graphical approach used in Example¬†12.8.1 and specifically in Figure¬†12.8.2 is useful for estimating solutions of constrained optimization problems in two dimensions, it's neither exact nor easy to visualize in higher dimensions. We therefore want to determine an algebraic approach for solving these problems. In these cases, Theorem¬†12.7.4 is not as useful because there's no guarantee that the critical points of \(f\) will lie on the curve \(g(x,y) = k\text{.}\) For this scenario, we must use the method of Lagrange multipliers.

    Algorithm 12.8.3. Method of Lagrange Multipliers.

    Suppose that \(f\) and \(g\) are both differentiable functions, and suppose that \(\grad g\neq\vb{0}\) on the curve \(g(x,y) = k\text{.}\) To find the extreme values of \(f(x,y)\) subject to the constraint \(g(x,y) = k\text{,}\) perform the following steps:

    1. Find all points \((x_{0},y_{0})\) such that \(\grad f(x_{0},y_{0}) = \lambda \grad g(x_{0},y_{0})\text{,}\) where \(\lambda\) is a constant we call the Lagrange multiplier.

    2. Compute \(f\) at all points from the first step and compare values to find any maxima or minima.

    Example 12.8.4. Maximizing volume.

    A lidless box may be made from 12 m2 of material. What dimensions give the maximum volume?

    Solution.

    If we denote the dimensions of the box by \(x,y,z\text{,}\) then the function we need to maximize is \(V(x,y,z) = xyz\text{.}\) However, not all dimensions are valid since we're only allowed to use 12 m2 of material. This means that the surface area of the box, which is given by \(A(x,y,z) = xy + 2xz + 2yz\text{,}\) must be equal to \(12\text{.}\) So our constraint is that \(A(x,y,z) = 12\text{,}\) and because we have this constraint we'll try to solve this using Algorithm¬†12.8.3.

    To begin, we need to solve \(\grad V = \lambda \grad A\text{.}\) Equivalently, we need to solve the system

    \begin{align*} V_{x} & = \lambda A_{x} \\ V_{y} & = \lambda A_{y} \\ V_{z} & = \lambda A_{z} \end{align*}

    which turns into

    \begin{align*} yz & = \lambda(y + 2z) \\ xz & = \lambda(x + 2z) \\ xy & = \lambda(2x + 2y) \end{align*}

    Now, we don't actually need to find a value for \(\lambda\text{.}\) We just need values for \(x,y,z\text{.}\) If we multiply these equations by \(x\text{,}\) \(y\) and \(z\) respectively, then we get

    \begin{align*} xyz & = \lambda(xy + 2xz) \\ xyz & = \lambda(xy + 2yz) \\ xyz & = \lambda(2xz + 2yz) \end{align*}

    or just

    \begin{equation*} \lambda(xy + 2xz) = \lambda(xy + 2yz) = \lambda(2xz + 2yz) \end{equation*}

    We can also divide by \(\lambda\text{,}\) since there's no way \(\lambda\) could equal \(0\) and satisfy the above equations (since \(x,y,z\) are all positive). This gives us

    \begin{equation} xy + 2xz = xy + 2yz = 2xz + 2yz\tag{7} \end{equation}

    The first two expressions in (7) simplify to \(x=y\text{,}\) while the first and third reduce to \(x = 2z\text{.}\) So

    \begin{equation*} x = y = 2z\text{.} \end{equation*}

    But these variables also need to satisfy \(A(x,y,z) = 12\text{:}\)

    \begin{align*} 12 & = xy + 2xz + 2yz \\ & = 4z^{2} + 4z^{2} + 4z^{2} \\ & = 12z^{2} \end{align*}

    Therefore \(z = \pm1\text{,}\) so we take \(z=1\) (since we're dealing with dimensions). This also means that \(x = y = 2(1) = 2\text{.}\) So the point \((2,2,1)\) is therefore our candidate for the dimensions that maximize volume subject to the constraint \(A = 12\text{.}\)

    In order to actually verify that \(V(2,2,1) = 4\) is actually the largest possible volume we can obtain, we need to show that it's actually a maximum and not a minimum. This means we need to find another point \((x,y,z)\) that also satisfies \(A = 12\text{,}\) and show that it gives a smaller value for \(V\text{.}\) One such point is just \((-2,-2,-1)\text{,}\) and \(V(-2,-2,-1) = -4 < V(2,2,1)\text{.}\) So \(4\) is actually a maximum value and not a minimum value.

    The points we'll obtain using Lagrange multipliers will be either maxima or minima, but the method itself doesn't always tell us which is which. As in the last example, sometimes it's up to us to find that on our own.

    Example 12.8.5. Extreme values on a paraboloid.

    Find the absolute maxima and minima for \(f(x,y) = x^{2} + 2y^{2}\) over this disk \(x^{2} + y^{2} \leq 1\text{.}\)

    Solution.

    First, recall from Section¬†12.7 that the absolute maxima and minima of a differentiable function over a bounded set occur at either a critical point or somewhere on the boundary. So first, we need to find if \(f(x,y)\) has any critical points inside the disk. If we solve \(f_{x} = f_{y} = 0\text{,}\) then we quickly get that \((0,0)\) is a critical point, and hence a possible maximum or minimum value. The only other place we need to check is the boundary \(x^{2} + y^{2} = 1\text{,}\) and this is something we can use Lagrange multipliers for.

    Set \(g(x,y) = x^{2} + y^{2}\text{.}\) Then we need to find the extreme values of \(f\) subject to the constraint \(g = 1\text{.}\) The system of equations we need to solve is then

    \begin{align*} 2x & = \lambda(2x) & \Rightarrow x = \lambda x \\ 4y & = \lambda(2y) & \Rightarrow 2y = \lambda y\\ x^{2} + y^{2} & = 1 \end{align*}

    The first equation is true if \(\lambda = 1\text{,}\) but then this forces \(y = 0\) in the second equation. If we then plug \(y = 0\) into the constraint, this forces \(x = \pm1\text{.}\) So the case \(\lambda=1\) gives us two points to test: \((-1,0)\) and \((1,0)\text{.}\) On the other hand, the second equation is true if \(\lambda = 2\text{.}\) This then forces \(x = 0\) and \(y = \pm 1\text{,}\) which gives us the points \((0,-1)\) and \((0,1)\text{.}\) Finally, if \(\lambda\neq1,2\text{,}\) then this forces \(x = y = 0\text{.}\) But there's no way to satisfy the constraint if \(x = y = 0\text{,}\) so we discard this possibility.

    Example 12.8.7. Maximizing the volume of a prism within an ellipsoid.

    A rectangular prism, centered at the origin and with sides parallel to the coordinate planes, is inscribed within the ellipsoid \(x^{2} + \frac{y^{2}}{4} + \frac{z^{2}}{4} = 1\text{.}\) Find the dimensions of the prism that maximize the volume.

    Solution.

    Let \((x,y,z)\) denote the corner of the prism located in the first octant. Then we want to maximize \(V(x,y,z) = (2x)(2y)(2z)\) subject to the constraint \(g(x,y,z) = x^{2} + \frac{y^{2}}{4} + \frac{z^{2}}{4} = 1\text{.}\) So we'll use the method of Lagrange multipliers again to get the system

    \begin{align*} 8yz & = 2\lambda x \\ 8xz & = \frac{\lambda}{2}y \\ 8xy & = \frac{\lambda}{2}z \\ x^{2} + \frac{y^{2}}{4} + \frac{z^{2}}{4} & =1 \end{align*}

    We can assume that \(x,y,z\) are all positive (since we're trying to find the maximum volume), so we can go ahead and solve each equation for \(\lambda\) to get

    \begin{equation*} \lambda = \frac{4yz}{x} = \frac{16xz}{y} = \frac{16xy}{z} \end{equation*}

    Setting the first two fractions equal and simplifying gives \(y^{2} = 4x^{2}\text{.}\) Similarly, the second and third fractions give \(y^{2} = z^{2}\text{.}\) So

    \begin{equation*} y^{2} = z^{2} = 4x^{2}. \end{equation*}

    Now we plug this into our constraint to get

    \begin{align*} 1 & = x^{2} + \frac{y^{2}}{4} + \frac{z^{2}}{4} = 1 \\ & = 3x^{2} \end{align*}

    so \(x = \pm\sqrt{\frac{1}{3}}\text{.}\) Since we're assuming that \((x,y,z)\) lies in the first octant, this means that \(x > 0\text{,}\) along with \(y \)and \(z\text{.}\) So

    \begin{align*} x & = \sqrt{\frac{1}{3}} \\ y & = 2\sqrt{\frac{1}{3}} \\ z & = 2\sqrt{\frac{1}{3}} \end{align*}

    To check that these values actually maximize \(V(x,y,z)\text{,}\) we can compare them with the point \((1,0,0)\text{,}\) which also satisfies the constraint \(g = 1\text{.}\) Since \(V(1,0,0) = 0\text{,}\) this means that the dimensions that maximize the volume are given by

    \begin{equation*} \frac{2}{\sqrt{3}}\times\frac{4}{\sqrt{3}}\times\frac{4}{\sqrt{3}} \end{equation*}
