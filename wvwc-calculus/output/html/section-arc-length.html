    Section 8.4 Arc Length

    Say the motion of a particle is described by \(y = f(x) \text{ for } a \leq x \leq b.\) Then the path that the particle travels will look like a curve, say

    Our goal now is to determine how far the particle travels as \(x\) goes from \(a\) to \(b\text{.}\) In other words, we want to find the length of this curve from \(x=a\) to \(x=b\text{.}\)

    If the curve were just a line we could easily find its length by using the Pythagorean theorem, but for a more general curve like this we're stuck. So we'll start by approximating the curve by using line segments, which we'll call \(L_1\text{,}\) \(L_2\text{,}\) and so on up to \(L_{n}\text{,}\) then we'll find the lengths of the line segments. Say the first line segment goes from \(x=x_0\) to \(x=x_1\text{,}\) the second from \(x=x_1\) to \(x=x_2\) and so on. To make life simpler, we'll assume that all these intervals have equal length \(\Delta x\text{.}\) That is,

    \begin{equation*} x_{i} - x_{i-1} = \Delta x \end{equation*}

    for \(0 \leq i\leq n\text{.}\) Also, we'll define \(\Delta y_{i} = f(x_{i}) - f(x_{i-1})\text{.}\)

    If we look at \(L_1\text{,}\) then the line goes \(\Delta x\) in the \(x\)-direction and \(\Delta y_1\) in the \(y\)-direction, so by the Pythagorean theorem,

    \begin{equation*} \text{length of } L_1 = \sqrt{(\Delta x)^2 + (\Delta y_1)^2} = \sqrt{1 + \left(\frac{\Delta y_{i}}{\Delta x}\right)}\Delta x. \end{equation*}

    Now if \(\Delta x \approx 0\text{,}\) then \(\frac{\Delta y_{1}}{\Delta x} \approx f'(x_{1})\text{.}\) Hence

    \begin{equation*} \text{length of } L_{1} \approx \sqrt{1 + f'(x_{1})^{2}}\Delta x \end{equation*}

    if \(\Delta x\approx 0\text{.}\)

    We can extend this estimate to the other line segments as well, so we can say that the total length \(L\) of the curve is approximately

    \begin{equation*} \sum_{i=1}^{n}\sqrt{1 + f'(x_{i})^{2}}\Delta x. \end{equation*}

    If the curve is nice enough, this approximation should become exact as \(\Delta x\to 0\text{.}\) So we can take this as the definition of the arc length of the curve \(y = f(x)\text{.}\)

    Definition 8.4.3. Arc Length of a Curve.

    Let \(y = f(x)\) denote a continuous function defined on some interval \([a,b]\text{.}\) The arc length of \(f(x)\) over this interval is defined to be

    \begin{equation*} L = \lim_{\Delta x\to0}\sum_{i=1}^{n}\sqrt{1 + f'(x_{i})^{2}}\Delta x = \int_{a}^{b}\sqrt{1 + f'(x)^{2}}\,dx. \end{equation*}

    Example 8.4.4. The Circumference of a Circle.

    Find the circumference of a circle of radius \(r\text{.}\)

    Solution.

    To find the circumference of this circle, we need to find an equation whose graph is the circle, or at least part of it. One choice is \(y = \sqrt{r^{2} - x^{2}}\) for \(-r\leq x\leq r\text{.}\) Since this gives the top half of a circle, the circumference is then

    \begin{align*} 2\int_{-r}^{r}\sqrt{1 + [y']^{2}}\,dx \amp = 2\int_{-r}^{r}\sqrt{1 + \frac{x^{2}}{r^{2} - x^{2}}}\,dx \\ \amp = 2r\int_{-r}^{r}\frac{1}{\sqrt{r^{2} - x^{2}}}\,dx \\ \amp = 2r\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}}\frac{r\cos\theta}{r\cos\theta}\,d\theta \\ \amp = 2\pi r. \end{align*}

    We could try to use Definition¬†8.4.3 to find the arc length of an ellipse, but it turns out the resulting integral is too complicated to have a "closed form" solution.

    We can use Definition¬†8.4.3 to define the arc length function:

    \begin{equation*} s(x) = \int_{a}^{x}\sqrt{1 + [f'(t)]^{2}}\,dt. \end{equation*}

    This function can be useful to parameterize movement along a curve, which you'll see in Calculus 3.

    Example 8.4.5.

    Find the arc length function for \(y = x^{2}\) assuming that \(x = 0\) is the starting point.

    Solution.

    We have

    \begin{equation*} s(x) = \int_{1}^{x}\sqrt{1 + 4t^{2}}\,dt. \end{equation*}

    To find this, we'll use trigonometric substitution with \(2t = \tan\theta\) to get

    \begin{align*} s(x) \amp = \frac{1}{2}\int_{0}^{\tan^{-1}(2x)}\sqrt{1 + \tan^{2}\theta}\sec^{2}\theta\,d\theta \\ \amp = \frac{1}{4}\left[\sec\theta\tan\theta + \ln(\sec\theta + \tan\theta)\right]_{0}^{\tan^{-1}(2x)} \\ \amp = \frac{x}{2}\sqrt{1 + 4x^{2}} + \frac{1}{4}\ln(\sqrt{1 + 4x^{2}} + 2x). \end{align*}
