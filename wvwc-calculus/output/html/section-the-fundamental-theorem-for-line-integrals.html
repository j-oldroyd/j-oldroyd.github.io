    Section 14.3 The Fundamental Theorem for Line Integrals

    From Theorem¬†6.4.2, we know that integrals and derivatives are closely related:

    \begin{equation*} \int_{a}^{b}\dv{f}{x}\,dx = f(b) - f(a)\text{.} \end{equation*}

    We want to extend this result to line integrals of vector fields over curves. In particular, we want to prove a relationship of the form

    \begin{equation*} \int_{C}\grad f\cdot d\vb{r} = \int_{A}^{B}\grad f\cdot d\vb{r} = f(B) - f(A) \end{equation*}

    where \(A\) and \(B\) are the endpoints of the curve \(C\text{.}\) This is the content of the main result in this section: Theorem¬†14.3.4.

    Subsection Path Independence

    Unless otherwise mentioned, we'll assume that we're always working with an open, simply connected domain \(D\text{.}\) This is a region that, roughly, does not contain its boundary and does not contain any holes. A curve \(C\) contained in \(D\) is closed if its initial and terminal points are the same. We'll denote the boundary of \(D\) by \(\partial D\text{.}\)

    Unless mentioned otherwise, we assume \(\partial D\) is traversed counterclockwise is \(D\) is in the \(xy\)-plane.

    We often denote line integrals over closed curves using the symbol \(\oint\text{.}\)

    The mathematical definition of a conservative vector field is that it's the gradient of some scalar function.

    Definition 14.3.1. Conservative Vector Fields (Mathematical Definition).

    Let \(\vb{F}\) be a vector field. We say that \(\vb{F}\) is conservative on a domain \(D\) if \(\vb{F} = \grad f\) on \(D\) for some scalar function \(f\text{.}\)

    However, we also have a physical definition! Which may be useful. This definition relies on the concept of path independence.

    Definition 14.3.2. Path Independence.

    A vector field \(\vb{F}\) is path independent on a domain \(D\) if

    \begin{equation*} \int_{C_{1}}\vb{F}\dr = \int_{C_{2}}\vb{F}\cdot d\vb{r} \end{equation*}

    for any piecewise smooth paths in \(D\) with the same endpoints. Equivalently, \(\vb{F}\) is path independent if

    \begin{equation*} \oint_{C}\vb{F}\cdot d\vb{r} = 0 \end{equation*}

    for any piecewise smooth closed curve \(C\) in \(D\text{.}\)

    Definition 14.3.3. Conservative Vector Fields (Physical Definition).

    Let \(\vb{F}\) denote a (vector) force field. We say that \(\vb{F}\) is conservative on a domain \(D\) if \(\vb{F}\) is path independent on \(D\text{.}\)

    In physical terms, Definition¬†14.3.3 says that the work done by a conservative force \(\vb{F}\) on a particle moving along a path depends only on the initial and terminal points of the path. Equivalently, the work done on a closed path must always be \(0\text{.}\)

    These two definitions are related by Theorem¬†14.3.4.

    Theorem 14.3.4. Fundamental Theorem of Line Integrals.

    Suppose that \(\vb{F}\) is a conservative vector field (in the sense of Definition¬†14.3.1) with continuous components on an open simply connected region \(D\text{.}\) Then \(\vb{F}\) is path independent. Furthermore, if \(\vb{F} = \grad f\) and \(C\) is a curve in \(D\) with initial point \(A\) and terminal point \(B\text{,}\) then

    \begin{equation*} \int_{C}\vb{F}\cdot d\vb{r} = f(B) - f(A)\text{.} \end{equation*}

    Example 14.3.5. An Awful Example.

    Let \(\vb{F} = \grad(x^{2}y^{3})\) and let \(C\) be the path in the \(xy\)-plane composed of the line segment from \((-1,1)\) to \((0,1)\) followed by the circular arc from \((0,1)\) to \((1,0)\) followed by the logarithmic arc \(y = \ln x\) from \((1,0)\) to \((e,1)\text{.}\) Find \(\int_{C}\vb{F}\cdot\vb{T}\,ds\text{.}\)

    Solution.

    \(C\) is piecewise smooth and consists of the components

    \begin{align*} C_{1} \amp : \vb{r}_{1}(t) = \dotprod{t - 1, 1}, 0\leq t\leq 1 \\ C_{2} \amp : \vb{r}_{2}(t) = \dotprod{\sin t, \cos t}, 0\leq t\leq \frac{\pi}{2} \\ C_{3} \amp : \vb{r}_{3}(t) = \dotprod{t, \ln t}, 1\leq t\leq e \text{.} \end{align*}

    To find \(\int_{C}\vb{F}\cdot\vb{T}\,ds\text{,}\) we can find \(\int_{C_{i}}\vb{F}\cdot\vb{T}\,ds\) for \(1\leq i\leq 3\) and add the resulting values. So let's do that!

    \begin{align*} \int_{C_{1}}\vb{F}\cdot d\vb{r} \amp = \int_{C_{1}} \dotprod{2xy^{3}, 3x^{2}y^{2}}\dr\\ \amp = \int_{0}^{1}\dotprod{2(t-1), 3(t-1)^{2}}\cdot\dotprod{1, 0}\,dt \\ \amp = \brackets{t^{2} - 2t}_{0}^{1} \\ \amp = -1 \text{.} \end{align*}

    Similarly,

    \begin{align*} \int_{C_{2}} \vb{F}\dr \amp = \int_{0}^{\frac{\pi}{2}}\dotprod{2\sin t\cos^{3}t, 3\sin^{2}t\cos^{2}t}\cdot\dotprod{\cos t, -\sin t}\,dt \\ \amp = 0 \end{align*}

    and finally

    \begin{align*} \int_{C_{3}}\vb{F}\dr \amp = \int_{1}^{e}\dotprod{2t(\ln t)^{3}, 3t^{2}(\ln t)^{2}}\cdot\dotprod{1, \frac{1}{t}}\,dt \\ \amp = e^{2}\text{.} \end{align*}

    Putting this all together, we get

    \begin{equation*} \int_{C}\vb{F}\dr = e^{2} - 1\text{.} \end{equation*}

    Now let's compare this approach with using Theorem¬†14.3.4. Since \(\vb{F} = \grad(x^{2}y^{3})\text{,}\) \(\vb{F}\) is conservative by definition. Hence Theorem¬†14.3.4 applies, and (setting \(f(x,y) = x^{2}y^{3}\)) we get

    \begin{equation*} \int_{C}\vb{F}\dr = f(e, 1) - f(-1, 1) = e^{2} - 1\text{.} \end{equation*}

    Example 14.3.6. Using Potential to Find Work Done.

    Let \(\vb{F} = \dotprod{xy^{2}, x^{2}y}\) be a force field and let \(C\) denote the parabolic arc \(y = x^{2}\) from \((1,1)\) to \((3,9)\text{.}\) Find the work done by \(\vb{F}\) along \(C\text{.}\)

    Solution.

    The work done is just \(\int_{C}\vb{F}\dr\text{.}\) Note that \(\vb{F} = \grad(\frac{1}{2}x^{2}y^{2})\text{,}\) and so the work done is just

    \begin{equation*} \frac{1}{2}3^{2}9^{2} - \frac{1}{2}1^{2}1^{2} \end{equation*}

    by Theorem¬†14.3.4.

    Subsection Conservative Vector Fields

    Theorem¬†14.3.4 shows that line integrals involving conservative vector fields are straightforward to evaluate if we know a corresponding potential function. So now we want to do two things:

    1. Given a vector field \(\vb{F}\text{,}\) determine if it's conservative.

    1. Given a conservative vector field \(\vb{F}\text{,}\) determine a potential function \(\varphi\text{.}\)

    For the first, we can use our intuition that conservative vector fields shouldn't rotate. So let's assume that \(\vb{F} = \dotprod{P,Q}\) is a (differentiable) vector field on \(\RR^{2}\) and let \((x,y)\) be a point in the plane. We can estimate the rotation, or circulation, of \(\vb{F}\) at \((x,y)\) by constructing a rectangle with length \(\Delta x\) and height \(\Delta y\) at \((x,y)\) and measuring how \(\vb{F}\) flows counterclockwise around the rectangle. If we do so, then we can estimate the circulation along each side as follows:

    \begin{align*} \text{Top:} \amp P(x,y + \Delta y)\cdot(-\Delta x\vb{i})\\ \text{Bottom:} \amp P(x, y)\cdot(\Delta x)\vb{i}\\ \text{Left:} \amp Q(x,y)\cdot(-\Delta y\vb{j})\\ \text{Right:} \amp Q(x + \Delta x,y)\cdot(\Delta y\vb{j})\text{.} \end{align*}

    Therefore the total circulation near \((x,y)\) should be about

    \begin{equation*} \Delta x\vb{i}\brackets{P(x,y) - P(x, y + \Delta y)} + \Delta y\vb{j}\brackets{Q(x + \Delta x, y) - Q(x,y)}\text{.} \end{equation*}

    If we divide by \(\Delta x\Delta y\) to normalize, then we can say that the circulation is about

    \begin{equation*} \pdv{Q}{x} - \pdv{P}{y}\text{.} \end{equation*}

    Theorem 14.3.7. Conservative Vector Fields are Irrotational.

    Let \(\vb{F} = \dotprod{P,Q}\) be a continuously differentiable vector field on an open simply connected region in \(\RR^{2}\text{.}\) Then \(\vb{F}\) is conservative on this region if and only if

    \begin{equation*} \pdv{Q}{x} - \pdv{P}{y} = 0\text{.} \end{equation*}

    Example 14.3.8. Finding a Line Integral.

    Let \(\vb{F} = (3x^{2} + 2y^{2})\vb{i} + (4xy - e^{-y^{2}})\vb{j}\) and let \(C\) denote the ellipse

    \begin{equation*} \frac{(x-1)^{2}}{9} + \frac{(3y + 2)^{2}}{10} = 5\text{.} \end{equation*}

    Find \(\oint_{C}\vb{F}\cdot\vb{T}\,ds\text{.}\)

    Solution.

    We can parameterize \(C\) and then compute \(\oint_{C}\vb{F}\dr\) as in Section¬†14.2, but we'll first check if \(\vb{F}\) is conservative. Since \(\pdv{Q}{x} = 4y = \pdv{P}{y}\text{,}\) it follows that \(\vb{F}\) is conservative. Hence \(\oint_{C}\vb{F}\dr = 0\text{.}\)

    The quantity \(\pdv{Q}{x} - \pdv{P}{y}\) in Theorem¬†14.3.7 is called the (two-dimensional) curl of \(\vb{F}\) and is denoted by \(\curlt \vb{F}\text{.}\) This represents the tendency of \(\vb{F}\) to rotate counterclockwise around a given point. This can be extended to three dimensions using the following formula:

    \begin{equation*} \curlt\vb{F} = \curl\vb{F} \end{equation*}

    where \(\nabla = \dotprod{\pdv{}{x}, \pdv{}{y}, \pdv{}{z}}\text{.}\) The direction of \(\curl\vb{F}\) provides the axis of rotation at a point, and its magnitude is the tendency of \(\vb{F}\) to rotate counterclockwise around this axis of rotation (viewed head on).

    Example 14.3.9. Testing a Vector Field in \(\RR^{3}\).

    Let \(\vb{F} = \dotprod{e^{x}\cos y + yz, xz - e^{x}\sin y, xy + z}\text{.}\) Determine if \(\vb{F}\) is conservative.

    This is taken from Example 2 on page 1165 of Thomas' Calculus, \(11^{\text{th}}\) edition.

    Solution.

    We need to check if \(\curl\vb{F} = \vb{0}\text{.}\) So we compute this like a cross product, giving \(\vb{0}\text{.}\) Hence the vector field is irrotational and therefore conservative.

    So now we have a good test for if a vector field is conservative. Next, we want to be able to find a corresponding potential function to a conservative vector field.

    Consider the vector field \(\vb{F}\) from Example¬†14.3.9. We know this is conservative, so there must exist a corresponding potential function \(f(x,y,z)\) such that \(\grad f = \vb{F}\text{.}\) To find this, we start by noting that whatever \(f\) is, its partial derivatives must be the components of \(\vb{F}\text{.}\) In particular,

    \begin{equation*} \pdv{f}{x} = e^{x}\cos y + yz \implies f = e^{x}\cos y + xyz + g(y,z)\text{.} \end{equation*}

    Now we look at \(\pdv{f}{y}\) to pin down \(g(y,z)\text{:}\)

    \begin{equation*} \pdv{f}{y} = xz - e^{x}\sin y = -e^{x}\sin y + xz + \pdv{g}{y} \implies \pdv{g}{y} = 0\text{,} \end{equation*}

    and so \(g(y,z) = g(z)\text{.}\) Finally,

    \begin{equation*} \pdv{f}{z} = xy + z = xy + g'(z) \implies g'(z) = z \end{equation*}

    and so \(g(z) = \frac{z^{2}}{2} + C\text{.}\) Therefore a potential function for \(\vb{F}\) is

    \begin{equation*} f(x,y,z) = e^{x}\cos y + xyz + \frac{z^{2}}{2}\text{.} \end{equation*}

    Example 14.3.10. Line Integral Along an Elliptical Arc.

    Compute \(\int_{C}(1 + xy)e^{xy}\,dx + x^{2}e^{xy}\,dy\text{,}\) where

    \begin{equation*} C: \vb{r}(t) = \cos t\vb{i} + 2\sin t\vb{j}, 0\leq t\leq \frac{\pi}{2}\text{.} \end{equation*}
    Solution.

    First, we'll check if \(\vb{F}\) is conservative. If it is, we can use Theorem¬†14.3.4. Since \(\pdv{Q}{x} - \pdv{P}{y} = 2xe^{xy} + x^{2}ye^{xy} - \brackets{xe^{xy} + \parens{x + x^{2}y}e^{xy}} = 0\text{,}\) we see that \(\vb{F}\) is in fact conservative.

    Now we need to find a potential function \(f(x,y)\text{.}\) Since \(f_{x} = (1 + xy)e^{xy}\text{,}\) we can integrate with respect to \(x\) to get

    \begin{equation*} f(x,y) = xe^{xy} + g(y)\text{.} \end{equation*}

    Now differentiate with respect to \(y\) to get

    \begin{equation*} x^{2}e^{xy} = x^{2}e^{xy} + g^{\prime}(y) \implies g(y) = C\text{.} \end{equation*}

    Hence a potential function for \(\vb{F}\) is \(f = xe^{xy}\text{,}\) and so

    \begin{equation*} \int_{C}(1 + xy)e^{xy}\,dx + x^{2}e^{xy}\,dy = -1\text{.} \end{equation*}
