Section 5.6 Antiderivatives

Suppose we're tracking a moving object, and through experimentation we know its acceleration to be equal to 3 m⁄s2 to the right. Say we want to find the velocity of the mass at time \(t\text{.}\) Then if we set \(v(t)\) equal to the velocity and \(a(t)\) equal to the acceleration, we know that

\begin{equation*} v'(t) = a(t) = 3. \end{equation*}

Therefore \(v(t)\) is a function whose derivative is \(3\text{.}\) So one possible choice for \(v(t)\) is \(v(t) = 3t\text{.}\) This leads us to the definition of an antiderivative.

Definition 5.6.1. Antiderivatives.

A function \(F(x)\) is called an antiderivative of the function \(f(x)\) if \(F'(x) = f(x)\) for all \(x\) in an interval \(I\text{.}\)

Example 5.6.2. Multiple antiderivatives.

Find three different antiderivatives for \(f(x) = \cos x\text{.}\)

Solution.

An antiderivative of \(\cos x\) is any function whose derivative is \(\cos x\text{.}\) So three antiderivatives could be

\begin{equation*} F_{1}(x) = \sin x, F_{2}(x) = \sin x + 4, F_{3}(x) = \sin x - 100. \end{equation*}

From Example 5.6.2 we see that any function of the form \(\sin x +C\) is an antiderivative of \(\cos x\text{.}\) In fact, this describes all possible antiderivatives of \(\cos x\text{.}\) This suggests the following theorem.

Theorem 5.6.3. General Antiderivatives.

Let \(f(x)\) be some function with antiderivative \(F(x)\) on some interval \(I\text{.}\) Then the most general antiderivative of \(f(x)\) is

\begin{equation*} F(x) + C \end{equation*}

where \(C\) is an arbitrary constant.

Proof.

It might not seem like this statement requires a proof, but it does! We can check very easily that every function of the form \(F(x)+C\) is an antiderivative of \(f(x)\text{,}\) but how do we know every antiderivative takes this form? To prove this, let \(G(x)\) be an arbitrary antiderivative of \(f(x)\text{.}\) Then we need to show that \(G(x) = F(x) + C\text{,}\) or equivalently that \(G(x) - F(x) = C\text{.}\) We can do this by taking the derivative of \(G(x) - F(x)\) to get

\begin{equation*} G'(x) - F'(x) = f(x) - f(x) = 0, \end{equation*}

and so by Theorem 5.2.5 we know that \(G(x) - F(x)\) must be constant on \(I\text{.}\) Hence \(G(x) - F(x) = C\) for some constant \(C\text{,}\) or equivalently \(G(x) = F(x)+C\text{.}\) Therefore every antiderivative of \(f(x)\) can be written as \(F(x) + C\text{.}\)

It might seem superfluous, but typically when dealing with models requiring us to find antiderivatives we'll want to find the most general antiderivative somewhere along the way. Moral of the story: don't forget to add \(C\text{.}\)

Example 5.6.4. Finding antiderivatives.

Find the most general antiderivatives of the following functions:

  1. \(\displaystyle x^{3}.\)

  2. \(\displaystyle -\frac{1}{2}x^{\frac{5}{3}}.\)

  3. \(\displaystyle \sqrt{x} - 3x^{3} + x^{-2}.\)

  4. \(\displaystyle \sin x + \cos (5x) + e^{4x}.\)

Solution.

We have the following:

  1. The general antiderivative of \(x^{3}\) is \(\frac{1}{4}x^{4}+C.\)

  2. The general antiderivative of \(-\frac{1}{2}x^{\frac{5}{3}}\) is \(-\frac{1}{2}\frac{3}{8}x^{\frac{8}{3}}+C.\)

  3. The general antiderivative of \(\sqrt{x} - 3x^{3} + x^{-2}\) is \(\frac{2}{3}x^{\frac{3}{2}} - \frac{3}{4}x^{4} - \frac{1}{x} + C\text{.}\)

  4. The general antiderivative of \(\sin x + \cos (5x) + e^{4x}\) is \(-\cos x + \frac{1}{5}\sin(5x) + \frac{1}{4}e^{4x} + C.\)

Example 5.6.5. The antiderivative of \(\frac{1}{x}\).

Find the general antiderivative of \(f(x) = \frac{1}{x}\text{.}\)

Solution.

We have two cases we need to consider, since the domain of \(f(x)\) consists of two intervals. First, suppose that \(x\) is in \((0,\infty)\text{.}\) Then we know that \(\dv{}{x}\ln x = \frac{1}{x}\text{,}\) so on this interval the most general antiderivative of \(\frac{1}{x}\) is \(\ln x + C_{1}\text{.}\)

Now let \(x\) be in \((-\infty,0)\text{.}\) Then \(\ln x\) isn't defined here. However, we can write

\begin{equation*} \frac{1}{x} = -\frac{1}{-x}, \end{equation*}

and if \(x < 0\) then \(-x > 0\) and \(\ln(-x)\) is defined, and in fact

\begin{equation*} \dv{}{x}\ln(-x) = -\frac{1}{(-x)} = \frac{1}{x}. \end{equation*}

So the most general antiderivative of \(\frac{1}{x}\) on \((-\infty,0)\) is \(\ln(-x) + C_{2}\text{.}\) Putting this all together, we can say that the most general antiderivative of \(f(x) = \frac{1}{x}\) is given by

\begin{equation*} F(x) = \begin{cases} \ln x + C_{1} & x > 0 \\ \ln(-x) + C_{2} & x < 0\end{cases}. \end{equation*}

If we know that we're only selecting values of \(x\) with the same sign, then we can just say that the antiderivative of \(\frac{1}{x}\) is \(\ln|x| + C\text{.}\)

Example 5.6.6. Antiderivatives involving secant.

Find the general antiderivatives of \(\sec^{2}t\) and \(-2\sec3t\tan3t\text{.}\)

Solution.

The first antiderivative isn't too hard; it's just \(\tan t+C\text{.}\) The second is a little more complicated but still not too bad. Since \(\dv{}{t}\sec t = \sec t\tan t\text{,}\) a good guess for an antiderivative of \(-2\sec3t\tan3t\) would be \(-2\sec3t.\) However, the derivative of this is \(-6\sec3t\tan3t\text{,}\) so we're off by a factor of \(3\text{.}\) So we just need to divide our guess by \(3\) to correct for this. Hence the general antiderivative of \(-2\sec3t\tan3t\) is \(-\frac{2}{3}\sec3t\text{.}\)

Example 5.6.9. Falling objects.

An object is dropped from a height of 100 m above sea level and falls with downward acceleration equal to 9.8 m⁄s2. Find the height \(h(t)\) of the object \(t\) seconds after it's dropped.

Solution.

Take downward to be the negative direction and sea level to be \(h=0\text{.}\) Since acceleration is the second derivative of position, then the position (i.e. height) of the object should be the second antiderivative of acceleration. At this step, it's tempting to say that since \(h''(t) = -9.8\text{,}\) we have

\begin{equation*} h'(t) = -9.8t\quad\text{and}\quad h(t) = -4.9 t^{2}\text{.} \end{equation*}

And indeed, \(-4.9t^{2}\) is a function whose second derivative is \(-9.8\text{.}\) However, we have a slight problem here. We know that \(h(0) = 100\) and \(h'(0) = 0\text{.}\) If \(h(t) = -4.9t^{2}\text{,}\) then it's impossible for \(h(0)\) to equal \(100\text{.}\) The problem here is we forgot about the arbitrary constants.

To get an accurate expression for \(h(t)\text{,}\) we go back to \(h''(t) = -9.8\text{.}\) Then \(h'(t) = -9.8t+C_{1}\) for some constant \(C_{1}\text{.}\) Since \(h'(0) = 0\text{,}\) this forces \(C_{1} = 0\text{.}\) So we have \(h'(t) = -9.8t\text{.}\) Now we antidifferentiate one more time to get \(h(t) = -4.9t^{2} + C_{2}\text{.}\) Since \(h(0) = 100\text{,}\) this forces \(C_{2} = 100\text{.}\) So

\begin{equation*} h(t) = -4.9t^{2} + 100\text{.} \end{equation*}

Some functions that do not have an obvious antiderivative can be simplified through algebra into a form that is perhaps more helpful.

Example 5.6.10. A tricky antiderivative.

Find the most general antiderivative of

\begin{equation*} f(x) = \frac{2+x^{2}}{1+x^{2}}. \end{equation*}
Solution.

It's tough to think of a function whose derivative is \(f(x)\text{,}\) though since the denominator is \(1+x^{2}\) it seems likely that this antiderivative will involve \(\tan^{-1}x\) in some way. In order to actually find the antiderivative, we'll rewrite \(f(x)\) into a more convenient form. First, note that the numerator is very close to the denominator, which means we can almost cancel it out. So we'll split up the numerator as follows:

\begin{equation*} \frac{2+x^{2}}{1+x^{2}} = \frac{1+(1+x^{2})}{1+x^{2}} = \frac{1}{1+x^{2}} + 1. \end{equation*}

We can find the antiderivative of this term by term, so the most general antiderivative of \(f(x)\) is

\begin{equation*} F(x) = \tan^{-1}x + x + C. \end{equation*}

At this point, it's natural to think about what an antiderivative means geometrically since the derivative of a function has such a nice interpretation. One approach to this is to think about net change of the antiderivative. Let \(F(x)\) be an antiderivative of \(f(x)\) on some interval containing \(a,b\text{.}\) The net change of \(F(x)\) from \(x=a\) to \(x=b\) is given by

\begin{equation*} F(b) - F(a). \end{equation*}

Now, \(F'(x) = f(x)\) represents instantaneous rate of change, not net change. However, if we can somehow add these instantaneous changes from \(x=a\) to \(x=b\text{,}\) they should accumulate until they give the net change of \(F\) from \(a\) to \(b\text{.}\) Graphically, adding these instantaneous rates of change looks like it's giving us the area under \(f(x)\text{,}\) which suggests the following geometric interpretation: the area under of \(f(x)\) from \(x=a\) to \(x=b\) is just the net change of an antiderivative \(F(x)\) from \(x=a\) to \(x=b\text{.}\) Much of the next chapter is devoted to making this idea precise.
