Section 9.2 Vectors

¶

One of our goals in this chapter is to adequately describe motion in space. A useful way to do this uses the concept of vector, which we think of as a quantity that has both direction and magnitude/length. A simple example would be velocity: velocity in space has a direction and also a magnitude (speed). We will typically denote vectors by using boldface letters such as \(\vv{x}\) or letters with a bar overhead such as \(\bar{x}\text{.}\) We represent vectors as arrows with an initial point and a terminal point:

We say that two vectors \(\vv{x},\vv{y}\) are equivalent, or equal, if they have the same magnitude and direction. We write this as \(\vv{x} = \vv{y}\text{.}\)

Subsection 9.2.1 Addition and Scalar Multiplication

¶

Given two vectors \(\vv{x} = \overrightarrow{AB},\vv{y}=\overrightarrow{BC}\text{,}\) we can add them to get the new vector

\begin{equation*} \vv{x}+\vv{y} = \overrightarrow{AC}. \end{equation*}

So the vector \(\vv{x}+\vv{y}\) is obtained by moving the tail of \(\vv{y}\) to the tip of \(\vv{x}\) and then drawing a vector from the tail of \(\vv{x}\) to the tip of \(\vv{y}\text{.}\)

The sum \(\vv{x}+\vv{y}\) of vectors \(\vv{x},\vv{y}\) can be computed using either the triangle law, illustrated above in Figure 9.2.2, or the similar parallelogram law. We can also scale vectors using scalar multiplication: if \(\alpha\) is a scalar 1 In other words, a number. then \(\alpha\vv{x}\) is defined to be the vector that has the same direction as \(\vv{x}\) if \(\alpha>0\) and the opposite direction if \(\alpha<0\text{,}\) but the magnitude is rescaled by the factor \(\alpha\text{.}\)

Example 9.2.3 Vector subtraction

Using the previous graph, we can compute \(2\vv{x}-\vv{y}\text{.}\) We just need to scale the vectors properly and then add \(2\vv{x}\) to \(-\vv{y}\text{.}\)

Subsection 9.2.2 Vector Components

¶

Although it isn't too hard to add and scale vectors visually, it'll be beneficial to do the same algebraically. We can do so by breaking a vector down into its components. Consider a vector in \(\vv{x}\) in \(\RR^{2}\text{,}\) and suppose if we move it to the origin then the tip of the vector is at the point \((3,-1)\text{.}\) Then the components of \(\vv{x}\) are \(3\) and \(-1\text{,}\) and we write

\begin{equation*} \vv{x} = \dotprod{3,-1}. \end{equation*}

Note the use of brackets here, since technically we are saying that the vector \(\vv{x}\) is distinct from the point \((3,-1)\text{,}\) even though they are closely related. We say that the position vector of a point is the vector whose components are the same as the coordinates of the point. Geometrically, the position vector of a point \(P\) is the vector with its tail at the origin and its tip at \(P\text{.}\)

So any vector in \(\RR^{2}\) can be represented using components by \(\vv{x} = \dotprod{x_{1},x_{2}}\text{.}\) Similarly, any vector in \(\RR^{3}\) can be represented as \(\vv{x} = \dotprod{x_{1},x_{2},x_{3}}\text{.}\) Once you represent a vector in component form, addition and scalar multiplication is straightforward.

Example 9.2.4 Vector addition with components

Let \(\vv{a} = \dotprod{-2,1,3}\) and \(\vv{b} = \dotprod{0,2,1}\text{.}\) Then

\begin{equation*} \vv{b}-3\vv{a} = \dotprod{6,-1,-8}. \end{equation*}

Finding magnitudes of vectors can also be done by applying the distance formula from Section 9.1 to the components of the vectors. The magnitude of a vector \(\vv{x}\) is denoted by \(|\vv{x}|\) or \(\|\vv{x}\|\text{.}\) For example, the magnitude of \(\vv{b}-3\vv{a}\) from the previous example is

\begin{equation*} \|\vv{b}-3\vv{a}\| = \sqrt{36+1+64} = \sqrt{101}. \end{equation*}

Given a vector \(\vv{x} = \dotprod{x_{1},x_{2},x_{3}},\)

\begin{equation*} \|\vv{x}\| = \sqrt{x_{1}^{2}+x_{2}^{2}+x_{3}^{2}}. \end{equation*}
Example 9.2.5 A vector equation

We can use vectors to describe curves and surfaces. For example, let \(\vv{r} = \dotprod{x,y,z}\) and \(\vv{r}_{0} = \dotprod{a,b,c}\text{.}\) Let \(r\geq0\text{.}\) Then the set of all points \((x,y,z)\) that satisfy the equation

\begin{equation*} \|\vv{r}-\vv{r}_{0}\| = r \end{equation*}

has a very nice description: it's just the sphere of radius \(r\) centered at \((a,b,c)\text{.}\)

Example 9.2.6 Finding components of vectors

Consider the points \(A(4,0,-2)\) and \(B(4,2,1)\text{.}\) We want to find the components of the vector \(\overrightarrow{AB}\text{.}\) We can do this by translating \(\overrightarrow{AB}\) to the origin, which is done by subtracting from each coordinate of \(B\) the corresponding coordinate of \(A\text{.}\) So the vector \(\overrightarrow{AB}\) is given by

\begin{equation*} \overrightarrow{AB} = \dotprod{4-4,2-0,1+2} = \dotprod{0,2,3}. \end{equation*}

In general, given \(A(x_{1},y_{1},z_{1})\) and \(B(x_{2},y_{2},z_{2}),\) the vector \(\vv{x} = \overrightarrow{AB}\) is given by

\begin{equation*} \vv{x} = \dotprod{x_{2}-x_{1},y_{2}-y_{1},z_{2}-z_{1}}. \end{equation*}
Theorem 9.2.7 Properties of Vector Addition and Scalar Multiplication

Let \(\vv{x},\vv{y},\vv{z}\) be vectors and let \(\alpha,\beta\) be scalars. Then the following are true:

  1. \(\vv{x}+\vv{y} = \vv{y}+\vv{x}\text{.}\)

  2. \(\vv{x}+(\vv{y}+\vv{z}) = (\vv{x}+\vv{y})+\vv{z}.\)

  3. \(\vv{x}+\vv{0} = \vv{x}.\)

  4. \(\vv{x}+(-\vv{x}) = \vv{0}.\)

  5. \(\alpha(\vv{x}+\vv{y}) = \alpha\vv{x}+\alpha\vv{y}.\)

  6. \((\alpha+\beta)\vv{x} = \alpha\vv{x}+\beta\vv{y}\)

  7. \((\alpha\beta)\vv{x} = \alpha(\beta)\vv{x}.\)

  8. \(1\vv{x} = \vv{x}.\)

Subsection 9.2.3 Basis Vectors and Unit Vectors

¶

Every vector in \(\RR^{3}\) can be written using three components: \(\vv{x} = \dotprod{x_{1},x_{2},x_{3}}.\) Each component corresponds to a coordinate axis, and we can rewrite \(\vv{x}\) as a linear combination of three different vectors, with each vector corresponding to a coordinate axis:

\begin{equation*} \vv{x} = x_{1}\dotprod{1,0,0}+x_{2}\dotprod{0,1,0}+x_{3}\dotprod{0,0,1}. \end{equation*}
standard basis vectors
Definition 9.2.8 Standard Basis Vectors

The standard basis for \(\RR^{3}\) is the set \(\{\vv{i},\vv{j},\vv{k}\}\text{,}\) where

\begin{equation*} \vv{i} = \dotprod{1,0,0},\quad\vv{j} = \dotprod{0,1,0}\quad\text{and}\quad\vv{k} = \dotprod{0,0,1}. \end{equation*}

As we've seen, every vector \(\vv{x}\) in \(\RR^{3}\) can be expressed using only these three vectors. The standard basis has two important properties: it is perpendicular (also called orthogonal) and every vector in the collection has magnitude \(1\text{.}\) In other words,

\begin{equation*} \|\vv{i}\| = \|\vv{j}\| = \|\vv{k}\| = 1. \end{equation*}

These vectors are essentially designed to capture the "coordinate directions", and are plotted below.

This also leads us to our next definition.

Definition 9.2.10 Unit Vectors

A vector \(\vv{x}\) is a unit vector if \(\|\vv{x}\| = 1\text{.}\)

Unit vectors are useful if we just need to indicate a direction, and we don't care about magnitude. Every nonzero vector can be rescaled to a unit vector: just divide the vector by its norm.

Example 9.2.11 Direction from one point to another

Consider the points \(A(1,2,-1)\) and \(B(0,-3,2)\text{.}\) Then we can find the unit vector indicating the direction from \(A\) to \(B\text{.}\) First, set

\begin{equation*} \vv{x} = \overrightarrow{AB} = \dotprod{-1,-5,3}. \end{equation*}

Then the unit vector that gives the direction from \(A\) to \(B\) is given by

\begin{align*} \vv{u} & = \frac{\vv{x}}{\|\vv{x}\|} \\ & = \frac{\dotprod{-1,-5,3}}{\sqrt{1+25+9}} \\ & = \dotprod{-\frac{1}{\sqrt{37}}, -\frac{5}{\sqrt{37}}, \frac{3}{\sqrt{37}}}. \end{align*}
Example 9.2.12 A vector equation for the unit sphere

Using the concept of a unit vector, we can very easily describe the unit sphere 2 The sphere of radius \(1\) centered at the origin. using a vector equation. If we set \(\vv{r} = (x,y,z)\text{,}\) then the unit sphere is just the set of all solutions of

\begin{equation*} \|\vv{r}\| = 1. \end{equation*}

Subsection 9.2.4 Applications

¶

Many physical quantities have both a direction and a magnitude, like velocity, acceleration and forces. Vectors are ideally suited to measure these quantities.

Example 9.2.13 Weight of a chain

A still chain is fixed to two ends of a level divide. The tension of the chain at each fixed end can be represented by vectors pointing away from the chain. Call these tension forces \(\vv{T}_{1}\) and \(\vv{T}_{2}\text{.}\) Suppose we know that each force vector makes an angle of \(45^{\circ}\) with the ground on either side of the chain's fixed ends, and that the magnitude of each tension is 43 N. Then we can use vector addition to find the weight of the chain.

Let \(\vv{w}\) denote the weight of the chain considered as a vector (so that it's pointing down). Since the chain is still, its resultant 3 The sum of all forces acting on the chain. must be \(\vv{0}\text{.}\) So we can say that

\begin{equation*} \vv{T}_{1}+\vv{T}_{2} + \vv{w} = \vv{0} \end{equation*}

or in other words

\begin{equation*} \vv{w} = -\vv{T}_{1}-\vv{T}_{2}. \end{equation*}

What we need to find is \(\|\vv{w}\|\text{,}\) which we can do without too much trouble if we can rewrite the tension vectors in component form. In fact, we have

\begin{align*} \vv{T}_{1} & = \dotprod{-\|\vv{T}_{1}\|\cos45^{\circ},\|\vv{T}_{1}\|\sin45^{\circ}} \\ \vv{T}_{2} & = \dotprod{\|\vv{T}_{2}\|\cos45^{\circ},\|\vv{T}_{2}\|\sin45^{\circ}} \end{align*}

So it follows that

\begin{align*} \|\vv{w}\| & = \|-\vv{T}_{1}-\vv{T}_{2}\| \\ & = \left\|\dotprod{0,-2\cdot43\frac{\sqrt{2}}{2}}\right\|\\ & = 43\sqrt{2} \end{align*}

Therefore the chain weighs \(43\sqrt{2}\)N.
