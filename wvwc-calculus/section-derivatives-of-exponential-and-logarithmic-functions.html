Section 3.3 Derivatives of Exponential and Logarithmic Functions

¶

Subsection 3.3.1 Derivatives of Exponentials

¶

We've already mentioned that \(\dv{}{x}e^{x} = e^{x}\text{,}\) but we've said nothing about differentiating functions such as \(3^{x}\) or \((\frac{1}{\pi})^{-x}\text{.}\) However, it turns out that we can get these derivatives from the derivative of \(e^{x}\) without too much trouble.

Theorem 3.3.1 Derivatives of Exponential Functions

Let \(a > 0\) and set \(f(x) = a^{x}\text{.}\) Then \(f'(x) = a^{x}\ln a\text{.}\)

Proof

To prove this, we need to use what's available to us: the derivative of \(e^{x}\text{.}\) From Example 3.2.11 we say that \(e^{x\ln 3} = 3^{x}\text{.}\) More generally, we can also say that \(e^{x\ln a} = a^{x}\text{.}\) Hence

\begin{align*} \dv{}{x}a^{x} & = \dv{}{x}e^{x\ln a} \\ & = (\ln a)e^{x\ln a} \\ & = (\ln a)a^{x}. \end{align*}
Example 3.3.2 Differentiating an exponential

Let \(g(t) = 2.6^{3t - \sin t}\text{.}\) Find \(g'(t)\text{.}\)

Solution

If we set \(u = 3t - \sin t\text{,}\) then \(g(t) = g(u) = 2.6^{u}\text{.}\) By the chain rule, \(g'(t) = g'(u)\dv{u}{t} = (\ln2.6)2.6^{u}(3-\cos t) = (\ln2.6)2.6^{3t-\sin t}(3-\cos t).\)

Although we certainly want to know the derivative of \(a^{x}\text{,}\) or at least how to find it, most applications involving exponential functions use the natural exponential function \(e^{x}\) instead. The derivative of \(e^{x}\) is probably the most important derivative in this course.

Example 3.3.3 Solutions of a differential equation

Show that \(x(t) = Ae^{-t} + Bte^{-t}\) satisfies the differential equation \(x'' + 2x' + x = 0\text{,}\) where \(A,B\) are arbitrary constants.

Solution

We need to show that if we compute \(x'\) and \(x''\) and plug these expressions into the differential equation, this will simplify out to \(0\text{.}\) Since

\begin{align*} x' & = -Ae^{-t} +Be^{-t} - Bte^{-t} \\ x'' & = Ae^{-t} - Be^{-t} - (Be^{-t} - Bte^{-t}) \\ & = Ae^{-t} - 2Be^{-t} + Bte^{-t}, \end{align*}

it follows that

\begin{align*} x'' + 2x' + x & = (Ae^{-t} - 2Be^{-t} + Bte^{-t}) + 2(-Ae^{-t} + Be^{-t} - Bte^{-r}) + Ae^{-t} + Bte^{-t} \\ & = 0. \end{align*}

Exponential functions often appear in the solutions of differential equations, which are themselves often viewed as mathematical models of physical systems. Hence exponential functions play a significant role in predicting physical quantities, which goes a long way towards justifying their importance.

Subsection 3.3.2 Derivatives of Logarithms

¶

Just as we can get the derivative of every exponential function \(a^{x}\) just by knowing the derivative of \(e^{x}\text{,}\) we can get the derivative of every logarithmic function \(\log_{a}x\) just by knowing that \(\dv{}{x}\ln x = \frac{1}{x}\text{.}\)

Theorem 3.3.4 Derivatives of Logarithmic Functions

Let \(a > 0, a\neq 1\) and set \(f(x) = \log_{a}x\text{.}\) Then

\begin{equation*} f'(x) = \frac{1}{x\ln a}. \end{equation*}
Example 3.3.5 Differentiating nested logarithms

Let \(f(x) = \log_{2}(\log_{5} x)\text{.}\) Find \(f'(x)\text{.}\)

Solution

By the chain rule, we have

\begin{equation*} f'(x) = \frac{1}{(\log_{5}x)\ln 2}\dv{}{x}\log_{5}x = \frac{1}{(\log_{5} x)\ln 2}\frac{1}{x\ln 5}, \end{equation*}

Logarithms can be used to greatly simplify derivatives involving products, division or exponentiation using a technique known as logarithmic differentiation.

Example 3.3.6 A simple fraction

Let \(f(x) = \frac{e^{-x}\cos x}{x^{2} + 4x + 4}\text{.}\) Find \(f'(x)\text{.}\)

Solution

We can find \(f'(x)\) without resorting to logarithms, but this would require using the product, quotient and chain rules. The algebra would be awful. So we'll using logarithms instead! Set \(y = f(x)\text{.}\) Then

\begin{equation*} \ln y = \ln e^{-x} + \ln\cos x - \ln (x^{2} + 4x + 4) = -x + \ln\cos x - 2\ln(x+2). \end{equation*}

Now we differentiate both sides implicitly to obtain

\begin{equation*} \frac{y'}{y} = -1 - \frac{\sin x}{\cos x} - \frac{2}{x+2}. \end{equation*}

Hence

\begin{equation*} y' = y\left(-1 - \tan x - \frac{2}{x+2}\right) = \frac{e^{-x}\cos x}{x^{2} + 4x + 4}\left(-1 - \tan x - \frac{2}{x+2}\right). \end{equation*}

Logarithmic differentiation is useful for finding derivatives of expressions containing complicated quotients, products or powers.

Example 3.3.7 A simple exponent

Let \(h(z) = z^{z}\text{.}\) Find \(h'(z)\text{.}\)

Solution

We'll use logarithmic differentiation again to simplify \(h\) and remove the exponent. Set \(y = h(z)\text{,}\) which gives

\begin{equation*} \ln y = z\ln z. \end{equation*}

So

\begin{equation*} \frac{y'}{y} = \ln z + 1, \end{equation*}

which means that

\begin{equation*} y' = y(\ln z + 1) = z^{z}(\ln z + 1). \end{equation*}
