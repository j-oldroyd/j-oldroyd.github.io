Section 11.1 Functions of Several Independent Variables

¶

Now we need to consider calculus with functions of more than one independent variable.

Subsection 11.1.1 Functions of Two Variables

¶

A function of several variables is a function that depends on more than one independent variable. One such example comes from differential equations, where quantities such as the temperature of a metal rod depend on both a position \(x\) and a time \(t\text{.}\)

Definition 11.1.1. Functions of Two Variables.

A function of two variables is a rule \(f\) or \(z = f(x,y)\) that assigns pairs \((x,y)\) of real numbers to real numbers. The domain is the set \(D\) in \(\RR^{2}\) for which the rule is defined. The range is the set \(R\) in \(\RR\) of all possible values of \(f\text{.}\) \(x,y\) are called independent variables and \(z\) is called the dependent variable.

Example 11.1.2. Domain of a function.

Let \(f(x,y) = \frac{1}{\sqrt{4 - x^{2} - y^{2}}}\text{.}\) Find the domain of \(f\text{.}\)

Solution

We'll break this down into pieces. First, since we're dealing with square roots in real numbers we need \(x^{2}+y^{2}\leq4\text{.}\) Second, we need to make sure that \(\sqrt{4 - x^{2} - y^{2}}\neq0\text{.}\) So the domain of \(f\) is the set of all points \((x,y)\) in \(\RR^{2}\) such that \(x^{2} + y^{2} < 4\text{.}\) In other words, the domain is just the inside of the circle of radius \(2\) centered at the origin.

We can also graph functions of two variables by viewing \(f(x,y)\) as representing the height of some surface over the point \((x,y)\) in the \(xy\)-plane. In other words, just as \(y = f(x)\) represents a curve in \(\RR^{2}\text{,}\) the equation \(z = f(x,y)\) represents a surface in \(\RR^{3}\text{.}\)

Example 11.1.3. Sketching a cone.

Sketch the surface given by \(f(x,y) = \sqrt{4x^{2} + y^{2}}\text{.}\)

Solution

We need to sketch \(z = \sqrt{4x^{2} + y^{2}}\text{.}\) We can start by squaring both sides to get \(z^{2} = 4x^{2} + y^{2}\text{.}\) If we look at the right hand side, it looks a lot like an ellipse, so we'll try to work with that. Assuming \(z\neq0\text{,}\) we can rewrite the equation to get

\begin{equation*} \frac{x^{2}}{z^{2}/4} + \frac{y^{2}}{z^{2}} = 1 \end{equation*}

For any given positive value of \(z\text{,}\) this represents an ellipse that extends to \(\frac{z}{2}\) in the \(x\) direction and \(z\) in the \(y\) direction. So as \(z\) increases, these ellipses increase in size as well.

Subsection 11.1.2 Contour Plots

¶
Definition 11.1.4. Contour Plots.

Given a function \(f(x,y)\text{,}\) a contour plot is the plot of an equation of the form \(f(x,y) = k\) for some constant \(k\text{.}\)

A contour plot can be viewed as slicing a surface in \(\RR^{3}\) with a level plane. The contour plot is exactly where the plane intersects the surface.

Example 11.1.5. Contours on a paraboloid.

What do the contours of \(f(x,y) = x^{2} + \frac{y^{2}}{9}\) look like?

Solution

If we pick different (nonnegative) values for \(k\text{,}\) then all of the contours of \(f(x,y)\) take the form

\begin{equation*} x^{2} + \frac{y^{2}}{9} = k \end{equation*}

which turns into

\begin{equation*} \frac{x^{2}}{k} + \frac{y^{2}}{{9k}} = 1. \end{equation*}

This is a family of ellipses.
