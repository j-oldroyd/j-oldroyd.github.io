Section 6.2 The Definite Integral

The process used in Section 6.1 can be generalized quite a bit. Start with a function \(f(x)\) defined on some interval \([a,b]\text{,}\) and divide \([a,b]\) into \(n\) subintervals

\begin{equation*} [x_{0},x_{1}], [x_{1},x_{2}],\ldots,[x_{n-1},x_{n}] \end{equation*}

where \(x_{0} = a, x_{n} = b\) and each subinterval has length \(\Delta x_{i}\) (i.e.

\begin{equation*} \Delta x_{i} = x_{i} - x_{i-1} \end{equation*}

). This is known as a partition of \([a,b]\text{.}\) Then we can choose sample points \(x_{1}^{*}\) from \([x_{0},x_{1}]\text{,}\) \(x_{2}^{*}\) from \([x_{1},x_{2}]\text{,}\) all the way up to \(x_{n}^{*}\) from \([x_{n-1},x_{n}]\text{.}\) Note that the only restriction we're placing on the sample point \(x_{i}^{*}\) is that is must come from the interval \([x_{i-1},x_{i}]\text{,}\) but it may be any point in this interval.

Using all of this, we can define the Riemann sum associated to \(f\) on this partition:

\begin{equation*} \sum_{i=1}^{n}f(x_{i}^{*})\Delta x_{i}. \end{equation*}

Although this can be more complicated than our definition in Definition 6.1.1, it still approximates the area under \(f(x)\) from \(x=a\) to \(x=b\text{.}\) And as each subinterval in this partition gets smaller (i.e. as \(\Delta x_{i}\to0\) or equivalently as \(n\to\infty\)), then this approximation should become exact, at least if \(f(x)\) is nice enough.

Definition 6.2.1. The Definite Integral.

Let \(f(x)\) be a function defined on \([a,b]\text{.}\) Then the definite integral of \(f(x)\) from \(a\) to \(b\) is the number

\begin{equation*} \int_{a}^{b}f(x)\,dx = \lim_{n\to\infty}\sum_{i=1}^{n}f(x_{i}^{*})\Delta x_{i}, \end{equation*}

provided that the limit exists (and that \(\Delta x_{i}\to0\)). If this limit exists, we say that \(f(x)\) is integrable on \([a,b]\text{.}\)

We can use Definition 6.2.1 to rewrite our area definition: the area under \(y=f(x)\) from \(x=a\) to \(x=b\) is defined to be the definite integral of \(f\) on \([a,b]\text{,}\) assuming it exists. For a function to be integrable, the particular partition of \([a,b]\) that we use cannot affect the limit in Definition 6.2.1. This gives us a great deal of freedom in approximating the definite integral/area under a curve: when choosing sample points, we can use left endpoints; right endpoints; midpoints and more. However, if \(f\) is integrable then we lose nothing by just using right endpoints and subintervals of equal length, as in Definition 6.1.1.

Theorem 6.2.2. Integrability of Piecewise Continuous Functions.

Let \(f(x)\) be a piecewise function on \([a,b]\) with finitely many jump discontinuities. Then \(f\) is integrable on \([a,b]\text{.}\)

Example 6.2.3. Expressing work done.

A particle moving from \(x=-1\) to \(x=3\) is acted upon by the force \(F(x) = 3x\text{.}\) Find the work done by the force on this particle.

Solution.

Using Example 6.1.7, we see that the work done should be

\begin{equation*} W = \lim_{n\to\infty}\sum_{i=1}^{n}3x_{i}^{*}\cdot\frac{4}{n} \end{equation*}

which we can rewrite as

\begin{equation*} W = \int_{-1}^{3}3x\,dx. \end{equation*}

So work is a definite integral.

In this case, we can also find the work done. Since the definite integral represents area, and the definite integral of \(3x\) exists (since this is continuous), then we can say that

\begin{equation*} W = \int_{-1}^{3}3x\,dx = \frac{1}{2}(1)(-3) + \frac{1}{2}(3)(9) = 12. \end{equation*}

Example 6.2.4. A radical integral.

Determine \(\int_{0}^{2}\sqrt{4-x^{2}}\,dx\text{.}\)

Solution.

We won't be able to solve this integral algebraically until Calculus 2, but we can solve it geometrically instead. If we graph \(y = \sqrt{4-x^{2}}\text{,}\) we get the upper half of the circle of radius \(2\) centered at the origin. So the integral given must be exactly one-fourth the area of this circle, so

\begin{equation*} \int_{0}^{2}\sqrt{4-x^{2}}\,dx = \frac{1}{4}\pi(2^{2}) = \pi. \end{equation*}

Since the definite integral is defined as a limit of sums, it shares many properties with sums. In particular, we have the following version of Theorem 6.1.5

Theorem 6.2.5. Properties of the Definite Integral.

Let \(f(x)\) and \(g(x)\) be integrable on \([a,b]\) and let \(c\) be a constant. Let \(t\) be any number in \([a,b]\text{.}\) Then the following are true:

\begin{align*} \int_{a}^{b}c\,dx & = c(b-a) \\ \int_{a}^{b}cf(x)\,dx & = c\int_{a}^{b}f(x)\,dx \\ \int_{a}^{b}(f(x)+g(x))\,dx & = \int_{a}^{b}f(x)\,dx + \int_{a}^{b}g(x)\,dx\\ \int_{a}^{b}(f(x) - g(x))\,dx & = \int_{a}^{b}f(x)\,dx - \int_{a}^{b}g(x)\,dx \\ \int_{a}^{t}f(x)\,dx + \int_{t}^{b}f(x)\,dx & = \int_{a}^{b}f(x)\,dx. \end{align*}

In other words, we can break definite integrals up over addition and subtraction, and we can move constants outside of it.

Computing integrals in practice often involves setting up and approximating Riemann sums, i.e. sums of the form \(\sum_{i=1}^{n}f(x_{i}^{*})\Delta x_{i}\text{.}\) These sums depend on how we choose the sample points \(x_{i}^{*}\) from each interval \([x_{i-1},x_{i}]\text{.}\) We've mostly worked with \(x_{i}^{*} = x_{i}\text{,}\) or right endpoints. We could also choose \(x_{i}^{*} = x_{i-1}\text{,}\) using left endpoints. However, it's often more accurate to use midpoints: \(x_{i}^{*} = \frac{x_{i-1}+x_{i}}{2}\text{.}\) The reason for this is that if we choose midpoints of intervals as our sample points, these tend to be less affected than either endpoint if the function is increasing or decreasing. In particular, we have the midpoint rule.

Definition 6.2.6. Midpoint Rule.

Let \(f(x)\) be a continuous function. Then

\begin{equation*} \int_{a}^{b}f(x)\,dx \approx M_{n} = \sum_{i=1}^{n}f(x_{i}^{*})\Delta x \end{equation*}

where

\begin{align*} x_{i}^{*} & = \frac{x_{i-1}+x_{i}}{2} \\ \Delta x & = \frac{b-a}{n}. \end{align*}

Note that this rule calls for evenly dividing \([a,b]\) into a partition \([x_{i-1},x_{i}]\) where each subinterval has equal length \(\Delta x\text{.}\)

Example 6.2.7. Approximating an integral.

Approximate \(\int_{1}^{3}\frac{1}{x}\,dx\) using Riemann sums with \(n=4\text{.}\)

Solution.

For this integral, we have

\begin{align*} \Delta n & = \frac{3-1}{4} = \frac{1}{2} \\ x_{0} & = 1 \\ x_{1} & = \frac{3}{2} \\ x_{2} & = 2 \\ x_{3} & = \frac{5}{2} \\ x_{4} & = 3 \end{align*}

We will have different estimates using left endpoints, right endpoints and midpoints. Using left endpoints, we have

\begin{equation*} \int_{1}^{3}\frac{1}{x}\,dx \approx L_{4} = \frac{1}{2}\left(1 + \frac{2}{3} + \frac{1}{2} + \frac{2}{5}\right) = 1.283.... \end{equation*}

Using midpoints, we have

\begin{equation*} \int_{1}^{3}\frac{1}{x}\,dx \approx M_{4} = \frac{1}{2}\left(\frac{4}{5} + \frac{4}{7} + \frac{4}{9} + \frac{4}{11}\right) \approx1.09. \end{equation*}

Using right endpoints, we have

\begin{equation*} \int_{1}^{3}\frac{1}{x}\,dx \approx R_{4} = \frac{1}{2}\left(\frac{2}{3} + \frac{1}{2} + \frac{2}{5} + \frac{1}{3}\right) = 0.95. \end{equation*}

Geometrically, we see that

\begin{equation*} 0.95 = R_{4} < \int_{1}^{3}\frac{1}{x}\,dx < L_{4} = 1.283... \end{equation*}

The exact value of this integral is actually \(\ln 3\text{,}\) and we see that \(M_{4}\text{,}\) the midpoint estimate, is clearly the best approximation to the integral in this case.

Theorem 6.2.8. More Properties of the Definite Integral.

Let \(f(x),g(x)\) be integrable functions on \([a,b]\text{.}\) Then

  1. \(\displaystyle \int_{a}^{b}f(x)\,dx = - \int_{b}^{a}f(x)\,dx\)

  2. \(\displaystyle \int_{a}^{a}f(x)\,dx = 0\)

  3. If \(f(x)\geq g(x)\) on \([a,b]\text{,}\) then \(\int_{a}^{b}f(x)\,dx \geq \int_{a}^{b}g(x)\,dx\text{.}\)

  4. If \(m\leq f(x)\leq M\) on \([a,b]\text{,}\) then \(m(b-a)\leq \int_{a}^{b}f(x)\,dx\leq M(b-a)\text{.}\)
