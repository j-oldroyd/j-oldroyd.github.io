Section 4.6 Inverse Trigonometric Functions

Recall that \(\sin x\) is not a one-to-one function since it fails the horizontal line test. Hence it has no inverse function. However, we can restrict the domain of \(\sin x\) so that what's left over passes the horizontal line test. For example, if we define \(f(x) = \sin x\) but restrict the domain to the interval \([-\frac{\pi}{2},\frac{\pi}{2}]\text{,}\) then \(f(x)\) is one-to-one and so has an inverse function. We call this function inverse sine or arcsine.

Definition 4.6.1. Inverse Sine.

Let \(f(x) = \sin x\) and have domain \([-\frac{\pi}{2},\frac{\pi}{2}]\text{.}\) Then \(f^{-1}(x)\) is called the inverse sine or arcsine of \(x\text{,}\) and is denoted by \(\sin^{-1}x\) or \(\arcsin x\text{.}\) The domain is the interval \([-1,1]\) and the range is \([-\frac{\pi}{2},\frac{\pi}{2}]\text{.}\)

We can think of \(\sin^{-1}x = \arcsin x\) as the angle required to turn sine into \(x\text{.}\)

Note that \(\sin^{-1}x\) does not mean \(\frac{1}{\sin x}\text{,}\) which is \(\csc x\text{.}\) It's an unfortunate, though standard, choice of notation.

Example 4.6.2. Finding inverse sine.

Find \(\sin^{-1}(\frac{\sqrt{3}}{2})\text{.}\)

Solution

This is the angle between \(-\frac{\pi}{2}\) and \(\frac{\pi}{2}\) that turns sine into \(\frac{\sqrt{3}}{2}\text{.}\) So \(\sin^{-1}(\frac{\sqrt{3}}{2}) = \frac{\pi}{3}\text{.}\)

Example 4.6.3. Simplifying cosine and inverse sine.

Simplify \(\cos(\arcsin x)\text{,}\) where \(-1\leq x\leq 1\text{.}\)

Solution

By definition, \(\sin(\arcsin x) = x\) as long as \(-1\leq x\leq 1\text{.}\) So we want to try to rewrite \(\cos(\arcsin x)\) to make use of this cancellation property. We can do this using the Pythagorean identity \(\cos \theta = \pm\sqrt{1-\sin^{2}\theta}\text{.}\) Hence

\begin{equation*} \cos(\arcsin x) = \sqrt{1-(\sin(\arcsin x))^{2}} = \sqrt{1 - x^{2}} \end{equation*}

whenever \(-1\leq x\leq 1\text{.}\)

Since \(\sin x\) is differentiable, this means \(\arcsin x\) is also differentiable.

Theorem 4.6.4. Derivative of Inverse Sine.

Let \(-1 < x < 1\text{.}\) Then \(\dv{}{x}\sin^{-1}x = \frac{1}{\sqrt{1-x^{2}}}\text{.}\)

Proof.

If we set \(y = \sin^{-1}x\text{,}\) then we get \(\sin y = x\text{.}\) Differentiating this gives \(y'\cos y = 1\) or just

\begin{equation*} y' = \frac{1}{\cos y} = \frac{1}{\sqrt{1-x^{2}}}. \end{equation*}

We can also define the inverse cosine function in much the same way as the inverse sine function. The idea once again is to restrict the domain of \(\cos x\) to \([0,\pi]\) to get an invertible function. We call the inverse of this function the inverse cosine, or arccosine, function. We denote this function by \(\cos^{-1}x\) or \(\arccos x\text{.}\)

Example 4.6.5. Inverse cosine value.

Find \(\csc(\cos^{-1}(-\frac{1}{2})\text{.}\)

Solution

First, note that \(\cos^{-1}(-\frac{1}{2}) = \frac{4\pi}{3}\text{.}\) So

\begin{equation*} \csc(\cos^{-1}(-\frac{1}{2})) = \csc(\frac{4\pi}{3}) = \frac{2}{\sqrt{3}}. \end{equation*}

Inverse cosine behaves in much the same way as inverse sine. However, this function won't be as useful to us as the inverse sine function or the next function we will look at: the inverse tangent.

If we restrict the domain of \(\tan x\) to \((-\frac{\pi}{2}, \frac{\pi}{2})\text{,}\) then we get an invertible function. We call the inverse of this function the inverse tangent or arctangent, and denote it by \(\tan^{-1}x\) or \(\arctan x\text{.}\) The domain of \(\arctan x\) is \((-\infty,\infty)\) and the range is \((-\frac{\pi}{2},\frac{\pi}{2})\text{.}\) \(\tan^{-1}x\) can be thought of as the angle between \(-\frac{\pi}{2}\) and \(\frac{\pi}{2}\) that makes tangent equal to \(x\text{.}\) It satisfies the following cancellation properties: \(\tan(\tan^{-1}x) = x\) for all \(x\) and \(\tan^{-1}(\tan y) = y\) for all \(y\) in \((-\frac{\pi}{2},\frac{\pi}{2})\text{.}\)

Example 4.6.6. Simplifying an inverse tangent.

Simplify \(\sin(2\arctan x)\) using the formula \(\sin2x = 2\sin x\cos x\text{.}\)

Solution

If we use the double-angle formula given, we get

\begin{equation*} \sin(2\arctan x) = 2\sin(\arctan x)\cos(\arctan x). \end{equation*}

So we just need to find \(\sin(\arctan x)\) and \(\cos(\arctan x)\text{.}\)

Set \(\theta = \arctan x\text{,}\) so that \(\tan\theta = x\text{.}\) Then we can find \(\sin\theta\) and \(\cos\theta\) using triangles, which gives

\begin{align*} \sin\theta & = \frac{x}{\sqrt{1+x^{2}}} \\ \cos\theta & = \frac{1}{\sqrt{1+x^{2}}} \end{align*}

Hence

\begin{align*} \sin(2\arctan x) & = \sin2\theta \\ & = 2\sin\theta\cos\theta \\ & = \frac{2x}{1+x^{2}}. \end{align*}
Example 4.6.7. Limit of inverse tangent.

Determine \(\lim_{x\to\infty}\tan^{-1}x\text{.}\)

Solution

Recall that \(\tan^{-1}x\) is the angle between \(-\frac{\pi}{2}\) and \(\frac{\pi}{2}\) for which tangent is equal to \(x\text{.}\) So finding \(\lim_{x\to\infty}\tan^{-1}x\) is equivalent to finding what angle between \(-\frac{\pi}{2}\) and \(\frac{\pi}{2}\) we have to approach in order for tangent to blow up to \(\infty\text{.}\) Either from looking at a graph or by using the definition of tangent, we see that the angle we need to approach is exactly \(\frac{\pi}{2}\text{.}\) Hence \(\lim_{x\to\infty}\tan^{-1}x = \frac{\pi}{2}\text{.}\)

Theorem 4.6.8. Derivative of Inverse Tangent.

The derivative of \(\tan^{-1}x\) is \(\frac{1}{1+x^{2}}\text{.}\)

Proof.

We can prove this the same way we proved that \(\dv{}{x}\arcsin x = \frac{1}{\sqrt{1+x^{2}}}\text{.}\) All we need to do is to set \(y = \tan^{-1}x\text{,}\) and then find \(y'\) by implicit differentiation.

Example 4.6.9. Tangent half-angle substitution.

An important substitution in integral calculus is the tangent half-angle substitution defined by \(t = \tan\frac{x}{2}\text{.}\) Use this equation to find \(\dv{x}{t}\text{.}\)

Solution

We can find \(x'\) implicitly, but we can also solve for \(x\) to get

\begin{equation*} x = 2\tan^{-1}t. \end{equation*}

Therefore

\begin{equation*} \dv{x}{t} = 2\dv{}{t}\tan^{-1}t = \frac{2}{1+t^{2}}. \end{equation*}
