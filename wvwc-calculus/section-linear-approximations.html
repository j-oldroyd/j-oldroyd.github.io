Section 3.8 Linear Approximations

Suppose we graph the tangent line to \(y=\sqrt{x}\) at \(x=4\text{.}\) The equation of this line will be

\begin{equation*} y-2 = \frac{x-4}{4}\quad\text{or}\quad y= 2 + \frac{x-4}{4}. \end{equation*}

If we look at the tangent line next to the graph, we'll see that the line is very close to the graph of \(y=\sqrt{x}\) if \(x\) is very close to \(4\text{.}\) So we can use the equation of the tangent line to approximate \(y=\sqrt{x}\) when \(x\) is near \(4\text{.}\) For example, we can say that

\begin{equation*} \sqrt{4.2} \approx 2 + \frac{4.2 - 4}{4} = 2.05. \end{equation*}

We call this the linear approximation of \(\sqrt{x}\) at \(4\text{.}\)

Definition 3.8.1. Linear Approximation.

Let \(f(x)\) be a function that is differentiable at \(x=a\text{.}\) The linear approximation (or linearization) of \(f\) at \(a\) is the function \(L(x)\) given by

\begin{equation*} L(x) = f(a) + f'(a)(x-a). \end{equation*}

The formula of the linear approximation to a function \(f\) at \(a\) is nothing more than the equation of the tangent line to \(f\) at \(a\text{.}\)

Example 3.8.2. Linear approximation of sine.

Find the linear approximation of \(f(\theta) = \sin\theta\) at \(\theta=0\) and use this to estimate \(\sin(-.02)\text{.}\)

Solution.

The linear approximation is given by

\begin{equation*} L(\theta) = f(0) + f'(0)(\theta - 0) = \sin0 + \cos0 (\theta - 0), \end{equation*}

so \(L(\theta) = \theta\text{.}\) Therefore

\begin{equation*} \sin(-.02)\approx-.02 \end{equation*}

since \(-.02\) is very close to \(0\text{.}\)

Example 3.8.3. Estimating a tangent.

Estimate \(\tan(\pi+\frac{3}{49})\text{.}\)

Solution.

We want to estimate this using a linear approximation, and we want to make sure the linear approximation is easy to set up. This means we want to base the linear approximation at a value of \(a\) that both tangent and its derivative are relatively easy to compute at, and is also close to \(\pi+\frac{3}{49}\text{.}\) So we'll pick \(a=\pi\) and find the linear approximation to \(\tan x\) at \(a = \pi\text{:}\) Set \(f(x) = \tan x\text{.}\) Then

\begin{align*} L(x) & = f(a) + f'(a)(x-a)\\ & = \tan\pi + \sec^{2}(\pi)(x-\pi) \\ & = x-\pi. \end{align*}

So

\begin{equation*} \tan(\pi+\frac{3}{49}) \approx (\pi+\frac{3}{49}) - \pi = \frac{3}{49}. \end{equation*}

Example 3.8.4. Estimating the population of the United States in 2020.

Use the differential equation from Example 3.6.4 and the fact that the current population of the US is about 323.1 million to estimate the population of the US in 2020.

Solution.

Recall that \(P(t)\) in Example 3.6.4 represented the population of the US (in millions) \(t\) years after 1990. To estimate the population in 2020, we want to find the linearization of \(P\) at \(t = 27\text{.}\) This is given by

\begin{equation*} L(t) = P(27) + P'(27)(t - 27). \end{equation*}

Since \(P(27) = 323.1\) and

\begin{equation*} P'(27) = \frac{1}{23500}P(532-P) = \frac{1}{23500}(323.1)(532-323.1) = 2.9, \end{equation*}

we have

\begin{equation*} L(t) = 323.1 + 2.9(t-27). \end{equation*}

Hence the population of the US in 2020 should be

\begin{equation*} L(30) = 323.1 + 2.9\cdot3 = 331.8, \end{equation*}

or about 331.8 million people.
