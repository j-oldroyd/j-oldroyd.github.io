Section 2.3 Differentiation Formulas

¶

Now we start to find methods that allow us to compute derivatives without going back to Definition 2.2.1. Perhaps the easiest rule is the constant rule, which just says that the derivative of a constant is \(0\text{.}\) We'll derive more complicated rules in this section and the next two.

Subsection 2.3.1 The Power Rule and Trigonometric Derivatives

¶

Our first goal will be to determine a general formula for the derivative of \(x^{n}\) for some power \(n\text{.}\) If \(n\geq0\) is a whole number, then we can find the derivative of \(x^{n}\) without too much trouble. In fact, from the Definition 2.2.1 it's not too hard to show that the derivative of \(x\) is \(1\text{,}\) the derivative of \(x^{2}\) is \(2x\text{,}\) the derivative of \(x^{3}\) is \(3x^{2}\text{,}\) and so on. This suggests our first derivative rule, the power rule.

Theorem 2.3.1 The Power Rule

Let \(f(x) = x^{n}\) where \(n\) is some real number. Then \(f'(x) = nx^{n-1}\text{.}\)

Note that Theorem 2.3.1 is actually quite general: it works for all powers of \(x\text{!}\) This includes negative powers and fractional powers.

Example 2.3.2 Derivatives using the power rule

Find the derivatives of the following functions:

  1. \(f(x) = x^{7}\text{.}\)
  2. \(g(x) = \frac{1}{\sqrt[5]{x^{11}}}\text{.}\)
  3. \(h(x) = x^{\pi}\text{.}\)
Solution

The derivative of \(f(x)\) isn't too hard to find using the power rule, and we quickly get \(f'(x) = 7x^{6}\text{.}\) For \(g(x)\text{,}\) first rewrite it as \(g(x) = x^{-\frac{11}{5}}\text{.}\) Then \(g'(x) = -\frac{11}{5}x^{-16}{5}\text{.}\) Finally, \(h'(x) = \pi x^{\pi-1}\text{.}\)

With a little bit of geometry and the squeeze theorem, we can get the derivatives of the basic trigonometric functions \(\sin x\) and \(\cos x\text{.}\)

Theorem 2.3.3 Derivatives of Sine and Cosine

Let \(x\) be in radians. Then

\begin{equation*} \dv{}{x}\sin x = \cos x\quad\text{ and }\quad\dv{}{x}\cos x = -\sin x. \end{equation*}

Note that if \(x\) is in degrees instead of radians these formulas don't work. Instead, they become

\begin{equation*} \dv{}{x}\sin x = \frac{\pi}{180}\cos x\quad\text{ and }\quad\dv{}{x}\cos x = -\frac{\pi}{180}\sin x. \end{equation*}
Example 2.3.4 Concavity of the sine function

On which intervals is \(f(x) = \sin x\) concave up?

Solution

We need to find where \(f''(x)\) is positive. Since \(f''(x) = -\sin x\text{,}\) this means we need to figure out where \(\sin x\) is negative. If we go back to the unit circle definition of sine, then we can see that \(\sin x < 0\) on the following intervals:

\begin{equation*} \ldots, (-3\pi,-2\pi), (-\pi,0),(\pi,2\pi), (3\pi,4\pi),\ldots \end{equation*}

So \(\sin x\) is concave up on every open interval of the form \(((2k-1)\pi, 2k\pi)\) where \(k\) is some integer.

Subsection 2.3.2 Derivatives of Sums and Constant Multiples

¶

Now that we have derivative formulas for some basic functions, we want to extend these to more complicated functions. For this section we'll look at what happens when we multiply a function by a constant or add it to another function. In the next two sections we'll consider more advanced rules.

Theorem 2.3.5 Constant Multiple Rule

Let \(f(x)\) be differentiable function and let \(c\) be some constant. Then \(\dv{}{x}[cf(x)] = cf'(x)\text{.}\)

Proof

To prove this, we go back to Definition 2.2.1:

\begin{align*} \dv{}{x}[cf(x)] & = \lim_{h\to0}\frac{cf(x+h) - cf(x)}{h} \\ & = c\lim_{h\to0}\frac{f(x+h) - f(x)}{h} \\ & = cf'(x). \end{align*}

Hence the derivative of \(cf(x)\) is \(cf'(x)\text{.}\)

We can find the derivative of the sum of two functions just as easily.

Theorem 2.3.6 Sum Rule

Let \(f(x)\) and \(g(x)\) be two differentiable functions. Then \([f(x)+g(x)]' = f'(x) + g'(x)\text{.}\)
