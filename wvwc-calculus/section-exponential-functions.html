Section 4.1 Exponential Functions

Many changing quantities can be modeled by using exponential functions, which are functions of the form \(f(x) = a^{x}\) where \(a > 0\text{.}\) Exponential functions have several important properties which we list below.

Theorem 4.1.1. Properties of Exponential Functions.

Let \(a > 0\) and \(x,y\) be real numbers. Then

  • \(a^{0} = 1\) and \(a^{1} = a\text{.}\)
  • \(a^{-x} = \frac{1}{a^{x}}\text{.}\)
  • \(a^{x+y} = a^{x}a^{y}\) and \(a^{x-y} = \frac{a^{x}}{a^{y}}\text{.}\)
  • \(a^{x/y} = \sqrt[y]{a^{x}}\) is \(y > 0\text{.}\)
  • \((a^{x})^{y} = a^{xy}\text{.}\)

Finally, if \(b > 0\) as well, then \((ab)^{x} = a^{x}b^{x}\text{.}\)

Example 4.1.2. Limit of an exponential function.

Let \(f(x) = 2^{-x}\text{.}\) Find \(\lim_{x\to\infty}f(x)\text{.}\)

Solution

First, note that

\begin{equation*} f(x) = \frac{1}{2^{x}}. \end{equation*}

So as \(x\to\infty\text{,}\) the denominator \(2^{x}\to\infty\) as well. Hence the fraction as a whole should decrease to \(0\text{,}\) and so

\begin{equation*} \lim_{x\to\infty}2^{-x} = 0. \end{equation*}

In general, we have the following:

\begin{align*} a > 1 & \Rightarrow \lim_{x\to\infty}a^{x} = \infty\quad\text{and}\lim_{x\to-\infty}\quad a^{x} = 0 \\ a < 0 & \Rightarrow \lim_{x\to\infty}a^{x} = 0\quad\text{and}\quad\lim_{x\to-\infty}\quad a^{x} = \infty \end{align*}

Every exponential function \(f(x) = a^{x}\) is continuous everywhere. In fact, they are differentiable everywhere, though we'll see more about that later. We can also say the following: if \(a > 1\) then \(a^{x}\) is an increasing function, and if \(0 < a < 1\) then \(a^{x}\) is a decreasing function. In neither case will \(a^{x}\) have any local maxima or minima.

Of all the different exponential functions, the most important one is the so-called natural exponential function, denoted \(e^{x}\) where

\begin{equation*} e = 2.71828... \end{equation*}

This is the unique exponential function by the property that its derivative at \(0\) is \(1\text{.}\) In fact, as we will see later \(e^{x}\) is its own derivative.

Example 4.1.3. Exponential derivative.

Using the fact that \(\dv{}{x}e^{x} = e^{x}\text{,}\) compute \(\dv{}{x}e^{5x - x^{4} + \cos x}\text{.}\)

Solution

We'll have to use the chain rule. If we do so, we get

\begin{equation*} \dv{}{x}e^{5x - x^{4} + \cos x} = (5 - 4x^{3} - \sin x)e^{5x - x^{4} + \cos x}. \end{equation*}

Since \(e^{x}\) is its own derivative, this makes computing derivatives of functions of the form \(e^{f(x)}\) relatively straightforward:

\begin{equation*} \dv{}{x}e^{f(x)} = f'(x)e^{f(x)}. \end{equation*}

Technically, we have worked with the natural exponential function before in this class. Recall from Example 3.6.4 that if \(P(t)\) represents the population of the USA (in millions) \(t\) years after 1990, then we said that \(P\) satisfied the differential equation

\begin{equation*} \dv{P}{t} = \frac{1}{23500}P(532 - P). \end{equation*}

If we say the population of the US in 1990 was 250 million (i.e. \(P(0) = 250\)), then

\begin{equation*} P(t) = \frac{133000}{250 + 282e^{-.023t}}. \end{equation*}

Now, in Example 3.8.4 we used linear approximation to estimate the population of the US using this model in the year 2020, where we got 331.8 million people. If we use the above formula for \(P(t)\text{,}\) we get that the population of the US in 2020 should be about

\begin{equation*} P(30) = 339.8, \end{equation*}

or 339.8 million people.

The Census Bureau predicted in 2014 that the population in 2020 would be 334.5 million. So our simple predictions before actually aren't too far off from what the Census Bureau is expecting. This suggests that the differential equation \(P' - \frac{1}{23500}P(532-P)\) is a reasonable model for the growth of the US.
