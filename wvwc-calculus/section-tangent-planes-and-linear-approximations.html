Section 12.4 Tangent Planes and Linear Approximations

In Calculus I, derivatives are used to find linear approximations to functions of the form \(f(x)\text{.}\) We can use partial derivatives to do the same for functions with several independent variables.

Subsection 12.4.1 Tangent Planes

Recall that if a curve \(y=f(x)\) is differentiable at a point \(x_{0}\text{,}\) then it has a tangent line passing through \((x_{0},f(x_{0})).\) The tangent line can be viewed as a linear approximation of the curve \(y=f(x)\) near the point \(x_{0}\text{.}\) We can apply a similar ideas to surfaces \(z = f(x,y)\text{.}\) It turns out that if we have a surface \(z = f(x,y)\) and a point \((x_{0},y_{0}, f(x_{0},y_{0}))\) on the surface, then every tangent vector at this point is contained in a single plane called the tangent plane.

We'd like to find the equation of this plane. First, recall that every plane can be described by an equation of the form

\begin{equation*} a(x-x_{0})+b(y-y_{0})+c(z-z_{0}) = 0. \end{equation*}

We can view \(x_{0},y_{0}\) and \(z_{0} = f(x_{0},y_{0})\) as given, so we just need to find \(a,b,c\text{.}\) If we assume that \(c\neq 0\text{,}\) then we can rewrite this equation to obtain

\begin{equation*} z-z_{0} = A(x-x_{0}) + B(y-y_{0}) \end{equation*}

where \(A = -\frac{a}{c}\) and \(B = -\frac{b}{c}\text{.}\) If we set \(y=y_{0}\text{,}\) then we have \(z-z_{0} = A(x-x_{0})\text{.}\) This is the equation of a line tangent to the surface and parallel to the \(x\)-axis, and so the slope of this line must be

\begin{equation*} A = f_{x}(x_{0},y_{0}) \end{equation*}

since the slope of a tangent line in the \(x\)-direction gives the rate of change in the \(x\)-direction. Similarly,

\begin{equation*} B = f_{y}(x_{0},y_{0}). \end{equation*}

Putting all of this together gives the following theorem.

Theorem 12.4.1. Tangent Planes to Surfaces.

Let \(z = f(x,y)\) be a surface and suppose that \(f\) has continuous partial derivatives at the point \((x_{0},y_{0})\text{.}\) Then the tangent plane to the surface \(z = f(x,y)\) at the point \((x_{0},y_{0})\) is given by

\begin{equation*} z - z_{0} = f_{x}(x_{0},y_{0})(x - x_{0}) + f_{y}(x_{0},y_{0})(y - y_{0}), \end{equation*}

where \(z_{0} = f(x_{0},y_{0})\text{.}\)

Example 12.4.2. Approximations by tangent planes.

Find the tangent plane to \(f(x,y) = xe^{xy}\) at the point \((2,0,2)\text{.}\) Use this to approximate \(f(1.9,0)\)

Solution.

The equation of the tangent plane is given by

\begin{equation*} z - 2 = (x - 2) + 4(y - 0) \end{equation*}

which we can rewrite as

\begin{equation*} z = x + 4y. \end{equation*}

So at \((1.9,0)\text{,}\) we should have

\begin{equation*} f(1.9,0)\approx 1.9 + 4(0) = 1.9. \end{equation*}

Subsection 12.4.2 Linear Approximations

Example 12.4.2 shows that we can use tangent planes to approximate complicated functions. This leads us to the idea of a linear approximation, or linearization of a function of the form \(f(x,y)\text{.}\)

Definition 12.4.3. Linearization.

Let \(f(x,y)\) be a function for which \(f_{x}\) and \(f_{y}\) are both continuous at a point \((x_{0},y_{0})\text{.}\) Then the linearization of \(f(x,y)\) at \((x_{0},y_{0})\) is the function \(L(x,y)\) given by

\begin{equation*} L(x,y) = f(x_{0},y_{0}) + f_{x}(x_{0},y_{0})(x-x_{0}) + f_{y}(x_{0},y_{0})(y-y_{0}). \end{equation*}

For well-behaved functions (i.e. functions that have continuous partial derivatives), \(L(x,y)\approx f(x,y)\) if \((x,y)\) is close to the point \((x_{0},y_{0})\text{.}\)

Example 12.4.4. Linearization of an exponential and a sinusoid.

Let \(u(x,t) = 30e^{-t}\sin x\text{.}\) Find the linearization of \(u\) at the point \((0,\frac{\pi}{2})\text{.}\)

Solution.

By Definition 12.4.3, the linearization is given by

\begin{equation*} L(x,y) = 30 - 30(t - 0) = 30(1 - t). \end{equation*}

Definition 12.4.3 can also be extended to functions with more than two variables.

Example 12.4.5. Linearization in three variables.

Let \(f(x,y,z) = xze^{-y^{2}-z^{2}}\text{.}\) Find the linearization \(L(x,y,z)\) at the point \((-3,\sqrt{\ln2},-\sqrt{\ln3})\text{.}\)

Solution.

The formula we need to use now is

\begin{equation*} L(x,y,z) = f_{x}(-3,\sqrt{\ln2},-\sqrt{\ln3})(x+3) + f_{y}(-3,\sqrt{\ln2},-\sqrt{\ln3})(y - \sqrt{\ln2}) + f_{z}(-3,\sqrt{\ln2},-\sqrt{\ln3})(z + \sqrt{\ln3}). \end{equation*}
