Solution 6.6.5.1.

Since \(P(t)\) must be an antiderivative of \(r(t)\text{,}\) we can write

\begin{equation*} P(t) = \int r(t)\,dt + C\text{.} \end{equation*}

However, the given information lets us be a little more specific:

\begin{equation*} P(t) = \int_{0}^{t}r(s)\,ds + 100\text{.} \end{equation*}

Note that \(P(0) = 100\) as required and \(P'(t) = r(t)\) by TheoremÂ 6.4.2. Now we just need to find the integral of \(r(s)\text{,}\) which we can do by rewriting it in terms of the natural exponential:

\begin{equation*} r(s) = e^{(3.08\ln2)s}\text{,} \end{equation*}

and so

\begin{equation*} P(t) = \frac{e^{(3.08\ln2)t} - 1}{3.08\ln2} + 100\text{.} \end{equation*}
