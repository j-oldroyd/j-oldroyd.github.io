We can define inverse hyperbolic trigonometric functions just as we did for trigonometric functions. Since \(f(x) = \sinh x\) is one-to-one, its inverse function exists for all real numbers. We can find it as follows: if \(y = \sinh^{-1}x\text{,}\) then \(\sinh y = x\text{.}\) By definition of \(\sinh\) we then have

\begin{equation*} \sinh y = \frac{e^{y} - e^{-y}}{2} \end{equation*}

which gives us the equation

\begin{equation*} e^{2y} - 2xe^{y} - 1 = 0\text{.} \end{equation*}

This equation is quadratic in \(e^{y}\) and solution

\begin{equation*} e^{y} = \frac{2x\pm\sqrt{4x^{2} + 4}}{2} = x \pm \sqrt{x^{2} + 1}\text{.} \end{equation*}

Since exponentials must always be positive, it follows that

\begin{equation*} y = \sinh^{-1}x = \ln(x + \sqrt{x^{2} + 1})\text{.} \end{equation*}

A similar argument also gives

\begin{equation*} \cosh^{-1}x = \ln(x + \sqrt{x^{2} - 1})\quad\text{and}\quad\tanh^{-1}x = \frac{1}{2}\ln\frac{1+x}{1-x}\text{.} \end{equation*}
