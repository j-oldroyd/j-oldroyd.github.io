In Section 14.3 we introduced the \(\del\) operator, which in \(\RR^{3}\) is given by

\begin{equation*} \del = \dotprod{\pdv{}{x}, \pdv{}{y}, \pdv{}{z}}\text{.} \end{equation*}

In this class, \(\del\) has no meaning when by itself. However, it gains meaning when “multiplied” by a scalar function \(f\text{,}\) leaving the gradient \(\grad f\text{.}\) We also used it to define the curl, which we repeat below.
