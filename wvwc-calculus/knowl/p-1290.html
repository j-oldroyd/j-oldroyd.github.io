If \(\vv{r}(t) = \dotprod{f(t),g(t),h(t)},\) then \(\vv{r}\) is differentiable if and only if \(f,g,h\) are, and \(\vv{r}'(t) = \dotprod{f'(t),g'(t),h'(t)}.\) Just as in Calculus I, the derivative represents how quickly a space curve is changing at some value of \(t\text{.}\) However, derivatives of vector functions also carry information about the direction a curve is moving.
