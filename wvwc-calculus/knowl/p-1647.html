Here we are using the physical definition of a conservative force, namely that the work done must be path independent. So we'll choose two paths between two points and find the work done on each path. Let the first path, \(C_{1}\text{,}\) denote the top half of the unit circle traversed counterclockwise. Similarly, let the second path, \(C_{2}\text{,}\) denote the bottom half of the unit circle traversed clockwise. Then both paths have the same initial and terminal points.
