If we use \(t = \tan(\frac{x}{2})\text{,}\) then we can solve for \(x\) to get

\begin{equation*} x = 2\arctan t \Rightarrow dx = \frac{2}{1+t^{2}}\,dt\text{.} \end{equation*}

Therefore

\begin{equation*} \cos x = \cos(2\arctan t) = \cos^{2}(\arctan t) - \sin^{2}(\arctan t) = \frac{1}{1+t^{2}} - \frac{t^{2}}{1+t^{2}}\text{,} \end{equation*}

and so

\begin{align*} \int\frac{1}{2+\cos x}\,dx \amp = \int\frac{2}{2(1+t^{2}) + (1-t^{2})}\,dt\\ \amp = \int\frac{2}{3 + t^{2}}\,dt\\ \amp = \frac{2}{3}\int\frac{1}{1 + (t/\sqrt{3})^{2}}\,dt\\ \amp = \frac{2}{3}\sqrt{3}\arctan(\frac{t}{\sqrt{3}}) + C\\ \amp = \frac{2}{\sqrt{3}}\arctan(\frac{\tan(x/2)}{\sqrt{3}}) + C\text{.} \end{align*}
