Theorem 13.3.7. Conservative Vector Fields are Irrotational.

Let \(\vb{F} = \dotprod{P,Q}\) be a continuously differentiable vector field on an open simply connected region in \(\RR^{2}\text{.}\) Then \(\vb{F}\) is conservative on this region if and only if

\begin{equation*} \pdv{Q}{x} - \pdv{P}{y} = 0\text{.} \end{equation*}
