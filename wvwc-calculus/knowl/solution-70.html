Solution 4.6.9.1.

This limit is a \(1^{\infty}\) indeterminate form. We'll try taking logarithms to rewrite it as a \(0\cdot\infty\) form, so set \(y = \lim_{x\to\infty}\left(1+\frac{1}{x}\right)^{x}\text{.}\) Then

\begin{equation*} \ln y = \lim_{x\to\infty}x\ln\left(1+\frac{1}{x}\right) \end{equation*}

is a \(0\cdot\infty\) form. We can use L'Hospital's by rewriting this limit at a \(\frac{0}{0}\) form or \(\frac{\infty}{\infty}\text{.}\) Either way, we get \(\ln y = 1\) and so the original limit is \(y = e\text{.}\)
