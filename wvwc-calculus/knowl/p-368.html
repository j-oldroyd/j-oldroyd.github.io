Consider a differentiable function \(f(x)\text{,}\) and suppose we want to find a root of \(f(x)\text{,}\) which is a number \(x_{0}\) such that \(f(x_{0}) = 0\text{.}\) In some cases this is very easy (like \(f(x) = x^{2} + 2x + 1\)), but in others this may be more complicated (such as \(f(x) = x^{5} - x - 1\)). So we'd like to find a way to approximate the root \(x_{0}\text{.}\)
