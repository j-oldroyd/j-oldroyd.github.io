Solution 4.3.2.1.

We need to find the critical points, which means we need to find \(G'(x)\text{:}\)

\begin{equation*} G'(x) = \frac{(x-1)(2x) - x^{2}}{(x-1)^{2}} = \frac{x(x-2)}{(x-1)^{2}}. \end{equation*}

So the critical points are \(x=0,1,2\text{.}\) Note that \(x=1\) cannot be a local extreme value of \(G\) since it's not in the domain of \(G\text{.}\) However, we still need to include it in our sign chart. If we do so, we find that \(G\) has a local minimum at \(x=2\) and a local maximum at \(x=0\) by the first derivative test.
