One way to do this is just start by finding \(f^{-1}\text{.}\) So we'll set \(y = \sqrt{x-1}\text{,}\) solve for \(x\) and then switch \(x\) and \(y\text{.}\) If we do this, we get

\begin{equation*} y = x^{2} + 1 \end{equation*}

so \(f^{-1}(x) = x^{2} + 1\) (and note that the domain of \(f^{-1}\) is \([0,\infty)\) since this is the range of \(f\text{!}\)). Hence

\begin{equation*} \dv{}{x}f^{-1}(x) = 2x. \end{equation*}
