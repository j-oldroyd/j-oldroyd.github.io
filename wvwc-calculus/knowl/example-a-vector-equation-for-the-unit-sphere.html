Example 11.2.12. A vector equation for the unit sphere.

Using the concept of a unit vector, we can very easily describe the unit sphere 2  using a vector equation. If we set \(\vv{r} = (x,y,z)\text{,}\) then the unit sphere is just the set of all solutions of

\begin{equation*} \norm{\vv{r}} = 1. \end{equation*}
