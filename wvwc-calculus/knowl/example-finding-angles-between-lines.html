Example 10.3.9. Finding angles between lines.

Consider the lines \(x+2y = 7\) and \(5x-y = 2\) in \(\RR^{2}\text{,}\) plotted below:

Suppose we want to find the acute angle \(\theta\) these lines make with each other. We can do so by finding vectors \(\vv{a}\) and \(\vv{b}\) that point in the same directions as these lines. We'll start by finding the vector \(\vv{a}\) which points in the same direction as the line \(x+2y=7\text{.}\) To do so, we need two points on this line: \(A_{1} = (7,0)\) and \(A_{2} = (5,1)\text{.}\) So we can take \(\vv{a}\) to be

\begin{equation*} \vv{a} = \overrightarrow{A_{1}A_{2}} = \dotprod{-2,1}. \end{equation*}

Similarly, since \(B_{1} = (0,-2)\) and \(B_{2} = (1,3)\) both lie on \(5x-y = 2\text{,}\) we can take

\begin{equation*} \vv{b} = \overrightarrow{B_{1}B_{2}} = \dotprod{1,5}. \end{equation*}

Then by TheoremÂ 10.3.5 we know that

\begin{equation*} \cos\theta = \frac{\vv{a}\cdot\vv{b}}{|\vv{a}||\vv{b}|} = \frac{3}{\sqrt{5}\sqrt{26}} = \frac{3}{\sqrt{130}}. \end{equation*}

So the acute angle between these two lines is given by

\begin{equation*} \theta = \cos^{-1}\frac{3}{\sqrt{130}}. \end{equation*}
