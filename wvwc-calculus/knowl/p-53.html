Let \(f(x)\) be a function defined on a set \(D\) and let \(c\) be a limit point of \(D\text{.}\)

This means that there are points in \(D\) distinct from \(c\) that get arbitrarily close to \(c\text{.}\)

Then \(\lim_{x\to c}f(x) = L\) if and only if for every \(\varepsilon gt; 0\) there exists some \(\delta > 0\) such that \(\abs{f(x) - L} < \varepsilon\) whenever \(\abs{x - c} < \delta\text{.}\)
