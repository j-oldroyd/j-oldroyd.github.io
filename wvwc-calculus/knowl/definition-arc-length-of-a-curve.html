Definition 8.4.3. Arc Length of a Curve.

Let $y = f(x)$ denote a continuous function defined on some interval $[a,b]\text{.}$ The arc length of $f(x)$ over this interval is defined to be

\begin{equation*} L = \lim_{\Delta x\to0}\sum_{i=1}^{n}\sqrt{1 + f'(x_{i})^{2}}\Delta x = \int_{a}^{b}\sqrt{1 + f'(x)^{2}}\,dx. \end{equation*}
