Consider the function \(f(x,y) = \frac{1}{2}(x^{2} + y^{2})\text{.}\) The gradient of this function is \(\grad f = \dotprod{x,y}\text{.}\) We can view the gradient as a function that assigns the vector \(\grad f(x,y)\) to each point \((x,y)\) in the plane. We often draw these vectors so that they're rescaled or normalized.

Such functions are the main object of this section.
