This is an optimization problem since we're trying to maximize distance. There are two things we need to do:

  1. Find the function we need to optimize.
  2. Find the extrema.

The function we need to optimize is just the distance function. In particular, if \((x,y)\) is a point on the ellipse then we need to maximize

\begin{equation*} d = \sqrt{(x-1)^{2} + (y-0)^{2}}. \end{equation*}
