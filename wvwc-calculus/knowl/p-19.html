We need to find where both the radicand \(x^{2} - 3x + 2\) is nonnegative and where \(3x + 4 \neq 0\text{.}\) If we solve \(x^{2} - 3x + 2 \geq 0\text{,}\) we see that \(x\) must be in \((-\infty, 1]\cup [2,\infty)\text{.}\) Likewise, \(3x + 4\neq 0\) if and only if \(x \neq -\frac{4}{3}\text{.}\) Hence the domain is

\begin{equation*} (-\infty, -\frac{4}{3})\cup (-\frac{4}{3}, 1]\cup[2,\infty)\text{.} \end{equation*}
