Solution 11.1.3.1

We need to sketch \(z = \sqrt{4x^{2} + y^{2}}\text{.}\) We can start by squaring both sides to get \(z^{2} = 4x^{2} + y^{2}\text{.}\) If we look at the right hand side, it looks a lot like an ellipse, so we'll try to work with that. Assuming \(z\neq0\text{,}\) we can rewrite the equation to get

\begin{equation*} \frac{x^{2}}{z^{2}/4} + \frac{y^{2}}{z^{2}} = 1 \end{equation*}

For any given positive value of \(z\text{,}\) this represents an ellipse that extends to \(\frac{z}{2}\) in the \(x\) direction and \(z\) in the \(y\) direction. So as \(z\) increases, these ellipses increase in size as well.
