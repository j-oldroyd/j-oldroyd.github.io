Definition 9.8.3 Curvature

Let \(\vv{r}(t)\) denote a smooth curve on some interval \(I\text{.}\) The curvature of \(\vv{r}(t)\) on \(I\) is defined to be the function \(\kappa(t)\) given by

\begin{equation*} \kappa(t) = \frac{|\vv{T}'(t)|}{|\vv{r}'(t)|}, \end{equation*}

where \(\vv{T}(t) = \frac{\vv{r}'(t)}{|\vv{r}'(t)|}\) is the unit tangent to the curve.
