Now we need to find a potential function \(f(x,y)\text{.}\) Since \(f_{x} = (1 + xy)e^{xy}\text{,}\) we can integrate with respect to \(x\) to get

\begin{equation*} f(x,y) = xe^{xy} + g(y)\text{.} \end{equation*}

Now differentiate with respect to \(y\) to get

\begin{equation*} x^{2}e^{xy} = x^{2}e^{xy} + g'(y) \implies g(y) = C\text{.} \end{equation*}

Hence a potential function for \(\vb{F}\) is \(f = xe^{xy}\text{,}\) and so

\begin{equation*} \int_{C}(1 + xy)e^{xy}\,dx + x^{2}e^{xy}\,dy = -1\text{.} \end{equation*}
