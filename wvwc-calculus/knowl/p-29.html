Given a one-to-one function \(f:X\to Y\) (where \(Y\) is the range), the inverse function is the unique function \(f^{-1}:Y\to X\) such that \(f^{-1}(y) = x\) if and only if \(f(x) = y\text{.}\) Equivalently,

\begin{equation*} f^{-1}(f(x)) = x \quad\text{for all}x\in X\quad\text{and}f(f^{-1}(y)) = y\quad\text{for all }y\in Y\text{.} \end{equation*}
