Proof

First, set

\begin{equation*} g(h) = f(x_{0}+ha, y_{0} +hb) \end{equation*}

Then by the Calculus I definition of the derivative,

\begin{equation*} g'(0) = \lim_{h\to0}\frac{g(h) - g(0)}{h} = \lim_{h\to0}\frac{f(x_{0}+ha, y_{0}+hb) - f(x_{0},y_{0})}{h} = D_{\vb{u}}f(x_{0},y_{0}). \end{equation*}

On the other hand, if we set \(x = x_{0} + ha\) and \(y = y_{0} + hb\text{,}\) then \(g(h) = f(x,y)\text{.}\) We can then find \(g'(h)\) using the chain rule from SectionÂ 3.5 to get

\begin{equation*} g'(h) = f_{x}\dv{x}{h} + f_{y}\dv{y}{h} = f_{x}(x,y)a + f_{y}(x,y)b \end{equation*}

So \(g'(0) = f_{x}(x_{0},y_{0})a + f_{y}(x_{0},y_{0})b.\) Therefore

\begin{equation*} D_{\vb{u}}f(x_{0},y_{0}) = f_{x}(x_{0},y_{0})a + f_{y}(x_{0},y_{0})b. \end{equation*}
