Example 11.2.5. A vector equation.

We can use vectors to describe curves and surfaces. For example, let \(\vv{r} = \dotprod{x,y,z}\) and \(\vv{r}_{0} = \dotprod{a,b,c}\text{.}\) Let \(r\geq0\text{.}\) Then the set of all points \((x,y,z)\) that satisfy the equation

\begin{equation*} \norm{\vv{r}-\vv{r}_{0}} = r \end{equation*}

has a very nice description: it's just the sphere of radius \(r\) centered at \((a,b,c)\text{.}\)
