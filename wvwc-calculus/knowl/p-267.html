This differential equation is a mathematical model that allows us to predict the growth of \(N(t)\text{.}\) We can actually use it to get an expression for \(N(t)\text{.}\) This differential equation is saying that \(N(t)\) is a function whose derivative looks quite a bit like \(N(t)\text{.}\) In fact, the solution of this differential equation is \(N(t) = Ce^{kt}\text{,}\) where \(C = N(0)\) is a constant that represents the number of people infected at time \(t=0\text{.}\) This is an example of exponential growth, and the proportionality constant \(k\) is called the relative growth rate.
