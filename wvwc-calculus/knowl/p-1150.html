It's usually easier to use Definition 10.3.2 to compute dot products. However, Theorem 10.3.5 gives us extremely useful geometric information about the dot product. For example, we get the following result very quickly: two vectors \(\vv{a}\) and \(\vv{b}\) are perpendicular if and only if \(\vv{a}\cdot\vv{b} = 0\text{.}\)
