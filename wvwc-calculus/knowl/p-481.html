Let \(f\) be some function continuous on \([a,b]\text{.}\) Then we can say that \(f\) must be continuous on every interval of the form \([a,x]\) for \(x\) in \([a,b]\text{.}\) This means that the definite integrals \(\int_{a}^{x}f(t)\,dt\) exist, and so we can use these to define a function, say \(g(x)\text{.}\)
