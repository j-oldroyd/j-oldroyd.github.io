In order to actually verify that \(V(2,2,1) = 4\) is actually the largest possible volume we can obtain, we need to show that it's actually a maximum and not a minimum. This means we need to find another point \((x,y,z)\) that also satisfies \(A = 12\text{,}\) and show that it gives a smaller value for \(V\text{.}\) One such point is just \((-2,-2,-1)\text{,}\) and \(V(-2,-2,-1) = -4 < V(2,2,1)\text{.}\) So \(4\) is actually a maximum value and not a minimum value.
