What we need to do is assume

\begin{equation*} \frac{1}{(x^{2}-1)^{2}} = \frac{A_{1}}{x-1} + \frac{A_{2}}{x+1} + \frac{A_{3}}{(x-1)^{2}} + \frac{A_{4}}{(x+1)^{2}}. \end{equation*}

If we don't account for each power, we're not guaranteed a partial fraction decomposition. Now let's clear fractions:

\begin{equation} 1 = A_{1}(x-1)(x+1)^{2} + A_{2}(x+1)(x-1)^{2} + A_{3}(x+1)^{2} + A_{4}(x-1)^{2}.\tag{6.3.2} \end{equation}

Quickly, using \(x=-1\) we find that \(A_{4} = \frac{1}{4}\text{,}\) and using \(x = 1\) we find that \(A_{3} = \frac{1}{4}\text{.}\)
