Example 11.4.10. Testing if vectors are coplanar.

We say that three vectors \(\vv{u},\vv{v}\) and \(\vv{w}\) are coplanar if they can all lie in a single plane. For example, \(\vv{i},\vv{k}\) and \(\dotprod{-4,0,56}\) are coplanar since they lie in the \(xz\)-plane, but \(\vv{i},\vv{j}\) and \(\vv{k}\) are not coplanar. Suppose we're given \(\vv{u} = \dotprod{-3,2,1},\vv{v} = \dotprod{1,2,0}\) and \(\vv{w} = \dotprod{1,1,1}\text{.}\) These vectors are coplanar if and only if the parallelepiped determined by these vectors has zero volume (i.e. is flat). Since

\begin{equation*} \vv{u}\cdot(\vv{v}\times\vv{w}) = -9, \end{equation*}

these vectors are not coplanar.
