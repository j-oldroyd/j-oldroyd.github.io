The gradient is also useful for telling us where a function is increasing and decreasing the greatest. By (12.6.1), we can write

\begin{equation*} D_{\vv{u}}f = \grad f\cdot\frac{\vv{u}}{|\vv{u}|} = |\grad f| \cos\theta, \end{equation*}

where \(\theta\) is the acute angle between \(\grad f\) and \(\vv{u}\text{.}\) It follows that

\begin{equation*} -|\grad f| \leq D_{\vv{u}}f \leq |\grad f|. \end{equation*}
