To use (6.1.1), we need to write \(\int x\sin x\,dx\) in the form \(\int u\,dv\text{.}\) Once we have \(u\) and \(dv\text{,}\) then we can set up the rest of the formula. One thing that can make this method tricky is the fact that we can have many different choices for \(u\) and \(dv\text{;}\) the only thing we need for certain is for \(u\,dv = x\sin x\) here. However, if we look at the formula (6.1.1) then we'll see two new terms on the right hand side: \(du\) and \(v\text{,}\) together in the second integral. This means that we'll need to differentiate \(u\) and integrate \(v\text{,}\) and this will suggest what both should be.
