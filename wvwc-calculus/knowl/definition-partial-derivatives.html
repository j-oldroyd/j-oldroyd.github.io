Definition 12.3.1. Partial Derivatives.

Let $f(x,y)$ be a function of two variables. The partial derivative of $f$ with respect to $x$ is the function

\begin{equation*} \pdv{f}{x} = f_{x}(x,y) = \lim_{h\to0}\frac{f(x+h,y)-f(x,y)}{h}\text{,} \end{equation*}

assuming that this limit exists. Similarly, the partial derivative of $f$ with respect to $y$ is the function

\begin{equation*} \pdv{f}{y} = f_{y}(x,y) = \lim_{h\to0}\frac{f(x,y+h)-f(x,y)}{h}\text{,} \end{equation*}

assuming that this limit exists.
