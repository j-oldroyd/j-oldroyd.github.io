Since \(\frac{1}{(x^{2} - 1)^{2}} = \frac{1}{(x-1)^{2}(x+1)^{2}}\text{,}\) it's tempting to write

\begin{equation*} \frac{1}{(x^{2}-1)^{2}} = \frac{A_{1}}{x-1} + \frac{A_{2}}{x+1}. \end{equation*}

Unfortunately, doing so gives \(A_{1} = A_{2} = 0\text{,}\) which obviously won't work. If we instead use

\begin{equation*} \frac{1}{(x^{2}-1)^{2}} = \frac{A_{1}}{(x-1)^{2}} + \frac{A_{2}}{(x+1)^{2}}, \end{equation*}

we get \(A_{1} = A_{2} = \frac{1}{4}\text{.}\) However,

\begin{equation*} \frac{1}{(x^{2}-1)^{2}} \neq \frac{\frac{1}{4}}{(x-1)^{2}} + \frac{\frac{1}{4}}{(x+1)^{2}}. \end{equation*}

What we need to do is assume

\begin{equation*} \frac{1}{(x^{2}-1)^{2}} = \frac{A_{1}}{x-1} + \frac{A_{2}}{x+1} + \frac{A_{3}}{(x-1)^{2}} + \frac{A_{4}}{(x+1)^{2}}. \end{equation*}

If we don't account for each power, we're not guaranteed a partial fraction decomposition. Now let's clear fractions:

\begin{equation} 1 = A_{1}(x-1)(x+1)^{2} + A_{2}(x+1)(x-1)^{2} + A_{3}(x+1)^{2} + A_{4}(x-1)^{2}.\tag{7.3.2} \end{equation}

Quickly, using \(x=-1\) we find that \(A_{4} = \frac{1}{4}\text{,}\) and using \(x = 1\) we find that \(A_{3} = \frac{1}{4}\text{.}\)

At this point there are no nice values left to plug into the right hand side of (7.3.2), so we need to find \(A_{1}\) and \(A_{2}\) another way. One possibility is to differentiate both sides to make the right hand side simpler, which gives

\begin{equation} 0 = A_{1}(x+1)^{2} + 2A_{1}(x^{2}-1) + A_{2}(x-1)^{2} + 2A_{2}(x^{2} - 1) + 2A_{3}(x+1) + 2A_{4}(x-1).\tag{7.3.3} \end{equation}

Now, entering \(x = -1\) into (7.3.3) gives

\begin{equation*} 0 = 4A_{2} - 4A_{4} \end{equation*}

and so \(A_{2} = A_{4} = \frac{1}{4}\text{.}\) Similarly, \(A_{1} = - A_{3} = -\frac{1}{4}\text{.}\) Therefore

\begin{equation*} \frac{1}{(x^{2} - 1)}^{2} = \frac{1}{4}\left[-\frac{1}{x-1} + \frac{1}{x+1} + \frac{1}{(x-1)^{2}} + \frac{1}{(x+1)^{2}}\right]. \end{equation*}
