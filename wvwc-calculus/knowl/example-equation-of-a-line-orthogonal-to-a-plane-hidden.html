Suppose we're given the plane \(3x-5z+2 = 0\text{,}\) and we want to find the equation of the line orthogonal to this plane and passing through the point \((3,-1,2)\text{.}\) Then the equation of this line is

\begin{equation*} \vv{r} = \vv{r}_{0}+t\vv{v} \end{equation*}

where \(\vv{r}_{0} = \dotprod{3,-1,2}\) and \(\vv{v}\) is a vector parallel to the line. Since the line must be orthogonal to the plane, we can take \(\vv{v} = \dotprod{3,0,-5}\text{.}\) So the equation of this line is

\begin{equation*} \vv{r} = \dotprod{3,-1,2}+\dotprod{3t,0,-5t}. \end{equation*}
