Solution 12.1.2.1.

We'll break this down into pieces. First, since we're dealing with square roots in real numbers we need \(x^{2}+y^{2}\leq4\text{.}\) Second, we need to make sure that \(\sqrt{4 - x^{2} - y^{2}}\neq0\text{.}\) So the domain of \(f\) is the set of all points \((x,y)\) in \(\RR^{2}\) such that \(x^{2} + y^{2} < 4\text{.}\) In other words, the domain is just the inside of the circle of radius \(2\) centered at the origin.
