Solution 5.4.3.1

By the Fundamental Theorem of Calculus, we have

\begin{equation*} erf'(x) = \frac{2}{\sqrt{\pi}}e^{-x^{2}}. \end{equation*}

Note that \(\erf'(x)\to0\) as \(x\to\infty\text{,}\) which suggests that the error function "levels out" as \(x\) increases, which is in fact true.
