Solution 10.1.4.1.

We can do this without too much trouble if we remember that the \(xz\)-plane is just the set of all points with \(y\)-coordinate equal to \(0\text{.}\) So the intersection of this sphere with the \(xz\)-plane traces out the curve \((x+2)^{2} + (0-3)^{2} + z^{2} = 9\) in the \(xz\)-plane, which is just the point \((-2,0,0)\text{.}\)
