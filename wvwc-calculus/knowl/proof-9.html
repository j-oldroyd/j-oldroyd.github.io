Proof

It might not seem like this statement requires a proof, but it does! We can check very easily that every function of the form \(F(x)+C\) is an antiderivative of \(f(x)\text{,}\) but how do we know every antiderivative takes this form? To prove this, let \(G(x)\) be an arbitrary antiderivative of \(f(x)\text{.}\) Then we need to show that \(G(x) = F(x) + C\text{,}\) or equivalently that \(G(x) - F(x) = C\text{.}\) We can do this by taking the derivative of \(G(x) - F(x)\) to get

\begin{equation*} G'(x) - F'(x) = f(x) - f(x) = 0, \end{equation*}

and so by TheoremÂ 5.2.5 we know that \(G(x) - F(x)\) must be constant on \(I\text{.}\) Hence \(G(x) - F(x) = C\) for some constant \(C\text{,}\) or equivalently \(G(x) = F(x)+C\text{.}\) Therefore every antiderivative of \(f(x)\) can be written as \(F(x) + C\text{.}\)
