We have two cases we need to consider, since the domain of \(f(x)\) consists of two intervals. First, suppose that \(x\) is in \((0,\infty)\text{.}\) Then we know that \(\dv{}{x}\ln x = \frac{1}{x}\text{,}\) so on this interval the most general antiderivative of \(\frac{1}{x}\) is \(\ln x + C_{1}\text{.}\)

Now let \(x\) be in \((-\infty,0)\text{.}\) Then \(\ln x\) isn't defined here. However, we can write

\begin{equation*} \frac{1}{x} = -\frac{1}{-x}, \end{equation*}

and if \(x < 0\) then \(-x > 0\) and \(\ln(-x)\) is defined, and in fact

\begin{equation*} \dv{}{x}\ln(-x) = -\frac{1}{(-x)} = \frac{1}{x}. \end{equation*}

So the most general antiderivative of \(\frac{1}{x}\) on \((-\infty,0)\) is \(\ln(-x) + C_{2}\text{.}\) Putting this all together, we can say that the most general antiderivative of \(f(x) = \frac{1}{x}\) is given by

\begin{equation*} F(x) = \begin{cases} \ln x + C_{1} & x > 0 \\ \ln(-x) + C_{2} & x < 0\end{cases}. \end{equation*}

If we know that we're only selecting values of \(x\) with the same sign, then we can just say that the antiderivative of \(\frac{1}{x}\) is \(\ln|x| + C\text{.}\)
