Solution 8.2.8.1.

First, we need to write \(.9\ldots\) as a geometric series. We can do so as follows:

\begin{equation*} .9\ldots = \frac{9}{10} + \frac{9}{100} + \cdots\text{,} \end{equation*}

and so we see that

\begin{equation*} .9\ldots = \sum_{k=1}^{\infty}\frac{9}{10^{k}}\text{.} \end{equation*}

This is a geometric series with \(a = \frac{9}{10}\) and \(r = \frac{1}{10}\) (and so is convergent!), and so

\begin{equation*} \sum_{k=1}^{\infty}\frac{9}{10^{k}} = \frac{9/10}{1 - \frac{1}{10}} = 1\text{.} \end{equation*}
