An easier way to find a normal vector in the last example would be to use our knowledge from SectionÂ 11.5. However, the following approach is more general. If \(S\) is described by the equation \(F(x,y,z) = 0\text{,}\) then \(\grad\vb{F}\) is guaranteed to be normal to \(S\text{.}\) This is especially useful if \(S\) can be described by an equation of the form \(z = f(x,y)\text{,}\) as above. To see how, note that

\begin{align*} x + y + z \amp = 1 \\ \amp \implies F(x,y,z) = 1 - x - y - z \\ \amp \implies \grad\vb{F} = \dotprod{-1, -1, -1} \text{.} \end{align*}
