To get an accurate expression for \(h(t)\text{,}\) we go back to \(h''(t) = -9.8\text{.}\) Then \(h'(t) = -9.8t+C_{1}\) for some constant \(C_{1}\text{.}\) Since \(h'(0) = 0\text{,}\) this forces \(C_{1} = 0\text{.}\) So we have \(h'(t) = -9.8t\text{.}\) Now we antidifferentiate one more time to get \(h(t) = -4.9t^{2} + C_{2}\text{.}\) Since \(h(0) = 100\text{,}\) this forces \(C_{2} = 100\text{.}\) So

\begin{equation*} h(t) = -4.9t^{2} + 100\text{.} \end{equation*}
