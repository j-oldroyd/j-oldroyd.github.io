Theorem 10.7.7. Vector Derivative Rules.

Let \(\vv{u}\) and \(\vv{v}\) be differentiable vector functions, \(c\) be a scalar and let \(f(t)\) be a differentiable (scalar) function. Then the following formulas hold:

  1. \(\dv{}{t}[\vv{u}(t)+\vv{v}(t)] = \vv{u}'(t)+\vv{v}'(t)\)
  2. \(\dv{}{t}[c\vv{u}(t)] = c\vv{u}'(t)\)
  3. \(\dv{}{t}[f(t)\vv{u}(t)] = f'(t)\vv{u}(t)+f(t)\vv{u}'(t)\)
  4. \(\dv{}{t}[\vv{u}(t)\cdot\vv{v}(t)] = \vv{u}'(t)\cdot\vv{v}(t)+\vv{u}(t)\cdot\vv{v}'(t)\)
  5. \(\dv{}{t}[\vv{u}(t)\times\vv{v}(t)] = \vv{u}'(t)\times\vv{v}(t)+\vv{u}(t)\times\vv{v}'(t)\)
  6. \(\dv{}{t}\vv{u}(f(t)) = f'(t)\vv{u}'(f(t))\)
