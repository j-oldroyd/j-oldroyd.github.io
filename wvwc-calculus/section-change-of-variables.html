Section 13.8 Change of Variables

The goal now is to determine a general method to change coordinates in multiple integrals. We'll start with double integrals.

Subsection 13.8.1 Change of Variables in Double Integrals

Any change of coordinates involves a transformation between new variables \(u,v\) and the original variables \(x,y\text{.}\) We indicate this as follows:

\begin{equation*} x = g(u,v) \qq{and} y = h(u,v)\text{.} \end{equation*}

In Section 13.3, the transformation was just

\begin{equation*} x = g(r,\theta) = r\cos\theta \qq{and} y = h(r,\theta) = r\sin\theta\text{.} \end{equation*}

So the goal behind integrating with change of variables is to find relate the integral \(\iint_{D}f(x,y)\,dA\text{,}\) where \(D\) is in the \(xy\)-plane, to a new integral \(\iint_{R}f(g(u,v),h(u,v))\,dA\text{,}\) where \(R\) is in the \(uv\)-plane. The main issue with such a transformation is one that we're familiar with from working with polar coordinates: \(dA = dx\,dy\) in the first set of coordinates is not necessarily equal to \(du\,dv\) in the new set of coordinates. In general we'll need a scaling factor that depends on \(u\) and \(v\text{.}\) As it turns out, the proper scaling factor comes from a determinant.

Definition 13.8.1. Jacobian.

The Jacobian determinant or Jacobian of the coordinate transformation \(x = g(u,v)\) and \(y = h(u,v)\) is the quantity

\begin{equation*} J(u,v) = \left|\begin{array}{cc} x_{u} \amp x_{v} \\ y_{u} \amp y_{v}\end{array}\right|\text{.} \end{equation*}

Example 13.8.2. Jacobian in Polar Coordinates.

Determine \(J(r,\theta)\) for the transformation to polar coordinates.

Solution.

We have

\begin{align*} x_{r} \amp = \cos\theta \\ x_{\theta} \amp = -r\sin\theta \\ y_{r} \amp = \sin\theta \\ y_{\theta} \amp = r\cos\theta \text{.} \end{align*}

Hence

\begin{equation*} J(r,\theta) = r\cos^{2}\theta + r\sin^{2}\theta = r\text{.} \end{equation*}

The Jacobian helps us in the following way: when making the change of variables \(x = g(u,v), y= h(u,v)\text{,}\) the area element \(dA\) becomes \(dA = |J(u,v)|\,du\,dv\text{.}\)

Example 13.8.3.

Compute \(J(u,v)\) for the transformation

\begin{equation*} u = x + 4y \qq{and} v = 3x + 2y\text{.} \end{equation*}
Solution.

We need to get \(x\) and \(y\) in terms of \(u\) and \(v\) first. Since

\begin{equation*} 3u - v = 10y \qq{and} 2v - u = 5x\text{,} \end{equation*}

it follows that

\begin{equation*} x = \frac{2v - u}{5} \qq{and} y = \frac{3u - v}{10}\text{.} \end{equation*}

Hence

\begin{equation*} J(u,v) = -\frac{1}{10}\text{.} \end{equation*}

Example 13.8.4. Change of Variables in Two Dimensions.

Find the volume of the region under the surface \(z = \sqrt{y^{2} - x^{2}}\) and over the region \(R\) bounded by \(y - x = 0, y - x = 2, y + x = 0\text{,}\) and \(y + x = 2\text{.}\)

Solution.

Both the integrand and the region of integration are awful here, but if we set \(u = y - x\) and \(v = y + x\) then the limits of integration become very simple:

\begin{equation*} 0 \leq u,v \leq 2\text{.} \end{equation*}

See Figure 13.8.5. The integrand gets better too: \(\sqrt{y^{2} - x^{2}} = \sqrt{uv}\text{.}\) So we can write

\begin{equation*} \iint_{R}\sqrt{y^{2} - x^{2}}\,dA = \int_{0}^{2}\int_{0}^{2}\sqrt{uv}|J(u,v)|\,du\,dv\text{.} \end{equation*}

Essentially, we're moving from integrating over a diamond in the \(xy\)-plane to integrating over a square in the \(uv\)-plane. If we can find \(J(u,v)\text{,}\) we can start computing the integral.

To find \(J(u,v)\text{,}\) we need to get \(x,y\) in terms of \(u,v\) in order to use Definition 13.8.1. Since \(u + v = 2y\) and \(u - v = -2x\text{,}\) we get

\begin{equation*} x = \frac{v - u}{2} \qq{and} y = \frac{v + u}{2}\text{.} \end{equation*}

Hence

\begin{equation*} J(u,v) = \left|\begin{array}{cc}-\frac{1}{2} \amp \frac{1}{2} \\ \frac{1}{2} \amp \frac{1}{2}\end{array}\right| = -\frac{1}{2}\text{,} \end{equation*}

and we have

\begin{align*} \iint_{R}\sqrt{y^{2} - x^{2}}\,dA \amp = \frac{1}{2}\int_{0}^{2}\int_{0}^{2}\sqrt{uv}\,du\,dv \\ \amp = \frac{1}{2}\left(\int_{0}^{2}\sqrt{u}\,du\right)^{2} \\ \amp = \frac{16}{9} \text{.} \end{align*}

Example 13.8.6. Integrating Between Hyperbolas.

Compute \(\iint_{R}e^{xy}\,dA\) where \(R\) is the region bounded between the curves \(xy = 1\text{,}\) \(xy = 4\text{,}\) \(y = x\) and \(y = 3x\) in the first quadrant.

Solution.

The integrand doesn't look too bad at first, but the region of integration would be very annoying here:

To deal with this, we'll use the change of variables \(u = xy\) and \(v = \frac{y}{x}\text{.}\) The corresponding limits are \(1\leq u\leq 4\) and \(1 \leq v\leq 3\text{,}\) and our integral becomes

\begin{equation*} \iint_{R} e^{xy}\,dA = \int_{1}^{3}\int_{1}^{4}e^{u}\abs{J(u,v)}\,du\,dv\text{.} \end{equation*}

We also get that the Jacobian is \(J(u,v) = \frac{1}{2v}\text{.}\) Therefore

\begin{align*} \iint_{R}e^{xy}\,dA \amp = \int_{1}^{3}\int_{1}^{4}\frac{1}{2v}e^{u}\,du\,dv \\ \amp = \frac{e^{4} - e}{2}\int_{1}^{3}\frac{1}{v}\,dv \\ \amp = \frac{e^{4} - e}{2}\ln 3 \text{.} \end{align*}

Subsection 13.8.2 Change of Variables in Triple Integrals

What we did in Subsection 13.8.1 carries over directly to triple integrals and beyond. We just need to compute \(3\times3\) determinants instead of \(2\times2\text{.}\)

Example 13.8.8. Volume of a Region Between Planes.

Find the volume of the region bounded by the planes \(y - 2x = 0\text{,}\) \(y - 2x = 1\text{,}\) \(z - 3y = 0\text{,}\) \(z - 3y = 1\text{,}\) \(z - 4x = 0\) and \(z - 4x = 3\text{.}\)

Solution.

If we let \(D\) denote the region we're trying to find the volume of, then the volume of this region is just

\begin{align*} \iiint_{D}\,dV \amp = \int_{0}^{3}\int_{0}^{1}\int_{0}^{1}\abs{J(u,v,w)}\,du\,dv\,dw \\ \amp = \frac{1}{2}\int_{0}^{3}\int_{0}^{1}\int_{0}^{1}\,du\,dv\,dw \text{,} \end{align*}

where \(u = y - 2x\text{,}\) \(v = z - 3y\) and \(w = z - 4x\text{.}\)

Example 13.8.9. Transforming to a Spherical Region.

Evaluate

\begin{equation*} \iiint_{D}\abs{xyz}\,dV \end{equation*}

where

\begin{equation*} D = \set{(x,y,z) : x^{2} + \frac{y^{2}}{9} + \frac{z^{2}}{4} \leq 1}\text{.} \end{equation*}

Problem adapted from Exercise 21 on page 1137 in Thomas' Calculus, \(11^{\text{th}}\) edition.

Solution.

The region of integration is an ellipsoid, but if we can change this to a sphere then we can use spherical coordinates to help us. A quick fix is to set

\begin{equation*} u = x, v = \frac{y}{3} \qq{and} w = \frac{z}{2}\text{.} \end{equation*}

Then these new variables satisfy

\begin{equation*} u^{2} + v^{2} + w^{2} \leq 1 \end{equation*}

and so the corresponding region \(S\) in \(uvw\)-space is the unit sphere.

Now we need to find \(J(u,v,w)\text{.}\) Thankfully, this is straightforward and we get \(\abs{J(u,v,w)} = \frac{1}{6}\text{.}\) Therefore

\begin{align*} \iiint_{D}\abs{xyz}\,dV \amp = \iiint_{S}\abs{uvw}\,du\,dv\,dw \\ \amp = 8\iiint_{S'}\abs{uvw}\,du\,dv\,dw \\ \amp = 8\int_{0}^{\frac{\pi}{2}}\int_{0}^{\frac{\pi}{2}}\int_{0}^{1}(\rho^{3}\sin^{2}\varphi\cos\varphi\sin\theta\cos\theta)\rho^{2}\sin\varphi\,d\rho\,d\varphi\,d\theta \\ \amp = 8\int_{0}^{\frac{\pi}{2}}\int_{0}^{\frac{\pi}{2}}\int_{0}^{1}\rho^{5}\sin^{3}\varphi\cos\varphi\sin\theta\cos\theta\,d\rho\,d\varphi\,d\theta\text{,} \end{align*}

which we can actually integrate without too much trouble.
