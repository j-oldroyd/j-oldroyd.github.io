Section 4.2 Logarithms

Subsection 4.2.1 Inverse Function Review

Suppose we want to solve the equation \(y = f(x)\) for \(x\text{.}\) Then we can only do so unambiguously if \(f\) is an invertible function.

Definition 4.2.1. Invertible Functions.

A function \(f(x)\) is invertible if and only if it is a one-to-one function. That is, \(f(x_{1}) = f(x_{2})\) forces \(x_{1} = x_{2}\text{.}\)

Since a function is one-to-one if and only if its graph passes the horizontal line test, the horizontal line test also gives us a useful way to check if a function is invertible.

Example 4.2.2.

Let \(f(x) = e^{x}\text{,}\) the natural exponential function introduced in Section 4.1. Then \(f(x)\) is an invertible function, since its graph passes the horizontal line test.

If a function \(f\) is invertible, then there exists an inverse function \(f^{1}\) satisfying the following equations:

\begin{align*} f(f^{-1}(y)) & = y \\ f^{-1}(f(x)) & = x. \end{align*}

Essentially, \(f^{-1}\) is the rule that undoes the transformation that \(f\) applies to \(x\text{.}\) The domain of \(f^{-1}\) is the range of \(f\) and the range of \(f^{-1}\) is the domain of \(f\text{.}\) Note that \(f^{-1}\neq\frac{1}{f}\text{!.}\)

The graph of an inverse function is related to the graph of the original function in a very nice way. To get the graph of \(f^{-1}\) from \(f\text{,}\) just reflect the graph of \(f\) across the line \(y=x\text{.}\)

Subsection 4.2.2 Calculus with Inverse Functions

There are two important things to notice about the graph of the inverse function in Example 4.2.2: it's continuous and differentiable. This is because the same was true of the original graph, and if we think about it this should be true in general as well since graphing the inverse doesn't add any new gaps or cusps to the graph.

Theorem 4.2.5. Continuity and Differentiability of Inverses.

Let \(f(x)\) be a continuous (respectively, differentiable) function. Suppose that \(f(x)\) is invertible, and has inverse \(f^{-1}(x)\text{.}\) Then \(f^{-1}(x)\) is continuous (respectively, differentiable).

Example 4.2.6. Finding the derivative of an inverse function.

Let \(f(x) = \sqrt{x-1}\) and note that \(f\) is invertible. Find the derivative of \(f^{-1}\text{.}\)

Solution.

One way to do this is just start by finding \(f^{-1}\text{.}\) So we'll set \(y = \sqrt{x-1}\text{,}\) solve for \(x\) and then switch \(x\) and \(y\text{.}\) If we do this, we get

\begin{equation*} y = x^{2} + 1 \end{equation*}

so \(f^{-1}(x) = x^{2} + 1\) (and note that the domain of \(f^{-1}\) is \([0,\infty)\) since this is the range of \(f\text{!}\)). Hence

\begin{equation*} \dv{}{x}f^{-1}(x) = 2x. \end{equation*}

The method used in Example 4.2.6 will certainly work in simple cases. But what do we do if we can't (or don't want to) find an explicit formula for the inverse function? The following formula will help us to do this.

Theorem 4.2.7. Derivative of an Inverse Function.

Let \(f(x)\) be a differentiable function with inverse \(f^{-1}(x)\text{.}\) Assume that \(f^{-1}\) is itself differentiable. Then

\begin{equation*} (f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))}. \end{equation*}

Proof.

First, let \(y = f^{-1}(x)\text{,}\) so that we need to find \(y'\text{.}\) Then we can say that \(f(y) = x\text{.}\) Now differentiate this equation implicitly to get

\begin{equation*} f'(y)y' = 1 \end{equation*}

or just

\begin{equation*} y' = \frac{1}{f'(y)} = \frac{1}{f'(f^{-1}(x))}. \end{equation*}

Example 4.2.8. Exponential inverse revisited.

Using the fact that the derivative of \(e^{x}\) is itself, find the derivative of its inverse.

Solution.

Let \(f(x) = e^{x}\text{.}\) Then by Theorem 4.2.7 we have

\begin{equation*} (f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))} = \frac{1}{f(f^{-1}(x))} = \frac{1}{x}. \end{equation*}

So the derivative of the inverse of \(e^{x}\) is \(\frac{1}{x}\text{.}\)

Example 4.2.9. Derivative at a point.

Let \(f(x) = x^{3} + 3\sin x + 2\cos x\text{.}\) Find \((f^{-1})'(2)\)

Solution.

By Theorem 4.2.7, we know that

\begin{equation*} (f^{-1})'(2) = \frac{1}{f'(f^{-1}(2))}. \end{equation*}

By inspection, \(f(0) = 2\) which means \(f^{-1}(2) = 0\text{.}\) Since

\begin{equation*} f'(x) = 3x^{2} + 3\cos x - 2\sin x, \end{equation*}

this gives

\begin{equation*} (f^{-1})'(2) = \frac{1}{f'(f^{-1}(2))} = \frac{1}{f'(0)} = \frac{1}{3}. \end{equation*}

Subsection 4.2.3 Logarithms

Since exponential functions are important in mathematics and its applications, their inverses are important as well. Each exponential function \(f(x) = a^{x}\) (\(a> 0\)) has an inverse function that we call the logarithm with base \(a\), and denote by \(f^{-1}(x) = \log_{a}x\text{.}\) The domain of each logarithm is \((0,\infty)\) and the range is \((-\infty,\infty)\text{.}\) The defining properties of the logarithm function are as follows:

\begin{align*} \log_{a}a^{x} & = x \\ a^{\log_{a}x} & = x. \end{align*}

Essentially, logarithms and exponentials cancel each other out. To put this another way, \(\log_{a}x\) is the exponent needed to turn \(a\) into \(x\text{.}\)

We can say a few things about logarithms just from looking at graphs of exponentials. Suppose that \(a>1\text{.}\) Then \(\log_{a}x\) is both continuous and differentiable for all \(x>0\text{,}\)

\begin{align*} \log_{a}1 & = 0 \\ \lim_{x\to0^{+}}\log_{a}x & = -\infty \\ \lim_{x\to\infty}\log_{a}x & = \infty \end{align*}

and the derivative of \(\log_{a}x\) approaches \(0\) as \(x\to\infty\text{.}\)

The properties of the exponential listed in Theorem 4.1.1 have corresponding properties for logarithms.

Theorem 4.2.10. Properties of Logarithm Functions.

Let \(a>0\) with \(a\neq1\text{.}\) Let \(x,y\) be positive real numbers, and let \(r\) be any real number. Then

  • \(\log_{a}1 = 0\) and \(\log_{a}a = 1\text{.}\)

  • \(\log_{a}(xy) = \log_{a}x + \log_{a}y\text{.}\)

  • \(\log_{a}\frac{x}{y} = \log_{a}x - \log_{a}y\text{.}\)

  • \(\log_{a}x^{r} = r\log_{a}x\text{.}\)

An important takeaway from Theorem 4.2.10 is that logarithms turn the complicated operations of multiplication and division into the simpler operations of addition and subtraction.

Since every exponential function has a corresponding logarithm, the natural exponential function \(e^{x}\) has a logarithm as well. We call this inverse the natural logarithm and denote it by \(\ln x\text{.}\) Note that \(\dv{}{x}\ln x = \frac{1}{x}\) by Example 4.2.8.

Example 4.2.11. Simplifying an exponential.

Simplify \(e^{-4\ln x}\) and \(e^{x\ln3}\text{.}\)

Solution.

We'll simplify by using the cancellation property \(e^{\ln y} = y\text{.}\) First, we need to put the entire exponent inside of the natural log:

\begin{equation*} e^{-4\ln x} = e^{\ln x^{-4}} = x^{-4} \end{equation*}

and

\begin{equation*} e^{x\ln3} = 3^{x}. \end{equation*}

Example 4.2.11 shows us that every exponential function can be written in terms of the natural exponential: \(a^{x} = e^{x\ln a}\text{.}\) Similarly, every logarithm can be written in terms of the natural logarithm by using the change of base formula:

\begin{equation*} \log_{a}x = \frac{\ln x}{\ln a}. \end{equation*}
