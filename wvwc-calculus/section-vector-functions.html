Section 11.7 Vector Functions

Recall from Subsection 11.5.1 that the equation of a line can be written as

\begin{equation*} \vv{r} = \vv{r}_{0}+t\vv{v} = \dotprod{x_{0}+at, y_{0}+bt, z_{0}+ct}. \end{equation*}

This is our first example of a vector function. Vector functions are functions of the form

\begin{equation*} \vv{r}(t) = \dotprod{f(t),g(t),h(t)}, \end{equation*}

and graphs of vector functions are called space curves. We call \(f,g,h\) the component functions of \(\vv{r}\text{.}\) We're interested in how these curves change, which means we're interested in how to do calculus on space curves. Although these curves live in \(\RR^{3}\text{,}\) there's still only one independent variable: \(t\text{.}\) So much of what we learned in Calculus I applies to space curves.

Subsection 11.7.1 Limits with Space Curves

We can take limits with vector functions just as we can with regular functions.

Let \(\vv{r}(t) = \dotprod{f(t),g(t),h(t)}\text{.}\) Then

\begin{equation} \lim_{t\to a}\vv{r}(t) = \dotprod{\lim_{t\to a}f(t),\lim_{t\to a}g(t),\lim_{t\to a}h(t)}.\label{limits-vector-functions}\tag{11.7.1} \end{equation}

In other words, if you want to take the limit of a vector function you can just take the limits of the component functions.

Example 11.7.1. Limit of a vector function.

Let

\begin{equation*} \vv{r}(t) = \frac{t^{2}-1}{t-1}\vv{i} + \sqrt{t+8}\vv{j} - \frac{\sin\pi t}{\ln t}\vv{k}. \end{equation*}

Suppose we want to find \(\lim_{t\to 1}\vv{r}(t)\text{.}\) Then we just need to take the limit of each component. So

\begin{equation*} \lim_{t\to1}\vv{r}(t) = 2\vv{i}+3\vv{j} -\pi\vv{k}. \end{equation*}

Just as in Calculus I, we say that a vector function \(\vv{r}(t)\) is continuous at \(t=a\) if \(\lim_{t\to a}\vv{r}(t) = \vv{r}(a)\text{.}\) In general, a vector function is continuous wherever all of its components functions are continuous.

Example 11.7.2. A horiztonal helix.

Let \(\vv{r}(t) = \dotprod{\sin\pi t, t, \cos\pi t}\text{,}\) and suppose we want to sketch this function. One way to do so is to plug in values for \(t\) and connect the resulting points with a curve, but we can also do the following to get an idea of what this looks like. First, note that we have \(x = \sin\pi t\) and \(z=\cos\pi t\text{.}\) So \(x^{2}+z^{2} = 1\text{,}\) which means that this looks like the unit circle in the \(xz\)-plane, traced clockwise. Since we also have \(y=t\text{,}\) this curve moves farther along the \(y\)-axis as \(t\) increases. If we trace this out, we get a helix (see the below plot). We can also see from the graph that it has no jumps or gaps, so \(\vv{r}(t)\) is continuous everywhere.

Example 11.7.3. Finding vector functions.

Consider the cylinder \(x^{2}+y^{2}=4\) and the surface \(z=xy\text{,}\) and suppose we want to trace out there intersection with a vector function. Here's how we can do this. First, we'll come up with the \(x\) and \(y\) components of \(\vv{r}(t)\text{.}\) Since \(x^{2}+y^{2} = 4\text{,}\) this suggests that we should take

\begin{align*} x & = 2\cos t \\ y & = 2\sin t \end{align*}

So that's two down, one to go. To get \(z\text{,}\) we just need to use the equation \(z=xy\text{.}\) So

\begin{equation*} z = xy = 4\cos t\sin t. \end{equation*}

So our vector function is

\begin{equation*} \vv{r}(t) = \dotprod{2\cos t, 2\sin t, 2\sin 2t}. \end{equation*}

This is also plotted below.

Subsection 11.7.2 Derivatives with Space Curves

Now that we know how to take limits with vector functions, we can take derivatives as well.

Definition 11.7.4. Derivatives of Vector Functions.

Let \(\vv{r}(t)\) denote a vector function. The derivative of \(\vv{r}\) is the new vector function \(\vv{r}'\) given by

\begin{equation*} \vv{r}'(t) = \frac{d\vv{r}}{dt} = \lim_{h\to0}\frac{\vv{r}(t+h)-\vv{r}(t)}{h}, \end{equation*}

assuming that the limit exists. If this limit exists, we say that \(\vv{r}\) is differentiable.

If \(\vv{r}(t) = \dotprod{f(t),g(t),h(t)},\) then \(\vv{r}\) is differentiable if and only if \(f,g,h\) are, and \(\vv{r}'(t) = \dotprod{f'(t),g'(t),h'(t)}.\) Just as in Calculus I, the derivative represents how quickly a space curve is changing at some value of \(t\text{.}\) However, derivatives of vector functions also carry information about the direction a curve is moving. We call \(\vv{r}'(t)\) the tangent vector to \(\vv{r}(t)\text{.}\) In particular, \(\vv{r}'(t)\) is parallel to the space curve \(\vv{r}\) at \(t\text{,}\) and its magnitude \(\norm{\vv{r}'(t)}\) represents how quickly the curve is changing at \(t\text{.}\) If we only care about direction, then we can define the unit tangent \(\vv{T}(t)\text{,}\) which is given by

\begin{equation*} \vv{T}(t) = \frac{\vv{r}'(t)}{\norm{\vv{r}'(t)}}. \end{equation*}

We also have the usual ideas from Calculus I and physics regarding motion: velocity is the derivative of position and acceleration is the derivative of velocity.

Example 11.7.5. Velocity on a saddle.

A particle moves counterclockwise along the "saddle" \(\vv{r}(t) = \dotprod{2\cos t, 2\sin t, 2\sin 2t}\text{.}\) We want its velocity at \(t=\frac{\pi}{2}\text{.}\) First, find \(\vv{r}'\) to get

\begin{equation*} \vv{r}'(t) = \dotprod{-2\sin t,2\cos t,4\cos2t}. \end{equation*}

At \(t=\frac{\pi}{2}\text{,}\) we have the velocity vector

\begin{equation*} \vv{r}'\left(\frac{\pi}{2}\right) = \dotprod{-2,0,-4}. \end{equation*}

So at the point \(t=\frac{\pi}{2}\text{,}\) the space curve is parallel to the vector \(\dotprod{-2,0,4}\text{.}\) In other words, the particle is moving in this direction at \(t=\frac{\pi}{2}\text{.}\)

Example 11.7.6. Tangents on a circle.

A particle moves along the circle \(y^{2}+z^{2}=5\) in the \(yz\)-plane, counterclockwise and with an angular frequency of 5\(\pi\) rad⁄s. Then we can assume that its position is described by

\begin{equation*} \vv{r}(t) = \dotprod{0,\sqrt{5}\cos5\pi t, \sqrt{5}\sin5\pi t}. \end{equation*}

Suppose we want to find the direction this particle is going at any given moment. Then we can just find the unit tangent vector \(\vv{T}\text{:}\)

\begin{equation*} \vv{T}(t) = \frac{\dotprod{0,-5\sqrt{5}\sin5\pi t, 5\sqrt{5}\cos5\pi t}}{\sqrt{125}}. \end{equation*}

We also have derivative rules for vector functions, based off of the familiar formulas from Calculus I.

Theorem 11.7.7. Vector Derivative Rules.

Let \(\vv{u}\) and \(\vv{v}\) be differentiable vector functions, \(c\) be a scalar and let \(f(t)\) be a differentiable (scalar) function. Then the following formulas hold:

  1. \(\displaystyle \dv{}{t}[\vv{u}(t)+\vv{v}(t)] = \vv{u}'(t)+\vv{v}'(t)\)
  2. \(\displaystyle \dv{}{t}[c\vv{u}(t)] = c\vv{u}'(t)\)
  3. \(\displaystyle \dv{}{t}[f(t)\vv{u}(t)] = f'(t)\vv{u}(t)+f(t)\vv{u}'(t)\)
  4. \(\displaystyle \dv{}{t}[\vv{u}(t)\cdot\vv{v}(t)] = \vv{u}'(t)\cdot\vv{v}(t)+\vv{u}(t)\cdot\vv{v}'(t)\)
  5. \(\displaystyle \dv{}{t}[\vv{u}(t)\times\vv{v}(t)] = \vv{u}'(t)\times\vv{v}(t)+\vv{u}(t)\times\vv{v}'(t)\)
  6. \(\displaystyle \dv{}{t}\vv{u}(f(t)) = f'(t)\vv{u}'(f(t))\)

Subsection 11.7.3 Integrals with Space Curves

We can also integrate vector functions without too much trouble. Just as taking the derivative of a vector function reduces down to differentiating each component, integrating a vector function reduces down to integrating each component. If \(\vv{r}(t) = \dotprod{f(t),g(t),h(t)},\) then

\begin{equation} \int_{a}^{b}\vv{r}(t)\,dt = \dotprod{\int_{a}^{b}f(t)\,dt,\int_{a}^{b}g(t)\,dt,\int_{a}^{b}h(t)\,dt}.\label{equation-vector-function-integration}\tag{11.7.2} \end{equation}
