Section 7.1 Areas Between Curves

¶

We already know that \(\int_{a}^{b}f(x)\,dx\) geometrically represents the area under \(f(x)\) from \(x = a\) to \(x = b\text{.}\) Now we'd like to try to extend the integral to other geometrical concepts such as surface areas and volumes. An easy generalization we can make right now is that of areas between curves, as in Figure 7.1.1.

It may not be too much of a stretch to say that the area \(S\) should be given by \(\int_{a}^{b}[f(x) - g(x)]\,dx\text{,}\) and indeed it is, but we will prove this using Riemann sums for the reason that Riemann sums appear a lot in applications. Furthermore, Riemann sums provide us with the formula we need to evaluate integrals numerically using libraries such as NumPy. So let's start by partitioning the interval \([a,b]\) into \(n\) subintervals

\begin{equation*} [x_{i-1},x_{i}]\quad\text{for}\quad 0\leq i\leq n, \end{equation*}

where \(x_{0} = a, x_{n} = b\) and

\begin{equation*} x_{i} - x_{i-1} = \Delta x = \frac{b-a}{n}. \end{equation*}

Now we'll use the subintervals to break the region \(S\) up into rectangles to approximate the area. So choose a point \(x_{i}^{*}\) from each subinterval \([x_{i-1}, x_{i}]\) and use it to determine the height of the corresponding rectangle: \(f(x_{i}^{*}) - g(x_{i}^{*})\text{.}\) Then the area is \(f(x_{i}^{*}) - g(x_{i}^{*})\Delta x\text{,}\) and adding up the areas of all of the rectangles lets us approximate the area of \(S\text{:}\)

\begin{equation} \text{area of }S \approx \sum_{i=1}^{n}[f(x_{i}^{*}) - g(x_{i}^{*})]\Delta x\label{equation-area-between-curves-approximation}\tag{7.1.1} \end{equation}

Now send \(n\to\infty\) (equivalently, send \(\Delta x\to 0\)) to make the approximation exact: \(\text{area of }S = \lim_{\Delta x\to0}\sum_{i=1}^{n}f(x_{i}^{*}) - g(x_{i}^{*})\Delta x = \int_{a}^{b}[f(x) - g(x)]\,dx.\)

Example 7.1.2. Area Between Parabolas.

Find the area between \(f(x) = x^{2} + 1\) and \(g(x) = 2x^{2}\text{.}\)

Solution

If we graph \(f\) and \(g\text{,}\) we see that the area will be given by \(\int_{a}^{b}[(x^{2} + 1) - 2x^{2}]\,dx\text{,}\) where \(a\) and \(b\) are the values of \(x\) where \(f(x) = g(x)\text{.}\) Using a bit of algebra, we get \(a = -1\) and \(b = 1\text{.}\) Therefore the area between the parabolas is

\begin{align*} \int_{-1}^{1}(1 - x^{2})\,dx \amp = 2\int_{0}^{1}(1-x^{2})\,dx \\ \amp = \frac{4}{3}. \end{align*}

Although we found an exact solution in Example 7.1.2, we can also use this to test our NumPy skills. (7.1.1) shows us how we can use NumPy to find this quantity.

Example 7.1.3. Area with Changing Limits.

Find the area of the region bounded by \(x = y^{4}\) and \(y^{2} + x = 2\text{.}\)

Solution

If we solve both equations for \(y\text{,}\) we get \(y = \pm\sqrt[4]{x}\) and \(y = \pm\sqrt{2 - x}\text{.}\) We can then graph the region below:

To find the area between these curves, we'll need to compute two integrals: one for \(0\leq x\leq 1\) and another for \(1\leq x\leq 2\text{.}\) In particular, the area will be given by \(\int_{0}^{1}[x^{\frac{1}{4}} - (-x^{\frac{1}{4}})]\,dx + \int_{1}^{2}[\sqrt{2 - x} - (-\sqrt{2 - x})]\,dx\text{.}\) Doing so, we get \(\frac{44}{15}\) for the area between the curves.

Part of the reason why finding the area in Example 7.1.3 was more complicated was because the curves did not represent functions of \(x\text{.}\) However, we can view each curve as representing a function of \(y\text{,}\) namely, \(x(y) = y^{4} \quad\text{and}\quad x(y) = 2 - y^{2}.\) Then the area can be found as \(\int_{-1}^{1}[(2 - y^{2}) - y^{4}]\,dy = \frac{44}{15}.\) So it can be much easier if we find the area by integrating with respect to \(y\) as opposed to \(x\text{.}\)

Example 7.1.5.

Find the area contained by the curves \(x = y^{2} - 4y\) and \(x = 2y - y^{2}\text{.}\)

Solution

If we graph the region, we get Figure 7.1.6.

Finding the area by using integrals with respect to \(x\) would require setting up and solving three different integrals! So, not terrific. However, we can integrate with respect to \(y\) fairly easily. By finding where the curves intersect, we see that \(0\leq y\leq 3\text{.}\) Hence the area should be

\begin{align*} \int_{0}^{3}[(2y - y^{2}) - (y^{2} - 4y)]\,dy \amp = \int_{0}^{3}[6y - 2y^{2}]\,dy \\ \amp = 3y^{2} - \frac{2}{3}y^{3}\Bigg]_{0}^{3} \\ \amp = 27 - 18 \end{align*}

and so the area is just \(9\) units.
