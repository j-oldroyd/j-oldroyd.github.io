Section 2.6 Implicit Differentiation

¶
Example 2.6.1 Derivative of an implicit function

Consider the curve given by the equation \(x^{3} - y^{2} + \sin y = 3\text{.}\) Find the slope of this curve at the point \((\sqrt[3]{3},0)\text{.}\)

Solution

We could try to solve for \(y\) and then differentiate that to find the slope, but we have a slight problem: it's impossible, at least in terms of "elementary functions". However, we can still use the chain rule to find \(y'\text{,}\) at least in terms of \(x\) and \(y\text{.}\) We just need to remember that \(y\) is a function of \(x\text{.}\) If we differentiate \(x^{3} - y^{2} + \sin y = 3\) with respect to \(x\text{,}\) we get

\begin{equation*} 3x^{2} - 2y\dv{y}{x} + \cos y \dv{y}{x} = 0 \end{equation*}

and so

\begin{equation*} \dv{y}{x} = -\frac{3x^{2}}{\cos y - 2y}. \end{equation*}

So the slope of the curve at \((\sqrt[3]{3},0)\) is just \(-3\sqrt[3]{3}^{2}\text{.}\)

The method we used to get \(\dv{y}{x}\) in Example 2.6.1 is called implicit differentiation. It's extremely useful if we want to solve for \(\dv{y}{x}\) without first solving for \(y\text{.}\) Even if we can solve for \(y\) without too much trouble, it's often easier to find \(\dv{y}{x}\) implicitly as the next example shows.

Example 2.6.2 Implicit differentiation to save algebra

Let \(f(x) = \sqrt{1-x^{2}}\text{.}\) Find \(f'(x)\text{.}\)

Solution

If we let \(y=f(x)\text{,}\) then \(x^{2} + y^{2} = 1\text{.}\) Then

\begin{equation*} 2x + 2yy' = 0 \end{equation*}

which means that

\begin{equation*} y' = -\frac{x}{y} = -\frac{x}{\sqrt{1-x^{2}}}. \end{equation*}

We must also be aware of when to use appropriate derivative rules when doing implicit differentiation.

Example 2.6.3 Chain and quotient rule

Suppose \(y\) is defined implicitly by \(\tan(x-y) = \frac{y}{1+x^{2}}\text{.}\) Find \(y'\text{.}\)

Solution

We start by taking the derivative with respect to \(x\) of each side of the equation:

\begin{align*} \dv{}{x}\tan(x-y) & = (1-y')\sec^{2}(x-y) \\ \dv{}{x}\frac{y}{1+x^{2}} & = \frac{y'(1+x^{2}) - 2xy}{(1+x^{2})^{2}}. \end{align*}

Therefore

\begin{equation*} (1-y')\sec^{2}(x-y) = \frac{y'(1+x^{2}) - 2xy}{(1+x^{2})^{2}}, \end{equation*}

and we can solve this for \(y'\text{:}\)

\begin{equation*} (1+x^{2})^{2}\sec^{2}(x-y) + 2xy = y'(1+x^{2}) + (1+x^{2})^{2}y'\sec^{2}(x-y) \end{equation*}

or just

\begin{equation*} y' = \frac{(1+x^{2})^{2}\sec^{2}(x-y) + 2xy}{(1+x^{2})+(1+x^{2})^{2}\sec^{2}(x-y)} \end{equation*}
Example 2.6.4 A differential equation

Let \(P(t)\) denote the population of the United States, where \(P\) is in millions and \(t\) is the number of years after 1990. Then the growth of \(P(t)\) can be modeled by the differential equation

\begin{equation*} P' = \frac{1}{23500}P(532-P). \end{equation*}

According to this model, it's not too hard to see that \(P'\) should be positive given the current population of the US, so the model predicts the population to increase. Is this rate of growth increasing or decreasing?

Solution

We need to find \(P''\text{,}\) which is the rate of change of \(P'\text{.}\) This means we need to differentiate both sides of the differential equation \(P' = \frac{1}{23500}P(532-P)\) with respect to \(t\text{:}\)

\begin{equation*} P'' = \frac{1}{23500}P'(532-P) - \frac{1}{23500}P(P'). \end{equation*}

The current population is about 323.1 million, so we can take \(P=323.1\text{,}\) which also gives

\begin{equation*} P' = \frac{1}{23500}(323.1)(532-323.1) = 2.9, \end{equation*}

and so

\begin{equation*} P'' = \frac{1}{23500}(2.9)(532-323.1) - \frac{1}{23500}(323.1)(2.9) < 0. \end{equation*}

Hence it appears that the rate of population increase is itself decreasing, which implies that the population growth of the US is slowing down.
