Section 3.5 The Chain Rule

At this point, we can take derivatives of sums, differences, products and quotients of functions. However, these rules aren't very useful for differentiating functions like \(f(x) = (3x^{2} + \sin x)^{50}\text{.}\) We could technically evaluate \(f'(x)\) using these rules but it would be an awful way to spend your weekend. But if we make the substitution \(u = 3x^{2} + \sin x\text{,}\) then we can rewrite \(f(x)\) as \(f(u) = u^{50}\text{,}\) which is much easier to differentiate: \(\dv{f}{u} = 50u^{49}\text{.}\) At this step we might be tempted to say that \(f'(x) = 50u^{49} = 50(3x^{2} + \sin x)^{49}\text{,}\) but this isn't quite right. To get the actual derivative we need to consider how the new variable \(u\) depends on \(x\) as well. The Chain Rule is what we need.

Theorem 3.5.1. The Chain Rule.

Let \(y = f(u)\) and \(u = g(x)\) be differentiable functions. Then

\begin{equation*} \dv{y}{x} = \dv{y}{u}\dv{u}{x}. \end{equation*}

Equivalently, if we set \(F(x) = f(g(x))\) we have

\begin{equation*} F'(x) = f'(g(x))g'(x). \end{equation*}
Example 3.5.2. Using the chain rule.

Let \(f(x) = (3x^{2} + \sin x)^{50}\text{.}\) Find \(f'(x)\text{.}\)

Solution

We'll try the same trick we used before and we'll set \(u = 3x^{2} + \sin x\text{.}\) Then the chain rule says that

\begin{align*} f'(x) & =\dv{f}{u}\dv{u}{x} \\ & = (50u^{49})(6x+\cos x) \\ & = 50(3x^{2} + \sin x)^{49}(6x + \cos x). \end{align*}

As the last example highlighted, we can use the chain rule in combination with any of the other derivative rules we know if the function we're differentiating is complicated.

Example 3.5.3. Combining rules.

Let \(f(s) = \sqrt{\frac{s^{2} - 1}{s^{2} + 1}}\text{.}\) Find \(f'(s)\text{.}\)

Solution

We'll let \(u = \frac{s^{2}-1}{s^{2}+1}\) stand in for the "inside function." Then we have \(f(u) = \sqrt{u}\) and so

\begin{align*} \dv{f}{s} & = \dv{f}{u}\dv{u}{s} \\ & = \frac{1}{2}u^{-\frac{1}{2}}\frac{2s(s^{2}+1) - 2s(s^{2}-1)}{(s^{2}+1)^{2}} \\ & = \frac{1}{2}\left(\frac{s^{2}-1}{s^{2}+1}\right)^{-\frac{1}{2}}\frac{2s(s^{2}+1) - 2s(s^{2}-1)}{(s^{2}+1)^{2}} \end{align*}
Example 3.5.4. Chain rule within a chain rule.

Find the slope of \(g(x) = \sin^{4}(\cos^{3}x)\) at \(x=\pi\text{.}\)

Solution

We need to compute \(g'(\pi)\text{.}\) First, note that \(g(x) = [\sin(\cos^{3}x)]^{4}\text{,}\) so let \(u = \sin(\cos^{3}x)\text{.}\) Now we could try to use the chain rule right here but this would require finding \(\dv{u}{x}\text{,}\) and \(u\) is itself a complicated function of \(x\text{.}\) So let \(v = [\cos x]^{3}\text{,}\) and finally let \(w = \cos x\text{.}\) Then we can say that

\begin{align*} g'(x) & = \dv{g}{u}\dv{u}{v}\dv{v}{w}\dv{w}{x} \\ & = \left(\dv{}{u}u^{4}\right)\left(\dv{}{v}\sin v\right)\left(\dv{}{w}w^{3}\right)\dv{}{x}\cos x \\ & = (4u^{3})(\cos v)(3w^{2})(-\sin x). \end{align*}

We could plug in what \(u,v,w\) are in terms of \(x\) and then plug in \(x=\pi\text{,}\) but it's easier to just find \(u,v,w\) at \(x=\pi\) and enter these values into the above. At \(x=\pi\) we have \(u = \sin(-1), v = -1\) and \(w = -1\text{,}\) so

\begin{equation*} g'(\pi) = 4(\sin(-1))^{3}\cos(-1)(3)\cdot0 = 0. \end{equation*}
