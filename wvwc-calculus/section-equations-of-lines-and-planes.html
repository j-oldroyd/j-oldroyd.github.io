Section 10.5 Equations of Lines and Planes

¶

Recall that the equation of a line in \(\RR^{2}\) has the general form \(ax+by + c = 0\text{.}\) However, this equation does not give a line in \(\RR^{3}\text{.}\) In fact, it actually gives a plane, as we'll see! What we want to do now is find exactly how to describe a given line or plane.

Subsection 10.5.1 Equations of Lines

¶

First, suppose we have a line \(L\) in \(\RR^{3}\) as follows:

Our goal is to find an equation that describes this line. Imagine that we have a vector \(\vv{r}_{0}\) pointing from the origin to the point \(A\text{,}\) like so:

Then we can get to every other point on the line by starting from \(\vv{r}_{0}\text{.}\) In particular, if \(\vv{u}\) is any vector parallel to \(\vec{AB}\text{,}\) then \(\vv{r} = \vv{r}_{0}+\vv{u}\) must also be on the line \(L\text{:}\)

So to get all possible points on \(L\text{,}\) we just need to look at all possible vectors \(\vv{r} = \vv{r}_{0}+\vv{u}\text{,}\) where \(\vv{u}\) is any vector parallel to \(\vec{AB}\text{.}\) If we set \(\vv{v} = \vec{AB}\text{,}\) 1 We don't actually have to pick \(\vv{v} = \vec{AB}\text{,}\) we just need \(\vv{v}\) to be parallel to \(\vec{AB}\text{.}\) then

\begin{equation*} \vv{r} = \vv{r}_{0}+t\vv{v} \end{equation*}

will trace out all points on \(L\) as \(t\) varies from \(-\infty\) to \(\infty\text{.}\)

Theorem 10.5.4 Parametric Equations of a Line

If \(P_{0}(x_{0},y_{0},z_{0})\) is a point on a line \(L\text{,}\) and if \(\vv{v} = \dotprod{a,b,c}\) is parallel to \(L\text{,}\) then

\begin{equation*} \vv{r} = \vv{r}_{0}+t\vv{v} \end{equation*}

traces out \(L\text{,}\) where

\begin{equation*} \vv{r} = \dotprod{x,y,z},\quad\vv{r}_{0} = \dotprod{x_{0},y_{0},z_{0}}. \end{equation*}

We can also write this as the system of equations

\begin{equation*} x = x_{0}+ta,\quad y=y_{0}+tb\quad\text{and}\quad z = z_{0}+tc. \end{equation*}
Example 10.5.5 Equation of a line with a given direction and point

Suppose we want to find the equation of a line \(L\) that passes through \((2,5,-3)\) and is perpendicular to \(\vv{u} = \dotprod{2,1,2}\text{.}\) Theorem 10.5.4 says that we only need to find the right choices for \(\vv{r}_{0}\) and \(\vv{v}\) in order to get the equation of such a line, and we can write down one of these right away:

\begin{equation*} \vv{r}_{0} = \dotprod{2,5,-3}. \end{equation*}

So we just need to find \(\vv{v}\text{.}\) Since \(L\) is orthogonal to \(\vv{u} = \dotprod{2,1,2}\text{,}\) this means we want \(\vv{v}\) to be orthogonal to \(\vv{u}\) as well. A great way to find such a \(\vv{v}\) is to use the cross product. If \(\vv{w}\) is any nonzero vector, then \(\vv{u}\times\vv{w}\) must be orthogonal to both \(\vv{u}\) and \(\vv{w}\text{,}\) and in particular it must be orthogonal to \(\vv{u}\text{.}\) So we can take

\begin{equation*} \vv{v} = \dotprod{2,1,2}\times\vv{i} = \dotprod{0,2,-1}. \end{equation*}

So the equation of this line is

\begin{equation*} \vv{r} = \dotprod{2,1+2t,2-t} \end{equation*}

As \(t\) varies, this will trace out our line.

When we begin working with line integrals, we'll need to know how to parameterize a line segment, so we'll discuss that now. Suppose we have a line segment between two points \(P_{0}\) and \(P_{1}\text{.}\) Let \(\vv{r}_{0}\) and \(\vv{r}_{1}\) denote the corresponding position vectors of these points (i.e. the vectors whose components are given by the coordinates of these points). Then the equation of the line between \(P_{0}\) and \(P_{1}\) is given by

\begin{equation*} \vv{r}] = \vv{r}_{0}+t(\vv{r}_{1}-\vv{r}_{0}) = (1-t)\vv{r}_{0}+t\vv{r}_{1}. \end{equation*}

To get the line segment we're actually looking for, we just need to restrict \(t\) to the interval \(0\leq t\leq 1\text{.}\) So we can say that

\begin{equation*} \vv{r} = (1-t)\vv{r}_{0}+t\vv{r}_{1} \quad\text{with}\quad0\leq t\leq 1 \end{equation*}

traces out the line segment between \(P_{0}\) and \(P_{1}\text{.}\)

Subsection 10.5.2 Equations of Planes

¶

Now we move on to finding the equation of a plane. We'll try a vector approach to this like we did for lines, and this won't be too bad to do. We just need to keep the following in mind:

Observation 10.5.6

A plane in \(\RR^{3}\) is uniquely determined by specifying a single point for it to pass through and a direction for it to face.

So in particular, vectors will provide a nice way to specify the direction of a plane, and any vector that does so must be orthogonal to the plane. If a (nonzero) vector \(\vv{n}\) is orthogonal to a given plane, then we call \(\vv{n}\) a normal vector to the plane. Now we can build up an equation of a given plane.

Let \(P_{0}(x_{0},y_{0},z_{0})\) denote a point on the plane, and let \(\vv{n} = \dotprod{a,b,c}\) be a normal vector to the plane. If \(\vv{r}=\dotprod{x,y,z}\) is the position vector of any other point on the plane, then it must be true that \(\vv{n}\cdot(\vv{r}-\vv{r}_{0}) = 0\text{,}\) where \(\vv{r}_{0}\) is the position vector of \(P_{0}\text{.}\) We can summarize this in the following theorem.

Theorem 10.5.7 Vector and Scalar Equations of a Plane

If \(\vv{r}_{0} = \dotprod{x_{0},y_{0},z_{0}}\) is the position vector for some point on a plane with normal vector \(\vv{n} = \dotprod{a,b,c}\text{,}\) then the position vector \(\vv{r} = \dotprod{x,y,z}\) of any other point on the plane must satisfy the vector equation of the plane:

\begin{equation*} \vv{n}\cdot(\vv{r}-\vv{r}_{0}) = 0. \end{equation*}

Equivalently, any point \((x,y,z)\) must also satisfy the equivalent scalar equation of the plane:

\begin{equation*} a(x-x_{0})+b(y-y_{0})+c(z-z_{0}) = 0. \end{equation*}
Example 10.5.8 Equation of a plane orthogonal to a line

Consider the two lines in \(\RR^{3}\) determined by the equations

\begin{align*} \vv{r}_{1} & = \dotprod{1,1,0} + t\dotprod{1,-1,2} \\ \vv{r}_{2} & = \dotprod{2,0,2} + s\dotprod{-1,1,0} \end{align*}

We want to find the equation of the plane that contains the intersection of these two lines and is orthogonal to the first line. By Theorem 10.5.7, we need a point on the plane and a normal vector. Since the plane must contain the intersection of these two lines, that's our point. To find it, we'll set \(\vv{r}_{1} = \vv{r}_{2}\) and go from there:

\begin{align*} \vv{r}_{1}=\vv{r}_{2} & \Rightarrow \dotprod{-1,1,-2} + \dotprod{t+s,s-t,2t} = \vv{0} \\ & \Rightarrow t+s-1 = s-t+1 = 2t-2 = 0 \\ & \Rightarrow t = 1\quad\text{and}\quad s=0 \end{align*}

So the position vector \(\vv{r}_{0}\) of the point where these lines intersect is given by \(\vv{r}_{0} = \dotprod{2,0,2}\text{.}\) We also need a normal vector to the plane, but this is easy to find: since the plane needs to be orthogonal to the first line, we can just take \(\vv{n} = \dotprod{1,-1,2}\text{.}\) So the equation of our plane is

\begin{equation*} \dotprod{1,-1,2}\cdot\dotprod{x-2,y,z-2} = 0 \end{equation*}

or equivalently

\begin{equation*} x-y+2z -6 = 0. \end{equation*}

We can rewrite the equation of a plane in the form

\begin{equation*} ax+by+cz+d = 0. \end{equation*}

A normal vector to this plane is given by \(\vv{n} = \dotprod{a,b,c}\text{.}\)

Example 10.5.9 Equation of a line orthogonal to a plane

Suppose we're given the plane \(3x-5z+2 = 0\text{,}\) and we want to find the equation of the line orthogonal to this plane and passing through the point \((3,-1,2)\text{.}\) Then the equation of this line is

\begin{equation*} \vv{r} = \vv{r}_{0}+t\vv{v} \end{equation*}

where \(\vv{r}_{0} = \dotprod{3,-1,2}\) and \(\vv{v}\) is a vector parallel to the line. Since the line must be orthogonal to the plane, we can take \(\vv{v} = \dotprod{3,0,-5}\text{.}\) So the equation of this line is

\begin{equation*} \vv{r} = \dotprod{3,-1,2}+\dotprod{3t,0,-5t}. \end{equation*}
Example 10.5.10 Equation of a plane containing a line and a point

Suppose we're trying to find a plane containing the point \((1,2,0)\) and containing the line \(\vv{r} = \dotprod{3-t,2,5+2t}\text{.}\) To get the equation of this plane, we also need a normal vector to the plane. We can find the normal vector by computing the cross product of two vectors parallel to the plane.

First, since the plane contains the line \(\dotprod{3-t,2,5+2t},\) the direction of this line must be parallel to the plane. So we know that \(\vv{v} = \dotprod{-1,0,2}\) is parallel to the plane. To get another vector parallel to this plane, we can use the given point and a second point in the plane, say \((3,2,5)\text{,}\) which comes from the line (just set \(t=0\)). So the vector \(\vv{u} = \dotprod{3-1,2-2,5-0} = \dotprod{2,0,5}\) must also be parallel to the plane. Hence a normal vector to this plane is

\begin{equation*} \vv{n} = \vv{u}\times\vv{v} = \dotprod{0,9,0}. \end{equation*}

So an equation of this plane is given by

\begin{equation*} 9(y-2) = 0. \end{equation*}
