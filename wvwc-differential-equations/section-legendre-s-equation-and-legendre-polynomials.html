Section 5.2 Legendre's Equation and Legendre Polynomials

An important differential equation in applications is the Legendre equation given by

\begin{equation} (1 - x^{2})y'' - 2xy' + k(k + 1)y = 0\text{.}\label{equation-legendre-ode}\tag{5.2.1} \end{equation}

Our first example of this equation (with \(n = 1\)) was examined in Example 5.1.9. By this example, we see that (5.2.1) has a series solution centered at \(x = 0\) with radius of convergence at least \(1\text{.}\) Therefore the power series method is appropriate.

Subsection 5.2.1 Solving the Legendre Equation

We'll proceed as we did in Example 5.1.9, altering the last sum as necessary to get

\begin{equation*} \sum_{n\geq0}(n+2)(n+1)c_{n+2}x^{n} - \sum_{n\geq0}n(n-1)c_{n}x^{n} - \sum_{n\geq0}2nc_{n}x^{n} + \sum_{n\geq0}k(k+1)c_{n}x^{n} = 0 \end{equation*}

which gives (after a bit of algebra, once again)

\begin{equation*} c_{n+2} = -\frac{(k - n)(k + n + 1)}{(n + 2)(n + 1)}c_{n}\text{.} \end{equation*}

This recurrence is valid for \(n \geq 0\text{,}\) and allows us to write out the solution \(y\) in terms of the parameter \(k\) and the arbitrary constants \(c_{0}\) and \(c_{1}\text{:}\)

\begin{equation*} y = c_{0}y_{1}(x) + c_{1}y_{2}(x) \end{equation*}

where

\begin{align} y_{1}(x) \amp = 1 - \frac{k(k+1)}{2!}x^{2} + \frac{(k - 2)k(k+1)(k + 3)}{4!}x^{4} - \cdots\label{equation-legendre-soln-y1}\tag{5.2.2}\\ y_{2}(x) \amp = x - \frac{(k-1)(k+2)}{3!}x^{3} + \frac{(k-3)(k-1)(k+2)(k+4)}{5!}x^{5} - \cdots \text{.}\label{equation-legendre-soln-y2}\tag{5.2.3} \end{align}

Note that \(y_{1}\) and \(y_{2}\) form a basis of solutions (Definition 2.1.13) of the Legendre equation, which means that \(y = c_{0}y_{1} + c_{1}y_{2}\) must also be the general solution.

Subsection 5.2.2 Legendre Polynomials

Our solution of (5.2.1) simplifies greatly if \(k\) happens to be an integer. In particular, if \(k\) is a nonnegative integer then

\begin{equation*} a_{k+2} = a_{k+4} = \cdots = 0\text{.} \end{equation*}

If \(k\) is even then the solution \(y_{1}\) given in (5.2.2) becomes a polynomial:

\begin{equation*} y_{1} = 1 - \frac{k(k+1)}{2!}x^{2} + \cdots + (-1)^{k/2}\frac{[k - (k-2)][k - (k-4)]\cdots[k + (k-3)][k + (k-1)]}{k!}x^{k}\text{.} \end{equation*}

Likewise, if \(k\) is odd then \(y_{2}\) given in (5.2.3) becomes a polynomial instead:

\begin{equation*} y_{2} = x - \frac{(k-1)(k+2)}{3!}x^{3} + \cdots + (-1)^{\frac{k-1}{2}}\frac{[k - (k - 2)][k - (k - 4)]\cdots[k + (k - 3)][k + (k - 1)]}{k!}x^{k}\text{.} \end{equation*}

By choosing \(c_{0}\) and \(c_{1}\) judiciously, we can guarantee that the polynomials \(c_{0}y_{1}\) (if \(k\) is even) or \(c_{1}y_{2}\) (if \(k\) is odd) are precisely equal to \(1\) at \(x = 1\text{.}\) Doing so gives us the Legendre polynomials \(P_{n}(x)\text{,}\) defined more precisely in (5.2.4):

\begin{equation} P_{n}(x) = \begin{cases} \sum_{j = 0}^{n/2}(-1)^{j}\frac{(2n - 2j)!}{2^{n}j!(n - j)!(n - 2j)!}x^{n - 2j} \amp \text{ if }n\text{ is even} \\ \sum_{j = 0}^{(n-1)/2}(-1)^{j}\frac{(2n - 2j)!}{2^{n}j!(n - j)!(n - 2j)!}x^{n - 2j} \amp \text{ if }n\text{ is odd} \\ \end{cases}\text{.}\label{equation-legendre-poly}\tag{5.2.4} \end{equation}

These polynomials satisfy several nice properties, but one of the most important characteristics they have is that \(\{P_{n}(x)\}\) forms an orthogonal set of polynomials on the interval \([-1,1]\text{.}\) This means that

\begin{equation*} \int_{-1}^{1}P_{m}(x)P_{n}(x)\,dx = 0 \end{equation*}

if \(m\neq n\text{.}\) It can also be shown that

\begin{equation*} \int_{-1}^{1}P_{m}(x)P_{n}(x)\,dx = \frac{2}{2n+1} \end{equation*}

if \(m = n\text{.}\) This property allows us to express any polynomial as a finite sum of Legendre polynomials in a computationally efficient manner. Furthermore, if we allow infinite series then we can use Legendre polynomials to express and continuous function on \([-1,1]\text{.}\)

For actually computing Legendre polynomials, instead of using (5.2.4) we often use Rodrigues' formula

\begin{equation} P_{n}(x) = \frac{1}{2^{n}n!}\dv[n]{}{x}[(x^{2} - 1)^{n}]\label{equation-rodrigues}\tag{5.2.5} \end{equation}

or Bonnet's recurrence

\begin{equation} (n + 1)P_{n + 1}(x) = (2n + 1)xP_{n}(x) - n P_{n-1}(x)\text{.}\label{equation-bonnet-recurrence}\tag{5.2.6} \end{equation}
