Section1.2Even and odd extensions; functions of arbitrary period¶ permalink

If you look at Example 1.1.5, then you'll notice that a lot of the Fourier coefficients were \(0\). In particular, \(a_{k} = 0\) for \(k\geq0\). The reason for this was because \(f(x) = x^{3}\) was an odd function, whereas each \(a_{k}\) was the coefficient of the even function \(\cos kx\). We can extend this observation into a theorem, but first it might be helpful to recall the definition of even/odd functions.

Definition1.2.1Even and Odd Functions
Let \(f(x)\) be a function. We say that \(f(x)\) is\begin{align*} \text{even if } & f(-x) = f(x)\\ \text{odd if } & f(-x) = -f(x). \end{align*}
Theorem1.2.2Fourier Series of Even and Odd Functions
Let \(f(x)\) be a periodic function with period \(2\pi\), with corresponding Fourier series \begin{equation*}a_{0} + \sum_{k=1}^{\infty}(a_{k}\cos kx+b_{k}\sin kx).\end{equation*} If \(f(x)\) is an odd function, then \(a_{k} = 0\) for \(k\geq0\). If \(f(x)\) is an even function, then \(b_{k}=0\) for \(k\geq1\).
