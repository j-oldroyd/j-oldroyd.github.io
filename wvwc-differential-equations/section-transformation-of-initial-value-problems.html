Section 6.2 Transformation of Initial Value Problems

Theorem 6.2.1. Laplace Transforms of Derivatives.

Suppose that the function \(f(t)\) is piecewise smooth and that \(F(s) = \Laplace{f(t)}\) exists. Then \(\Laplace{f'(t)}\) exists and

\begin{equation*} \Laplace{f'(t)} = s\Laplace{f(t)}-f(0) = sF(s)-f(0). \end{equation*}

The above theorem says that differentiation in the ``time domain'' corresponds to multiplication by \(s\) in the ``frequency domain.'' This will be the most useful property of the Laplace transform for us: it will turn differential equations in the time domain into algebraic equations in the frequency domain.

Example 6.2.2. Laplace Transform of a Second Derivative.

Let \(f(t)\) be piecewise smooth with Laplace transform \(F(s)\text{.}\) Compute \(\Laplace{f''(t)}\) using Theorem 6.2.1.

Solution.

By Theorem 6.2.1 we know that

\begin{equation*} \Laplace{f'(t)} = sF(s)-f(0). \end{equation*}

We can apply Theorem 6.2.1 to \(f'(t)\) as well, since it must also be piecewise smooth:

\begin{align*} \Laplace{f''(t)} \amp= s\Laplace{f'(t)} - f'(0)\\ \amp= s(sF(s)-f(0)) - f'(0)\\ \amp= s^{2}F(s) - sf(0) - f'(0)\text{.} \end{align*}

This process can be continued indefinitely. In general,

\begin{equation*} \Laplace{f^{(n)}(t)} = s^{n}F(s) - s^{n-1}f(0) - s^{n-2}f'(0) - \cdots -sf^{(n-2)}(0) - f^{(n-1)}(0). \end{equation*}

This gives us everything we need to start solving ODEs using Laplace transforms.

Example 6.2.3. Solving IVPs with Laplace Transforms.

Solve the IVP

\begin{equation*} x''+8x'+15x = 0 \qq{} x(0) = 2, x'(0) = -3 \end{equation*}

where \(x\) is a function of \(t\text{.}\)

Solution.

We will solve this by using Laplace transforms. If we set \(X(s) = \Laplace{x(t)}\text{,}\) then

\begin{align*} \Laplace{x'(t)} \amp= sX(s) - 2\\ \Laplace{x''(t)} \amp= s^{2}X(s) - 2s +3\text{.} \end{align*}

So if we take the Laplace transform of the entire ODE, we get

\begin{equation*} \underbrace{s^{2}X(s) - 2s + 3}_{\Laplace{x''}} + \underbrace{8sX(s) - 16}_{\Laplace{8x'}} + 15X(s) = 0 \end{equation*}

which simplifies to

\begin{equation*} s^{2}X(s)+8sX(s)+15X(s) = 2s+13. \end{equation*}

To solve the ODE we need to find \(x(t) = \mathcal{L}^{-1}\{X(s)\}\text{,}\) so we now solve the above for \(X(s)\) in order to find the inverse Laplace transform. If we do so, we get

\begin{equation*} X(s) = \frac{2s+13}{s^{2}+8s+15} = \frac{2s+13}{(s+3)(s+5)}. \end{equation*}

This doesn't look like the Laplace transform of anything we know at the moment, but we can simplify the right hand side using partial fractions. If we do so, we get

\begin{equation*} X(s) = \frac{\frac{7}{2}}{s+3} - \frac{\frac{3}{2}}{s+5}. \end{equation*}

So

\begin{align*} x(t) \amp= \frac{7}{2}\mathcal{L}^{-1}\left\{\frac{1}{s+3}\right\} - \frac{3}{2}\mathcal{L}^{-1}\left\{\frac{1}{s+5}\right\}\\ \amp= \frac{7}{2}e^{-3t} - \frac{3}{2}e^{-5t}\text{.} \end{align*}

Example 6.2.4. Nonhomogeneous IVPs with Laplace Transforms.

Solve the IVP given by

\begin{equation*} y''+9y = 10e^{-t},\quad y(0) = y'(0) = 0. \end{equation*}
Solution.

We begin by taking the Laplace transform of both sides to get

\begin{equation*} s^{2}Y(s) - sy(0) - y'(0) + 9Y(s) = \frac{10}{s+1} \end{equation*}

or just

\begin{equation*} s^{2}Y(s) + 9Y(s) = \frac{10}{s+1}. \end{equation*}

So

\begin{equation*} Y(s) = \frac{10}{(s+1)(s^{2}+9)}. \end{equation*}

By partial fractions, this becomes

\begin{equation*} Y(s) = \frac{1-s}{s^{2}+9} + \frac{1}{s+1} = \frac{1}{s^{2}+9} - \frac{s}{s^{2}+9} + \frac{1}{s+1}, \end{equation*}

so the solution of the ODE is

\begin{align*} y(t) \amp = \iLaplace{Y(s)}\\ \amp= \frac{1}{3}\sin3t -\cos3t + e^{-t}\text{.} \end{align*}
