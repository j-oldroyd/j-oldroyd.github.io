Note that we are computing a translation of the usual unit step function. We still compute \(\Laplace{u(t-3)}\) using the definition of the Laplace transform:

\begin{align*} \Laplace{u(t)} \amp= \Int{0}{\infty}u(t)e^{-st}\,dt\\ \amp= \Int{3}{\infty}e^{-st}\,dt \amp\text{since is for values of }\\ \amp= -\frac{1}{s}\left[e^{-st}\right]_{3}^{\infty}\\ \amp= \frac{e^{-3s}}{s}\text{.} \end{align*}
