By Hooke's Law and Newton's Second Law, we have

\begin{equation*} mx''+kx = F_{E}(t) \end{equation*}

where \(F_{E}\) is the external force at time \(t\text{.}\) Since \(m = 1\text{,}\) \(k = 4\) and

\begin{equation*} F_{E}(t) = \begin{cases} \cos3t \amp\text{if }0\leq t\lt 2\pi\\ 0 \amp\text{if }t\geq2\pi \end{cases} \end{equation*}

the motion of the mass satisfies the IVP

\begin{equation*} x''+4x = F_{E}(t), x(0) = 1, x'(0) = 0. \end{equation*}

We can rewrite \(F_{E}(t)\) as follows:

\begin{equation*} F_{E}(t) = [u(t-0)-u(t-2\pi)]\cos3t = u(t)\cos3t - u(t-2\pi)\cos3t. \end{equation*}

So the IVP we need to solve is

\begin{equation*} x''+4x = u(t)\cos3t - u(t-2\pi)\cos3t, x(0) = 1, x'(0) = 0. \end{equation*}

If we take Laplace transforms, this becomes

\begin{equation*} s^{2}X(s) - s + 4X(s) = \Laplace{F_{E}(t)}, \end{equation*}

where

\begin{align*} \Laplace{F_{E}(t)} \amp= \Laplace{u(t)\cos3t} - \Laplace{u(t-2\pi)\cos3t}\\ \amp= \frac{s}{s^{2}+9} - \Laplace{u(t-2\pi)\cos(3(t-2\pi)+6\pi)}\\ \amp= \frac{s}{s^{2}+9} - \Laplace{u(t-2\pi)\cos(3(t-2\pi))}\\ \amp= \frac{s}{s^{2}+9} - e^{-2\pi s}\frac{s}{s^{2}+9}\text{.} \end{align*}

So

\begin{equation*} s^{2}X(s) - s + 4X(s) = \frac{s}{s^{2}+9} - e^{-2\pi s}\frac{s}{s^{2}+9}. \end{equation*}

If we solve this for \(X(s)\text{,}\) we get

\begin{equation*} X(s) = \frac{s}{s^{2}+4} + \frac{s}{(s^{2}+9)(s^{2}+4)} - e^{-2\pi s}\frac{s}{(s^{2}+9)(s^{2}+4)}, \end{equation*}

and if we simplify this using partial fractions this becomes

\begin{equation*} X(s) = \frac{s}{s^{2}+4} + \frac{1}{5}\brackets{\frac{s}{s^{2}+4} - \frac{s}{s^{2}+9}}\parens{1-e^{-2\pi s}}. \end{equation*}

So the solution of the IVP is

\begin{align*} x(t) \amp= \iLaplace{\frac{s}{s^{2}+4} + \frac{1}{5}\brackets{\frac{s}{s^{2}+4} - \frac{s}{s^{2}+9}}\parens{1-e^{-2\pi s}}}\\ \amp= \cos2t + \frac{1}{5}\parens{\cos2t-\cos3t} - \frac{1}{5}\parens{u(t-2\pi)\cos[2(t-2\pi)] - u(t-2\pi)\cos[3(t-2\pi)]}\text{.} \end{align*}
