If we assume that \(X(x),T(t)\) are both nonzero, then we can rewrite this to get

\begin{equation*} \frac{T''(t)}{c^{2}T(t)} = \frac{X''(x)}{X(x)}. \end{equation*}

This may not look that helpful, but it actually places some serious restrictions on \(X\) and \(T\text{.}\) The left hand side of this equation only depends on \(t\) whereas the right hand side depends only on \(x\text{.}\) So the only way for this equation to be true for all \(x,t\) is if both sides are constant:

\begin{equation*} \frac{T''(t)}{c^{2}T(t)} = \frac{X''(x)}{X(x)} = k \end{equation*}

for some \(k\text{.}\) This now gives us two separate ordinary differential equations for \(T(t)\) and \(X(x)\text{:}\)
