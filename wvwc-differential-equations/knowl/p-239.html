We can start this example the same way we've done the previous ones. First, we find \(y_{c}\) by solving \(y''+9y=0\text{.}\) So

\begin{equation*} y_{c} = c_{1}\cos(3x)+c_{2}\sin(3x). \end{equation*}

Now we find \(y_{p}\text{.}\) Since the RHS of the ODE is \(3\cos(3x)\text{,}\) we'll guess that \(y_{p} = A\cos(3x)+B\sin(3x)\text{.}\) However, this will cause us problems! Since \(\cos(3x),\sin(3x)\) are both solutions of the corresponding homogeneous ODE, then plugging \(y_{p}\) into \(y''+9y\) will just give \(0\) again, instead of \(3\cos(3x)\text{.}\)
