Let \(f(x)\) be some piecewise-differentiable function, not necessarily periodic. Then we can't find it's Fourier series. However, we can truncate the graph of \(f(x)\text{,}\) and replace it with a periodic function that is equal to \(f(x)\) on some interval \((-L,L)\text{.}\) Then we can find the Fourier series of this function, which by Theorem TheoremÂ 5.3.1 is given by \(\sum_{k=-\infty}^{\infty}c_{k}e^{i\frac{k\pi}{L}x}\) where

\begin{equation*} c_{k} = \frac{1}{2L}\int_{-L}^{L}f(x)e^{-i\frac{k\pi}{L}x}\,dx. \end{equation*}

So we can write

\begin{equation*} f(x) = \sum_{k=-\infty}^{\infty}\left(\frac{1}{2L}\int_{-L}^{L}f(x)e^{-i\frac{k\pi}{L}x}\,dx\right)e^{i\frac{k\pi}{L}x} \end{equation*}

wherever \(f(x)\) is continuous on \(-L<x<x\text{.}\) The idea now is that the larger that \(T\) gets, this expression can be used to represent \(f(x)\) for more and more values of \(x\text{.}\) So we want to see what happens to this as \(T\to\infty\text{.}\) First, we'll clean this up a little bit by writing \(w_{k} = \frac{k\pi}{L}\) and \(\Delta w = w_{k+1} - w_{k}\text{,}\) so that \(\frac{1}{2L} = \frac{\Delta w}{2\pi}\text{.}\) Then if \(x\) is in \((-L,L)\) and \(f\) is continuous at \(x\text{,}\) then we can say

\begin{align*} f(x) & = \sum_{k=-\infty}^{\infty}\left(\frac{1}{2L}\int_{-L}^{L}f(x)e^{-i\frac{k\pi}{L}x}\,dx\right)e^{i\frac{k\pi}{L}x} \\ & = \sum_{k=-\infty}^{\infty}\left(\frac{1}{2\pi}\int_{-L}^{L}f(x)e^{-iw_{k}x}\,dx\right)e^{iw_{k}x}\Delta w. \end{align*}

As awful as this looks, we can relate this to a Riemann sum! As \(L\to\infty,\) \(\Delta w\to0\text{,}\) we can replace \(w_{k}\) with the new variable \(\omega\) and this expression becomes

\begin{equation*} f(x) = \int_{-\infty}^{\infty}\left(\frac{1}{2\pi}\int_{-\infty}^{\infty}f(x)e^{-i\omega x}\,dx\right)e^{i\omega x}\,d\omega. \end{equation*}

This leads to the definition of the Fourier transform. But first we need another definition.
