We know that the position will look like \(x = x_{c} + x_{p}\) where \(x_{c}\to0\) as \(t\to\infty\text{,}\) but we can't find the exact form of the transient solution without knowing the initial conditions. So we'll assume that we can estimate the position of the mass using the steady-state solution \(x_{p}\text{.}\) Since

\begin{equation*} \omega = \omega_{0} = \sqrt{3}, \quad F_{0} = 5\quad\text{and}\quad c = 2, \end{equation*}

we get

\begin{equation*} x_{p} = \frac{5}{6}\sqrt{3}\sin(\sqrt{3}t). \end{equation*}

So after 40 s more seconds the mass should be around \(x_{p}(47)\text{,}\) or about -0.388 m.
