If you go back to the first plot, you may notice from the code used to generate the function \(f(x)\) that \(f(x)\) is just the sum of two basic trigonometric functions:

\begin{equation*} f(x) = \cos5x-5\sin3x. \end{equation*}

In other words, the (finite) sum of functions of the form \(\sin nx,\cos mx\) where \(n,m\) are integers is also periodic, and also has period \(2\pi\text{.}\) One of the greatest accomplishments in mathematics was the realization that many other periodic functions can be written in this way if we allow infinite sums of this form, which we call trigonometric series.
