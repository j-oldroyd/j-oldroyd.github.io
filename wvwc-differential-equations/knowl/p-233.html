Note that in Example 2.4.2, we didn't need any initial conditions to find \(y_{p}\text{.}\) This means that if a non-homogeneous ODE like the one in Example 2.4.2 represents some physical system, then the initial configuration of that system has no effect on \(y_{p}\). We will see soon that particular solutions correspond to external forces on a system, like gravity, whereas complementary solutions correspond to internal forces in a system, such as the spring force.
