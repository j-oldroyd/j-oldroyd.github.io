Solution 7.4.4.1.

If we find the Fourier coefficients of \(f\text{,}\) we get

\begin{equation*} b_{k} = \frac{2 - (-1)^{k}}{k\pi}\text{.} \end{equation*}

By Bessel's inequality, we know that

\begin{equation*} \sum_{k=1}^{N}\left(\frac{2 - (-1)^{k}}{k\pi}\right) \leq 2 \end{equation*}

for any \(N\text{.}\) As \(N\to\infty\text{,}\) Parseval's gives the identity

\begin{equation*} 2\pi^{2} = \frac{3^{2}}{1^{2}} + \frac{1^{2}}{2^{2}} + \frac{3^{2}}{3^{2}} + \frac{1^{2}}{4^{2}} + \cdots\text{.} \end{equation*}
