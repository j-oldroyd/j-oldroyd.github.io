Consider the PDE given by

\begin{equation*} u_{t}+uu_{x} = 0. \end{equation*}

This PDE fails to be linear because the second term involves multiplying \(u\) with its derivative \(u_{x}\text{.}\) However, it's not too hard to check that \(u(x,t) = \frac{x}{t+1}\) is a solution of the PDE, since if we plug this function into the PDE we get

\begin{align*} u_{t} + uu_{x} \amp = -\frac{x}{(t+1)^{2}} + \frac{x}{t+1}\frac{1}{t+1} \\ \amp = 0 \text{.} \end{align*}

However, the closely related function \(v(x,t) = 3u(x,t) = \frac{3x}{t+1}\) is not a solution of the same PDE, since

\begin{equation*} v_{t}+vv_{x} = \frac{6x}{(t+1)^{2}}\neq0 \end{equation*}

So the superposition principle does not hold for this PDE.
