We can add a few more restrictions to these ODEs to help us solve them. Note that the boundary conditions (8.2.2) force either \(X(0) = X(L) = 0\) or \(T(t) = 0\) for all \(t\text{,}\) which leads to \(u(x,t) = 0\text{.}\) So to avoid this trivial solution, we'll set \(X(0) = X(L) = 0\text{.}\)
