Section 5.4 Bessel's Equation

¶

As with Legendre's Equation (5.2.1), another important differential equation in applications is Bessel's equation:

\begin{equation} x^{2}y'' + xy' + (x^{2} - \nu^{2})y = 0\text{,}\label{equation-bessel}\tag{5.4.1} \end{equation}

where \(\nu \geq 0\text{.}\) By Theorem 5.3.6, this equation has a series solution at \(x = 0\) of the form

\begin{equation*} y = x^{r}\sum_{k\geq0}c_{k}x^{k} \end{equation*}

where \(r\) is a solution of the indicial equation

\begin{equation*} r(r - 1) + r - \nu^{2} = 0\text{,} \end{equation*}

or just

\begin{equation*} r = \pm\nu\text{.} \end{equation*}

In particular, there we're guaranteed a series solution by setting \(r = \nu\text{,}\) since this is the larger root. Note that Example 5.3.7 is actually a Bessel equation with parameter \(\nu = \frac{1}{2}\text{.}\)

Let

\begin{equation*} y = x^{\nu}\sum_{k\geq0}c_{k}x^{k}\text{.} \end{equation*}

Then we can plug this into (5.4.1) to obtain

\begin{equation} \sum_{k\geq0}(k+\nu)(k+\nu-1)c_{k}x^{k + \nu} + \sum_{k\geq0}(k+\nu)c_{k}x^{k + \nu} + \sum_{k\geq2}c_{k-2}x^{k + \nu} - \sum_{k\geq0}\nu^{2}c_{k}x^{k + \nu} = 0\text{,}\label{equation-bessel-plugged-in}\tag{5.4.2} \end{equation}

which gives

\begin{equation*} c_{k} = -\frac{c_{k-2}}{k(k+2\nu)}\qq{for}k\geq2\text{.} \end{equation*}

Since this only gives us data about \(c_{k}, k\geq 2\text{,}\) we should go back to (5.4.2) to see if we can say anything about \(c_{0}\) or \(c_{1}\text{.}\) In fact, we get

\begin{equation*} (2\nu + 1)c_{1} = 0 \implies c_{1} = 0\text{.} \end{equation*}

Hence our series solution only contains even-indexed coefficients. Rewriting the recurrence to reflect this, we get

\begin{equation} c_{2k} = -\frac{c_{2k - 2}}{2^{2}k(k + \nu)} = \frac{(-1)^{k}c_{0}}{2^{2m}k!(\nu + 1)(\nu + 2)\cdots(\nu + k)}\qq{for} k\geq2.\label{equation-bessel-recurrence}\tag{5.4.3} \end{equation}

Subsection 5.4.1 Bessel Functions for Integer \(\nu\)

¶

Now we consider what happens to solutions given by (5.4.3) if \(\nu = n\) is a nonnegative integer. To simplify matters (somewhat...), we add the restriction that \(c_{0} = \frac{1}{2^{n}n!}\text{.}\) This allows us to write (5.4.3) more simply as

\begin{equation} c_{2k} = \frac{(-1)^{k}}{2^{2k + n}(n + k)!}.\label{equation-bessel-function-nu-integer}\tag{5.4.4} \end{equation}

The resulting series

\begin{equation} J_{n}(x) = x^{n}\sum_{k\geq0}\frac{(-1)^{k}}{2^{2k + n}k!(n + k)!}x^{2k}\label{equation-bessel-function-first-kind}\tag{5.4.5} \end{equation}

is known as the Bessel function of the first kind of order \(n\text{.}\)

Example 5.4.1. Finding \(J_{0}\) and \(J_{1}\).

Find the zeroth order and first order Bessel functions of the first kind.

Solution

Using (5.4.5), we get

\begin{align*} J_{0}(x) \amp = \sum_{k\geq0}\frac{(-1)^{k}}{2^{2k}(k!)^{2}}x^{2k} \amp= 1 - \frac{1}{4}x^{2} + \frac{1}{16(4)}x^{4} - \cdots\\ J_{1}(x) \amp = x\sum_{k\geq0}\frac{(-1)^{k}}{2^{2k + 1}k!(1 + k)!}x^{2k} \amp= \frac{1}{2}x - \frac{1}{8(2)}x^{3} + \cdots\text{.} \end{align*}

Subsection 5.4.2 Bessel Functions of the First Kind for Nonnegative Order

¶

Now we try to find a formula for \(J_{\nu}(x)\) assuming \(\nu\geq0\text{.}\) To do so, we need to make sense of expressions like \((\nu + k)!\text{.}\) Thankfully, we can do so using the Gamma function.

Definition 5.4.3. Gamma Function.

The Gamma function is the function \(\Gamma(x)\) given by

\begin{equation*} \Gamma(x) = \int_{0}^{\infty}t^{x-1}e^{-t}\,dt\text{.} \end{equation*}

An important property of the Gamma function is the following:

\begin{equation*} \Gamma(x + 1) = x\Gamma(x)\text{.} \end{equation*}

If we replace \(x\) with an integer \(n\geq0\text{,}\) we get

\begin{equation*} \Gamma(n + 1) = n!\text{.} \end{equation*}

It turns out that we can replace \((\nu + k)!\) in (5.4.5) with \(\Gamma(\nu + k + 1)\text{,}\) giving

\begin{equation} J_{\nu}(x) = x^{\nu}\sum_{k\geq0}\frac{(-1)^{k}}{2^{2k + \nu}k!\Gamma(\nu + k + 1)}x^{2k} = \sum_{k\geq0} \frac{(-1)^{k}}{k!\Gamma(\nu + k + 1)}\left(\frac{x}{2}\right)^{2k + \nu}\text{.}\label{equation-bessel-first-kind-nonnegative-order}\tag{5.4.7} \end{equation}

Note that the asymptotic expansion in (5.4.6) holds for noninteger \(\nu\) as well.

Subsection 5.4.3 General Solution of Bessel's Equation

¶

Since (5.4.2) is second-order, we need a second linearly independent solution to get the general solution. If \(\nu\) is not an integer then we can find the second solution very quickly: \(J_{-\nu}(x)\text{.}\) However, if \(\nu\) is an integer then it turns out that \(J_{-\nu}(x) = (-1)^{n}J_{\nu}(x)\text{,}\) and so fails to be linearly independent from \(J_{\nu}\text{.}\)

It turns out that a second, linearly independent solution \(Y_{\nu}\) is given as follows:

\begin{align} Y_{\nu}(x) \amp = \frac{1}{\sin(\nu\pi)}[J_{\nu}(x)\cos(\nu\pi) - J_{-\nu}(x)]\label{equation-bessel-second-kind-noninteger}\tag{5.4.8}\\ Y_{n}(x) \amp = \lim_{\nu\to n}Y_{\nu}(x)\text{.}\label{equation-bessel-second-kind-integer}\tag{5.4.9} \end{align}
