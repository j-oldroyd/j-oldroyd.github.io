Section 4.4 Nonlinear Systems

Now we apply phase plane methods to study nonlinear autonomous systems,

Autonomous just means we can write the system without explicitly referring to the independent variable \(t\text{.}\)

which for systems involving two ODEs take the form

\begin{align*} y'_{1} & = f_{1}(y_{1},y_{2})\\ y'_{2} & = f_{2}(y_{1},y_{2})\text{.} \end{align*}

where \(y_{i} = y_{i}(t)\text{.}\)

We can also write such a system as a vector equation:

\begin{equation} \vec{y}' = \vec{f}(\vec{y})\text{.}\tag{7} \end{equation}

although not as a matrix ODE (if \(f_{i}\) are nonlinear).

Just as in the previous sections, the phase plane is still the \(y_{1}y_{2}-plane\text{,}\) trajectories are still the solutions \(\vec{y}\) of (7) (represented as curves in the phase plane), and the phase portrait of (7) is the set of all trajectories in the phase plane. We call a point \(P\) in the phase plane a critical point of (7) if it is a solution of the same system. Equivalently, the critical points of (7) are precisely those points \((y_{1},y_{2})\) for which

\begin{equation*} f_{1}(y_{1},y_{2}) = 0\quad\text{and}\quad f_{2}(y_{1},y_{2}) = 0. \end{equation*}

Example 4.4.1.

Express the pendulum equation \(\theta''+\frac{g}{l}\sin\theta=0\text{,}\) where \(\theta=\theta(t)\) represents the angular displacement of a pendulum from the vertical, as a nonlinear system \(\vecm{\theta}' = \vec{f}(\vecm{\theta})\) and then find its critical points.

Solution.

First, we have to rewrite the pendulum ODE as a first order system. We can do this without too much trouble as follows: set

\begin{equation*} \theta_{1} = \theta\quad\text{and}\quad\theta_{2} = \theta_{1}' = \theta'\text{.} \end{equation*}

Then the ODE \(\theta''+\frac{g}{l}\sin\theta = 0\) turns into the system

\begin{align*} \theta'_{1} & = \theta_{2}\\ \theta'_{2} & = -\frac{g}{l}\sin\theta_{1}\text{,} \end{align*}

which we can also write as \(\boldsymbol{\theta}' = \vec{f}(\boldsymbol{\theta})\) using

\begin{equation*} \vecm{\theta} = \begin{bmatrix}\theta_{1} \\ \theta_{2}\end{bmatrix}\quad\text{and}\quad \vec{f}(\vecm{\theta}) = \begin{bmatrix}\theta_{2} \\ -\frac{g}{l}\sin\theta_{2}\end{bmatrix}. \end{equation*}

Now we need to find the critical points \(\vecm{\theta}\) in the \(\theta_{1}\theta_{2}\)-plane that make \(\vec{f}(\vecm{\theta}) = \vec{0}\text{.}\) So we need \(\theta_{2} = 0\) and \(\theta_{1} = \pm k\pi\) for \(k=0,\pm1,\pm2,\dots\text{.}\) So the critical points of this system are all points of the form \((\pm k\pi,0)\text{.}\)

Subsection Classification of Critical Points and Linearization

Critical points of systems are important because they can represent long-term behavior of a system. For example, if we have a first-order system representing the population of two species, and it turns out the the origin is asymptotically stable, then this suggests that both species could be driven to extinction. So we want to classify critical points for nonlinear systems in addition to what we have already for linear systems; unfortunately, nonlinear systems are often difficult, if not outright impossible, to solve exactly.

Thankfully, in many cases we can approximate a nonlinear system \(\vec{y}' = \vec{f}(\vec{y})\) with critical points \(P_{i}\) by a suitably chosen linear system \(\vec{y}' = A\vec{y}\) at each critical point \(P_{i}\text{;}\) we call such a system the linearization at \(P_{i}\text{.}\)

Definition 4.4.2. The Jacobian of a Nonlinear System.

Let

\begin{equation*} \vec{y} = \begin{bmatrix}y_{1} \\ y_{2}\end{bmatrix}\quad\text{and}\quad\vec{f}(\vec{y}) = \begin{bmatrix}f_{1}(y_{1},y_{2}) \\ f_{2}(y_{1},y_{2})\end{bmatrix}. \end{equation*}

The Jacobian of \(\vec{f}\) is the matrix \(J(y_{1},y_{2})\) given by

\begin{equation*} J(y_{1},y_{2}) = \begin{bmatrix}\pdv{f_{1}}{y_{1}} & \pdv{f_{1}}{y_{2}} \\ \pdv{f_{2}}{y_{1}} & \pdv{f_{2}}{y_{2}}\end{bmatrix}. \end{equation*}

The linearization of \(\vec{y}' = \vec{f}(\vec{y})\) at the point \(P = (p_{1},p_{2})\) is the linear system \(\vec{y}' = A\vec{y}\text{,}\) where

\begin{equation*} A = J(p_{1},p_{2}). \end{equation*}

Example 4.4.3.

Find the linearization of the pendulum system \(\vecm{\theta}' = \vec{f}(\vecm{\theta})\) at the critical point \((0,0)\text{.}\)

Solution.

For this system, we have \(f_{1}(\theta_{1},\theta_{2}) = \theta_{2}\) and \(f_{2}(\theta_{1},\theta_{2}) = -\frac{g}{l}\sin\theta_{1}\text{.}\) The Jacobian is then given by

\begin{equation*} J(\theta_{1},\theta_{2}) = \begin{bmatrix}0 & 1 \\ -\frac{g}{l}\cos\theta_{1} & 0\end{bmatrix}. \end{equation*}

So to get the linearization we need to set

\begin{equation*} A = J(0,0) = \begin{bmatrix}0 & 1 \\ -\frac{g}{l} & 0\end{bmatrix}. \end{equation*}

The linearization of a nonlinear system isn't just useful for approximating the nonlinear system. It's also incredibly useful for classifying the critical points of a nonlinear system; for the most part, the eigenvalues of the matrix \(A\) from the linearization also classify the critical points of the system \(\vec{y}' = \vec{f}(\vec{y})\text{.}\)

Example 4.4.4.

Find and classify the critical points of the nonlinear system

\begin{align*} \dv{x}{t} & = x - y - x^{2} + xy \\ \dv{y}{t} & = -x^{2} - y \text{.} \end{align*}

This example taken from here. 1 

Solution.

The critical points occur at \(x' = y' = 0\text{.}\) If we set \(x' = y'\) and solve the resulting equation, we get \(x = 0\) or \(y = -1\text{.}\) If \(x = 0\text{,}\) then \(x' = -y\text{,}\) and for this to be \(0\) we must have \(y = 0\text{.}\) We can verify that \(x = y = 0\) makes \(y' = 0\) as well, so the origin is a critical point. Similarly, \(y = -1\) forces \(x = \pm1\text{,}\) and we can verify that \(x' = y' = 0\) at \((-1,-1)\) and \((1,-1)\text{.}\) So we have three critical points to check.

To determine the behavior of solutions at these critical points, we'll find the Jacobian at each point. First, we have

\begin{equation*} J(x,y) = \begin{bmatrix} 1 - 2x + y & x - 1 \\ -2x & -1\end{bmatrix}\text{.} \end{equation*}

At \((0,0)\text{,}\) we get

\begin{equation*} J(0,0) = \begin{bmatrix} 1 & -1 \\ 0 & -1\end{bmatrix}\text{.} \end{equation*}

The eigenvalues are \(1\) and \(-1\text{,}\) meaning that this critical point is a saddle point. At \((-1,-1)\) we get

\begin{equation*} J(-1,-1) = \begin{bmatrix}2 & -2 \\ 2 & -1\end{bmatrix} \end{equation*}

which has eigenvalues \(\lambda = \frac{1}{2}\pm\frac{1}{2}\sqrt{7}i\text{.}\) Hence \((-1,-1)\) should be a spiral point. Finally, at \((1,-1)\) we get

\begin{equation*} J(1,-1) = \begin{bmatrix} -2 & 0 \\ -2 & -1\end{bmatrix}\text{,} \end{equation*}

which has eigenvalues \(-2,-1\text{.}\) Hence \((1,-1)\) is an asymptotically stable node.

We can also use nonlinear systems to get more accurate population models.

Example 4.4.5.

Predator-prey populations can be modeled using the Lotka-Volterra model. Let \(y_{1}(t)\) denote the population of a prey species at time \(t\) and let \(y_{2}(t)\) denote the population of a predator species at time \(t\text{.}\) Then the Lotka-Volterra model says that

\begin{align*} y'_{1} & = ay_{1}-by_{1}y_{2}\\ y'_{2} & = ky_{1}y_{2} - ly_{2}\text{,} \end{align*}

where \(a,b,k,l>0\text{.}\) Find and classify the critical points of this system.

Solution.

The critical points are the points \((y_{1},y_{2})\) that satisfy the equations

\begin{equation*} ay_{1}-by_{1}y_{2} = 0\quad\text{and}\quad ky_{1}y_{2} - ly_{2} = 0. \end{equation*}

Equivalently, we need

\begin{equation*} y_{1}(a-by_{2}) = 0\quad\text{and}\quad y_{2}(ky_{1}-l) = 0. \end{equation*}

This has solutions \(y_{1} = y_{2} = 0\) and \(y_{1} = \frac{l}{k},y_{2} = \frac{a}{b}\text{.}\) So the critical points are \((0,0)\) and \((\frac{l}{k},\frac{a}{b})\text{.}\)

To classify these critical points we need to linearize the system, so we'll compute the Jacobian of

\begin{equation*} \vec{f}(\vec{y}) = \begin{bmatrix}ay_{1}-by_{1}y_{2} \\ ky_{1}y_{2} - ly_{2}\end{bmatrix} \end{equation*}

to get

\begin{equation*} J(y_{1},y_{2}) = \begin{bmatrix}a-by_{2} & -by_{1} \\ ky_{2} & ky_{1} - l\end{bmatrix}. \end{equation*}

At \((0,0)\text{,}\) we get

\begin{equation*} A = \begin{bmatrix}a & 0 \\ 0 & -l\end{bmatrix}\text{,} \end{equation*}

which has eigenvalues \(\lambda=a,-l\text{.}\) So the origin is an saddle point of the original system. In particular, there exist trajectories heading into the origin, so it's possible for both species to go extinct in this case.

Now we'll classify the second critical point \((\frac{l}{k},\frac{a}{b})\text{.}\) The Jacobian at this point gives us the matrix

\begin{equation*} A = J(\frac{l}{k},\frac{a}{b}) = \begin{bmatrix}0 & -\frac{bl}{k} \\ \frac{ak}{b} & 0\end{bmatrix}\text{.} \end{equation*}

This matrix has characteristic equation \(\lambda^{2} + al = 0\text{,}\) and so has eigenvalues \(\lambda=\pm i\sqrt{al}\text{.}\) Since the eigenvalues are pure imaginary, this suggests that \((\frac{l}{k},\frac{a}{b})\) is a center, which is indeed the case. In particular, trajectories near this critical point must be periodic.
