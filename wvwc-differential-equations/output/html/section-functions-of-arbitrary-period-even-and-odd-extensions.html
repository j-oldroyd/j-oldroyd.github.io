Section 7.2 Functions of Arbitrary Period; Even and Odd Extensions

Now that we know how to find Fourier series of periodic functions with period \(2\pi\text{,}\) we'd like to extend this idea to more general periodic functions, and then eventually to find Fourier series to represent (in some way) functions that aren't periodic.

Subsection Fourier Series of Functions of Arbitrary Period

We know how to find the Fourier series of a function of period \(2\pi\) by using Theorem 7.5. So we'd like to adapt this to functions that have period \(p = 2L\) instead of \(2\pi\text{.}\) This actually won't be too hard to do, since any function of period \(2L\) can be scaled into a function with period \(2\pi\text{.}\)

To see how, let \(f(x)\) denote a function of period \(2L\text{.}\) Then we want to find a constant \(c\) so that \(f(cx)\) has period \(2\pi\text{,}\) that is, so that

\begin{equation*} f(c(x+2\pi)) = f(cx). \end{equation*}

For this to be true, we need \(2\pi c = 2L\text{,}\) or in other words \(c = \frac{L}{\pi}\text{.}\) So if \(f(x)\) has period \(2L,\) then \(f(\frac{L}{\pi}x)\) has period \(2\pi\text{,}\) and furthermore \(f(\frac{L}{\pi}x)\) has Fourier coefficients given by

\begin{align*} a_{0} \amp= \frac{1}{2\pi}\int_{-\pi}^{\pi}f\left(\frac{L}{\pi}x\right)\,dx\\ a_{k} \amp= \frac{1}{\pi}\int_{-\pi}^{\pi}f\left(\frac{L}{\pi}x\right)\cos kx\,dx\text{ for $k\geq1$.}\\ b_{k} \amp= \frac{1}{\pi}\int_{-\pi}^{\pi}f\left(\frac{L}{\pi}x\right)\sin kx\,dx\text{ for $k\geq1$.} \end{align*}

Now we need to get everything back in terms of \(f(x)\text{,}\) the function we started with. We can do this by making the substitution \(u = \frac{L}{\pi}x\text{.}\) This gives us the following theorem after a little algebra.

Theorem 7.8. Fourier Series of Functions with Arbitrary Period.

Let \(f(x)\) be a function with period \(p = 2L\text{.}\) Then the Fourier coefficients of \(f(x)\) are given by

\begin{align*} a_{0} \amp= \frac{1}{2L}\int_{-L}^{L}f\left(x\right)\,dx\\ a_{k} \amp= \frac{1}{L}\int_{-L}^{L}f\left(x\right)\cos\left(\frac{k\pi}{L}x\right)\,dx\text{ for $k\geq1$}\\ b_{k} \amp= \frac{1}{L}\int_{-L}^{L}f\left(x\right)\sin\left(\frac{k\pi}{L}x\right)\,dx\text{ for $k\geq1$} \end{align*}

and the corresponding Fourier series is

\begin{equation*} a_{0}+\sum_{k=1}^{\infty}\left(a_{k}\cos\left(\frac{k\pi}{L}x\right)+b_{k}\sin\left(\frac{k\pi}{L}x\right)\right). \end{equation*}

Example 7.9. Fourier Series of \(f(x) = x^{2}\).

Let \(f(x) = x^{2}\) for \(-1\leq x\leq 1\) and have period \(p=2\text{.}\) We can find its Fourier series using Theorem 7.8. If we do so, we get

\begin{align*} a_{0} \amp= \frac{1}{3}\\ a_{k} \amp= \frac{4 \, \left(-1\right)^{k}}{\pi^{2} k^{2}}\\ b_{k} \amp= 0 \end{align*}

So the Fourier series of \(f(x)\) is given by

\begin{equation*} \frac{1}{3} + \sum_{k=1}^{\infty}\frac{4 \, \left(-1\right)^{k}}{\pi^{2} k^{2}}\cos\left(k\pi x\right)\text{.} \end{equation*}

Subsection Even and Odd Extensions

If you look at Examples Example 7.6 and Example 7.9, then you'll notice that a lot of the Fourier coefficients were \(0\text{.}\) In particular, \(a_{k} = 0\) for \(k\geq0\) for the first example, and \(b_{k}=0\) for the second. The reason for this has to do with even and odd functions.

Definition 7.10. Even and Odd Functions.

Let \(f(x)\) be a function. We say that \(f(x)\) is

\begin{align*} \text{even if } \amp f(-x) = f(x)\\ \text{odd if } \amp f(-x) = -f(x). \end{align*}

In other words, a function \(f(x)\) is even if and only if its graph is symmetric about the \(y\)-axis, and is odd if and only if its graph is symmetric about the origin.

In Example 7.6 and Example 7.9 we began with functions that typically aren't thought of as periodic and found their Fourier series. Essentially what we did was we restricted our view of each function to a limited interval 5  and then viewed that segment as defining a periodic function. This is the idea behind even and odd extensions of functions.

Definition 7.11. Even and Odd Extensions.

Let \(f(x)\) be a function defined on \([0,L]\text{.}\) The even extension of \(f(x)\) is the even periodic function defined by extending the graph of \(f(x)\) on \([0,L]\) to the rest of the real numbers in such a way that the resulting function is even and has period \(2L\text{.}\) The odd extension of \(f(x)\) is defined similarly.

Computing Fourier series for even and odd functions is simpler than the general case.

Theorem 7.12. Fourier Coefficients of Even and Odd Functions.

Let \(f(x)\) be periodic with period \(2L\text{.}\) If \(f\) is even, then the Fourier coefficients of \(f(x)\) satisfy

\begin{equation*} a_{0} = \frac{1}{L}\int_{0}^{L}f(x)\,dx, a_{k} = \frac{2}{L}\int_{0}^{L}f(x)\cos\left(\frac{k\pi}{L}x\right)\,dx\text{ and } b_{k} = 0 \end{equation*}

for \(k\geq1\text{.}\) If \(f\) is odd, then the Fourier coefficients of \(f(x)\) are

\begin{equation*} b_{k} = \frac{2}{L}\int_{0}^{L}f(x)\sin\left(\frac{k\pi}{L}x\right)\,dx\text{ and } a_{k} = 0. \end{equation*}

Example 7.13. Even Extension of a Piecewise Function.

Define the piecewise function \(f(x)\) by

\begin{equation*} f(x) = \begin{cases}x \amp 0\leq x\leq \frac{\pi}{2} \\ \frac{\pi}{2} \amp \frac{\pi}{2}\leq x\leq \pi\end{cases}. \end{equation*}

Then the even extension of \(f(x)\) is the new function \(f_{1}(x)\) given by

\begin{equation*} f_{1}(x) = \begin{cases}|x| \amp -\frac{\pi}{2}\leq x\leq \frac{\pi}{2} \\ \frac{\pi}{2} \amp \frac{\pi}{2}\leq |x|\leq \pi\end{cases}. \end{equation*}

We can use Theorem 7.12 to help us find the Fourier series for \(f_{1}(x)\text{.}\) With a little bit of help, we get \(a_{0} = \frac{3\pi}{8}\) and \(a_{k} = \frac{2\cos(\frac{\pi k}{2})-2}{\pi k^{2}}\text{,}\) and so the Fourier series of \(f_{1}(x)\) is

\begin{equation*} \frac{3\pi}{8}+\sum_{k=1}^{\infty}\frac{(-1)^{k}-1}{2\pi k^{2}}\cos(2kx). \end{equation*}
\([-\pi,\pi]\) in the case of \(x^{3}\) and \([-1,1]\) in the case of \(x^{2}\)
