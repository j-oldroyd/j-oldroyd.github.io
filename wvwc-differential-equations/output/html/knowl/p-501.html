For actually computing Legendre polynomials, instead of using (5.6) we often use Rodrigues’ formula
\begin{equation} P_{n}(x) = \frac{1}{2^{n}n!}\dv[n]{}{x}[(x^{2} - 1)^{n}]\tag{5.7} \end{equation}
or Bonnet’s recurrence
\begin{equation} (n + 1)P_{n + 1}(x) = (2n + 1)xP_{n}(x) - n P_{n-1}(x)\text{.}\tag{5.8} \end{equation}
