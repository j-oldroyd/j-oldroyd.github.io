Although it wasn't too hard to see that the functions in ExampleÂ 2.1.9 were linearly independent, in other cases it can be much trickier (especially when we move to higher order ODEs). For example, suppose we set \(f(t) = -5(t-15)^{2} + 450\) and \(g(t) = 12t(30-t)\text{.}\) Then it's not obvious at all that these two functions are linearly dependent! In fact,

\begin{equation*} \frac{2}{5}f(t) - \frac{1}{6}g(t) = 0. \end{equation*}

So generally when we try to determine if two functions are linearly independent, we'll make use of the Wronskian.
