Solution 3.2.2.1.

We need to find \(x_{c}\) first since \(x_{p}\) will change depending on \(x_{c}\text{.}\) Since the characteristic equation of the associated homogeneous ODE \(\dv[7]{x}{t}+8\dv[5]{x}{t}+16\dv[3]{x}{t}=0\) is

\begin{equation*} r^{7}+8r^{5}+16r^{3} = 0\quad\text{or}\quad r^{3}(r^{2}+4)^{2}=0, \end{equation*}

we get

\begin{equation*} x_{c} = c_{1}+c_{2}t+c_{3}t^{2}+A\cos2t+B\sin2t + t\brackets{C\cos2t+D\sin2t}. \end{equation*}

Now we come up with a guess for \(x_{p}\) using the right hand side of the original ODE and dividing it into ``components:''

However, we now have a problem with overlaps between \(x_{p}\) and \(x_{c}\text{.}\) The guess corresponding to the first component overlaps with \(x_{c}\text{,}\) so we need to multiply it by \(t^{3}\) to remove the overlap. Similarly, the guess corresponding to the third component overlaps, so we must multiply it by \(t^{2}\text{.}\) Therefore, our guess for \(x_{p}\) should be

\begin{equation*} x_{p} = C_{1}t^{5}+C_{2}t^{4}+C_{3}t^{3}+e^{-t}\brackets{C_{4}\cos2t+C_{5}\sin2t}+(C_{6}t^{3}+C_{7}t^{2})\cos2t+(C_{8}t^{3}+C_{9}t^{2})\sin2t. \end{equation*}
