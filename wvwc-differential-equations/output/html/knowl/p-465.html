We can use a similar approach to solve for \(c_{1}\) by plugging in \(x=0\text{,}\) but we need to get rid of the power of \(x\) attached to it. This is done by taking the derivative of \(f(x)\) and then setting \(x = 0\text{:}\)
\begin{align*} f^\prime(x) \amp= c_{1} + 2c_{2}x + 3c_{3}x^{2} + \dots\\ f^\prime(0) \amp= c_{1}\text{.} \end{align*}
The same trick works for \(c_{2}\text{:}\)
\begin{align*} f''(x) \amp= 2\cdot1c_{2} + 3\cdot2c_{3}x+\dots\\ f''(0) \amp= 2\cdot1c_{2} \end{align*}
so \(c_{2} = \frac{f''(0)}{2\cdot1}\text{.}\) Letâ€™s try this one more time to get \(c_{3}\text{:}\)
