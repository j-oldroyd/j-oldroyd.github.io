Example 6.1.10. Finding an Inverse Transform.

A continuous function \(f(t)\) has Laplace transform
\begin{equation*} F(s) = \frac{3s}{s^{2}+5} + \frac{7}{s^{3}}. \end{equation*}
Find \(f(t)\text{.}\)
Solution.
We can find \(f(t)\) by taking the inverse Laplace transform of each term in \(F(s)\text{:}\)
\begin{align*} \iLaplace{\frac{3s}{s^{2}+5}} &= 3\iLaplace{\frac{s}{s^{2}+5}} = 3\cos(\sqrt{5}t)\\ \iLaplace{\frac{7}{s^{3}}} &= \frac{7}{2!}\iLaplace{\frac{2!}{s^{3}}} = \frac{7}{2}t^{2}\text{.} \end{align*}
So
\begin{equation*} f(t) = 3\cos(\sqrt{5}t)+\frac{7}{2}t^{2}. \end{equation*}
