Now we find eigenvectors. So let
\begin{equation*} \vec{v} = \begin{bmatrix}v_{1} \\ v_{2}\end{bmatrix}. \end{equation*}
If \(\vec{v}\) is an eigenvector for \(\lambda\text{,}\) then we know \(A\vec{v} = \lambda\vec{v}\text{,}\) which gives the system
\begin{align*} (-\frac{1}{2}-\lambda)v_{1} & = 0\\ \frac{1}{2}v_{1} + (-\frac{5}{7}-\lambda)v_{2} & = 0 \end{align*}
If we set \(\lambda=-\frac{1}{2}\text{,}\) then we just get \(v_{1} = \frac{3}{14}v_{2}\text{.}\) So an eigenvector corresponding to \(\lambda_{1} = -\frac{1}{2}\) is
\begin{equation*} \vec{v}_{1} = \begin{bmatrix}3\\14\end{bmatrix}. \end{equation*}
