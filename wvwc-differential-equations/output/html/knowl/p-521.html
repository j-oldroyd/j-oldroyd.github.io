Consider the ODE
\begin{equation*} x^{2}y''+xp(x)y^\prime+q(x)y=0. \end{equation*}
Let \(r_{1}\geq r_{2}\) be (real) roots of the indicial equation \(r(r-1)+p(0)r+q(0)=0\text{.}\)
  1. There is a solution of the ODE of the form \(y_{1}(x) = x^{r_{1}}\sum_{n\geq0}^{}c_{n}x^{n}\text{.}\)
  2. If \(r_{1}-r_{2}\) is not equal to an integer, then there exists a second linearly independent solution of the form \(y_{2}(x) = x^{r_{2}}\sum_{n\geq0}^{}d_{n}x^{n}\text{.}\)
  3. If \(r_{1} = r_{2}\text{,}\) there exists a second linearly independent solution of the form \(y_{2}(x) = y_{1}(x)\ln x + x^{r_{1}}(C_{0} + C_{1}x + \cdots )\text{.}\)
  4. If \(r_{1} - r_{2}\) is a nonzero integer, there exists a second linearly independent solution of the form \(y_{2}(x) = ky_{1}(x)\ln x + x^{r_{2}}(C_{0} + C_{1}x + \cdots )\text{.}\)
