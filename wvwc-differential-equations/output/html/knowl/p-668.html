Let \(f(x)\) be some piecewise-differentiable function, not necessarily periodic. Then we can't find it's Fourier series. However, we can truncate the graph of \(f(x)\text{,}\) and replace it with a periodic function that is equal to \(f(x)\) on some interval \((-L,L)\text{.}\) Then we can find the Fourier series of this function, which by TheoremÂ 7.3.1 is given by \(\sum_{k=-\infty}^{\infty}c_{k}e^{i\frac{k\pi}{L}x}\) where

\begin{equation*} c_{k} = \frac{1}{2L}\int_{-L}^{L}f(x)e^{-i\frac{k\pi}{L}x}\,dx. \end{equation*}

So we can write

\begin{equation*} f(x) = \sum_{k=-\infty}^{\infty}\left(\frac{1}{2L}\int_{-L}^{L}f(x)e^{-i\frac{k\pi}{L}x}\,dx\right)e^{i\frac{k\pi}{L}x} \end{equation*}

wherever \(f(x)\) is continuous on \(-L \lt x \lt x\text{.}\)
