Let \(f(x)\) be a real function defined for all \(x\text{.}\) We say that \(f\) is a periodic function if there exists a positive number \(T\) such that

\begin{equation*} f(x+T) = f(x) \end{equation*}

for all \(x\text{.}\) In this case we say that \(f(x)\) is \(T\)-periodic. The (fundamental) period of \(f(x)\) is the smallest positive value of \(T\) for which \(f\) is \(T\)-periodic, assuming this value exists.
