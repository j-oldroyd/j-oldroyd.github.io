It must be noted that convergence of the power series of some function \(f(x)\) does not guarantee that \(f(x)\) can be represented by its power series. For instance, consider the function \(f(x)\) given by
\begin{equation*} f(x) = \begin{cases}e^{-x^{-2}} \amp\text{if }x\neq0 \\ 0 \amp\text{if }x=0\end{cases}\text{.} \end{equation*}
This has the power series representation \(0\) at \(x=0\) since \(f^{(k)}(0) = 0\) for all integers \(k\) with \(k\geq0\text{.}\) However, \(f(x)\neq0\) except at \(x=0\text{,}\) so this power series representation is not very useful for representing \(f(x)\) despite the fact that it converges for all \(x\text{.}\)
