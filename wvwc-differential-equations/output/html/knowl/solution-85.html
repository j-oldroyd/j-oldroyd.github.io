Solution 6.2.2.1.

By Theorem 6.2.1 we know that
\begin{equation*} \Laplace{f'(t)} = sF(s)-f(0). \end{equation*}
We can apply Theorem 6.2.1 to \(f^\prime(t)\) as well, since it must also be piecewise smooth:
\begin{align*} \Laplace{f''(t)} &= s\Laplace{f^\prime(t)} - f^\prime(0)\\ &= s(sF(s)-f(0)) - f^\prime(0)\\ &= s^{2}F(s) - sf(0) - f^\prime(0)\text{.} \end{align*}
This process can be continued indefinitely. In general,
\begin{equation*} \Laplace{f^{(n)}(t)} = s^{n}F(s) - s^{n-1}f(0) - s^{n-2}f^\prime(0) - \cdots -sf^{(n-2)}(0) - f^{(n-1)}(0). \end{equation*}
This gives us everything we need to start solving ODEs using Laplace transforms.
