Since this two-step recurrence relation is only valid for \(n\geq1\text{,}\) it places no restrictions on \(c_0\) or \(c_1\text{.}\) However, the original ODE was first-order! So we should only expect to have one arbitrary constant in our solution, which suggests that one of \(c_0\) or \(c_1\) must be zero. To determine which constant must vanish, we write out the first couple terms of the sums in (5.8) and compare terms:
\begin{equation*} c_{1}+2c_{2}x+\dots = -2c_{0}x -2c_{1}x^{2}+\cdots\text{.} \end{equation*}
