The first step is to set up and solve the indicial equation, which in this case is given by
\begin{equation*} r(r-1)+r-\frac{1}{4} = 0\text{.} \end{equation*}
We solve this algebraically for \(r\) to get the roots \(r_{1}=\frac{1}{2}\) and \(r_{2}=-\frac{1}{2}\text{.}\) Since \(r_{1}-r_{2} = 1\) is an integer, we are guaranteed a solution based on \(r_{1} = \frac{1}{2}\) and a second solution based on \(r_{2} = -\frac{1}{2}\) and the natural logarithm.
