Solution 4.3.8.1.

Let \(\vec{y} = \vec{y}(t)\) denote a nonzero solution of the matrix ODE (and therefore a trajectory). Then in order for \(\vec{0}\) to be asymptotically stable, we need \(\vec{y}\to\vec{0}\) as \(t\to\infty\text{.}\) Let \(\lambda_{1},\lambda_{2}\) denote the eigenvalues of \(A\text{.}\) Then \(\vec{y}\) will have the form
\begin{equation*} \vec{y} = c_{1}e^{\lambda_{1}t}\vec{y}_{1}+c_{2}e^{\lambda_{2}t}\vec{y}_{2}\text{.} \end{equation*}
The previous paragraph shows that \(\vec{y}\) must go to \(\vec{0}\) as \(t\to\infty\) if either \(c_{1}=c_{2}=0\) or if each exponential goes to \(0\) as \(t\to\infty\text{.}\) Since we assume \(\vec{y}\neq\vec{0}\text{,}\) this means we need \(e^{\lambda_{i}t}\to0\) for \(i=1,2\) as \(t\to\infty\text{.}\) This implies that the real part of each eigenvalue must be negative, because the real part of each eigenvalue is what determines the growth of \(e^{\lambda_{i}t}\text{:}\) if \(\lambda_{i} = a+bi\text{,}\) then
\begin{equation*} e^{\lambda_{i}t} = e^{at}[A\cos bt+B\sin bt]\text{.} \end{equation*}
So \(\vec{0}\) is asymptotically stable if the real parts of both eigenvalues are negative.
