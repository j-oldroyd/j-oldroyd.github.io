Example 4.2.6. Finding the eigenvalues of a triangular matrix.

Let
\begin{equation*} A = \begin{bmatrix}1 \amp 3 \amp -4 \amp 5 \\ 0 \amp 3 \amp -2 \amp -2 \\ 0 \amp 0 \amp 1 \amp 10^{50\pi-300} \\ 0 \amp 0 \amp 0 \amp 0\end{bmatrix}\text{.} \end{equation*}
Find the eigenvalues of \(A\text{.}\)
Solution.
\(A\) is a triangular matrix since everything below the main diagonal is \(0\text{.}\) Therefore the eigenvalues of \(A\) are \(1,3,1,0\text{.}\)
