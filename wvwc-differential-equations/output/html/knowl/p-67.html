To answer this question we need to find \(T\text{,}\) and although we don’t know \(k\) at the moment we can still make some progress just by remembering that it’s a constant. This ODE is separable, so we’ll separate variables and integrate both sides to get
\begin{equation*} \ln(T-15) = kt+C \end{equation*}
which simplifies to \(T-15 = C_{1}e^{kt}\) or just \(T = 15+C_{1}e^{kt}\text{.}\)
