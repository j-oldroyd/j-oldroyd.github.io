Now that we can take the Laplace transform of both sides of the modified IVP, we do so and obtain the Laplace transform \(\widetilde{Y}(s)\) of \(\widetilde{y}(t)\text{.}\) Using our initial conditions and a bit of algebra, we get
\begin{equation*} \widetilde{Y}(s) = \frac{-3s-11}{s^2-2s-3}. \end{equation*}
At this point we can take the inverse Laplace transform (either using partial fractions or using technology) to get
\begin{equation*} \widetilde{y}(t) = -5e^{3t} + 2e^{-t}. \end{equation*}
