Let \(\vec{y} = \vec{y}(t)\) denote a nonzero solution of the matrix ODE (and therefore a trajectory). Then in order for \(\vec{0}\) to be asymptotically stable, we need \(\vec{y}\to\vec{0}\) as \(t\to\infty\text{.}\) Let \(\lambda_{1},\lambda_{2}\) denote the eigenvalues of \(A\text{.}\) Then \(\vec{y}\) will have the form

\begin{equation*} \vec{y} = c_{1}e^{\lambda_{1}t}\vec{y}_{1}+c_{2}e^{\lambda_{2}t}\vec{y}_{2}. \end{equation*}
