Solution 5.1.4.1.

We can’t integrate \(e^{x^{2}}\) using elementary functions but this is straightforward to integrate using power series:
\begin{align*} \int_{0}^{1}e^{x^{2}}\dd{x} \amp= \int_{0}^{1}\sum_{n=0}^{\infty}\frac{x^{2n}}{n!}\dd{x}\\ \amp= \sum_{n=0}^{\infty}\int_{0}^{1}\frac{x^{2n}}{n!}\dd{x}\\ \amp= \sum_{n=0}^{\infty}\frac{1}{(2n+1)n!}\text{.} \end{align*}
We can also write the integral of \(e^{x^2}\) in terms of the error function 3 :
\begin{equation*} \int_0^1 e^{x^2}\dd{x} = -\frac{1}{2} i \, \sqrt{\pi} \operatorname{erf}\left(i\right)\text{.} \end{equation*}
