For \(\sin(4x)+\cos(6x)\) we have \(\frac{m}{n} = \frac{4}{6}\text{,}\) which in lowest terms is \(\frac{2}{3}\text{.}\) Therefore the period is \(T = \frac{2\pi}{\frac{4}{2}} = \pi\text{.}\) For \(\cos(5x) - 5\sin(3x)\text{,}\) we have \(\frac{m}{n} = \frac{5}{3}\) which is already in lowest terms. Therefore its period is \(\frac{2\pi}{\frac{5}{5}} = 2\pi\text{.}\)
