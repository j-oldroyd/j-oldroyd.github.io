Solution 6.1.7.1.

Note that we are computing a translation of the usual unit step function. We still compute \(\Laplace{u(t-3)}\) using the definition of the Laplace transform:

\begin{align*} \Laplace{u(t)} &= \int_{0}^{\infty}u(t)e^{-st}\,dt\\ &= \int_{3}^{\infty}e^{-st}\,dt &\text{ since }u(t-3) \text{ is }0 \text{ for values of }t\lt3\\ &= -\frac{1}{s}\left[e^{-st}\right]_{3}^{\infty}\\ &= \frac{e^{-3s}}{s}\text{.} \end{align*}
