Now we need to make sure that \(y\) is equal to \(1\) if \(x=0\text{.}\) We do this by setting \(y=1\text{,}\) \(x=0\) and choosing the right value for \(C\) to make the resulting equation true:
\begin{equation*} 1 = \frac{1}{2} + C \Rightarrow C = \frac{1}{2}. \end{equation*}
So the solution of this IVP is the function \(y = \frac{e^{-x^{2}}+1}{2}\text{.}\)
