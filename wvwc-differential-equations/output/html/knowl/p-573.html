Now we consider what happens to solutions given by (5.20) if \(\nu = n\) is a nonnegative integer. To simplify matters (somewhat...), we add the restriction that \(c_{0} = \frac{1}{2^{n}n!}\text{.}\) This allows us to write (5.20) more simply as
\begin{equation} c_{2k} = \frac{(-1)^{k}}{2^{2k + n}(n + k)!}\text{.}\tag{5.21} \end{equation}
