To find the solution, we return to the original equation and substitute \(y = \sum_{n=0}^{\infty}c_{n}x^{n}\) to get

\begin{equation*} \sum_{n\geq0}n(n-1)c_{n}x^{n-2} - \sum_{n\geq0}n(n-1)c_{n}x^{n} - \sum_{n\geq0}2nc_{n}x^{n} + \sum_{n\geq0}2c_{n}x^{n} = 0 \end{equation*}

which becomes

\begin{equation*} \sum_{n\geq0}(n + 2)(n+1)c_{n+2}x^{n} - \sum_{n\geq0}n(n-1)c_{n}x^{n} - \sum_{n\geq0}2nc_{n}x^{n} + \sum_{n\geq0}2c_{n}x^{n} = 0\text{.} \end{equation*}

After a little algebra, we get the recurrence relation

\begin{equation*} c_{n+2} = \frac{n-1}{n+1}c_{n}\quad\text{for}\quad n\geq2\text{.} \end{equation*}

This recurrence is valid for \(n\geq0\text{.}\)
