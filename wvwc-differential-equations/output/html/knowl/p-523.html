This property allows us to express any polynomial as a finite sum of Legendre polynomials in a computationally efficient manner. Furthermore, if we allow infinite series then we can use Legendre polynomials to express any continuous function defined on \([-1,1]\text{.}\) In particular, if \(f(x)\) is continuous on \([-1,1]\) then
\begin{equation*} f(x) = \sum_{n=0}^{\infty}c_nP_n(x) \end{equation*}
where
\begin{equation*} c_n = \frac{2n+1}{2}\int_{-1}^{1}f(x)P_{n}(x)\dd{x}\text{.} \end{equation*}
