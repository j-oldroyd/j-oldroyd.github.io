Now we can make a guess as to what \(y_{p}\) should be, once again based on the right hand side of the ODE. If we want to differentiate \(y_{p}\) and obtain \(x - \sin(2x)\text{,}\) then \(y_{p}\) should include both an \(x\) term and a \(\sin(2x)\) term. If we make the guess that \(y_{p} = A_{1}x + A_{2}\sin(2x)\text{,}\) then we get
\begin{align*} A_{1}x - 2A_{1} & = x \\ -4A_{2}\cos(2x) - 3A_{2}\sin(2x) & = -\sin(2x) \end{align*}
This forces \(A_{1} = 1\) and \(A_{1} = 0\text{,}\) as well as \(A_{2} = \frac{1}{3}\) and \(A_{2} = 0\text{.}\) Obviously, this is a problem!
