We could sketch the phase portrait for this system, but we can also determine the behavior of the trajectories \(\vec{y} = \begin{bmatrix} y_{1} \\ y_{2} \end{bmatrix} \) if we can find a relationship between \(y_{1}\) and \(y_{2}\text{.}\) To do so, we “cross-multiply” the system to get
\begin{equation*} 4y_{1}y^\prime_{1} = -4y_{2}y^\prime_{2} \quad\text{or}\quad 4y_{1}\dd{y_{1}} = -4y_{2}\dd{y_{2}}. \end{equation*}
So we can integrate this to get
\begin{equation*} 2y_{1}^{2} = -2y_{2}^{2}+C\qq{or}y_{1}^{2}+y_{2}^{2} = C_{1}. \end{equation*}
