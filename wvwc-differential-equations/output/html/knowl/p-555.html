Now we consider what happens to solutions given by (5.17) if \(\nu = n\) is a nonnegative integer. To simplify matters (somewhat...), we add the restriction that \(c_{0} = \frac{1}{2^{n}n!}\text{.}\) This allows us to write (5.17) more simply as
\begin{equation} c_{2k} = \frac{(-1)^{k}}{2^{2k + n}(n + k)!}.\tag{5.18} \end{equation}
The resulting series
\begin{equation} J_{n}(x) = x^{n}\sum_{k\geq0}\frac{(-1)^{k}}{2^{2k + n}k!(n + k)!}x^{2k}\tag{5.19} \end{equation}
is known as the Bessel function of the first kind of order \(n\text{.}\)
