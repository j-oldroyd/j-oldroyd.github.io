The question now is, how do we find the right values of the coefficients \(c_{k}\) to make (5.1) true? If we look at the equation, we can solve for \(c_{0}\) very easily: just set \(x=0\) to make all other terms disappear:
\begin{equation*} f(0) = c_{0} + c_{1}\cdot0+\dots = c_{0}. \end{equation*}
To solve for \(c_{1}\) by plugging in \(x=0\text{,}\) we need to get rid of the power of \(x\) attached to it. We can do this by taking the derivative of \(f(x)\text{:}\)
\begin{align*} f'(x) &= c_{1} + 2c_{2}x + 3c_{3}x^{2} + \dots\\ f'(0) &= c_{1}\text{.} \end{align*}
The same trick works for \(c_{2}\text{:}\)
\begin{align*} f''(x) &= 2\cdot1c_{2} + 3\cdot2c_{3}x+\dots\\ f''(0) &= 2\cdot1c_{2} \end{align*}
so \(c_{2} = \frac{f''(0)}{2\cdot1}\text{.}\) Letâ€™s try this one more time to get \(c_{3}\text{:}\)
