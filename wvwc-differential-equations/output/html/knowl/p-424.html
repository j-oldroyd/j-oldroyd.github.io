The critical points occur at \(x^\prime = y^\prime = 0\text{.}\) If we set \(x^\prime = y^\prime\) and solve the resulting equation, we get \(x = 0\) or \(y = -1\text{.}\) If \(x = 0\text{,}\) then \(x^\prime = -y\text{,}\) and for this to be \(0\) we must have \(y = 0\text{.}\) We can verify that \(x = y = 0\) makes \(y^\prime = 0\) as well, so the origin is a critical point. Similarly, \(y = -1\) forces \(x = \pm1\text{,}\) and we can verify that \(x^\prime = y^\prime = 0\) at \((-1,-1)\) and \((1,-1)\text{.}\) So we have three critical points to check.
