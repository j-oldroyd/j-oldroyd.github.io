We say that the origin is a stable critical point of \(\vec{y}^\prime=A\vec{y}\) if all trajectories that start ``close’’ to \(\vec{0}\) remain close at all future times. Equivalently, it’s stable if each trajectory will eventually be contained within some circle centered at the origin as \(t\to\infty\text{.}\) Otherwise, we say that \(\vec{0}\) is unstable. If it so happens that every trajectory that starts close to \(\vec{0}\) tends to \(\vec{0}\) as \(t\to\infty\text{,}\) we then say that \(\vec{0}\) is a stable and attractive (or asymptotically stable) critical point. Equivalently, \(\vec{0}\) is asymptotically stable if every trajectory goes to \(\vec{0}\) as \(t\to\infty\text{.}\)
