Just as we were able to plot direction fields for first-order autonomous ODEs, we can do something similar for autonomous first-order systems with two equations and constant coefficients. These are precisely the systems that can be written as a matrix ODE of the form
\begin{equation*} \vec{x}^\prime = A\vec{x} \end{equation*}
where \(A\) is a \(2\times2\) matrix.
