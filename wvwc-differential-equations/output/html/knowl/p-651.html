A very good question at this point is, what relationship does the Fourier series that we found in the previous example have with the original function \(f(x)\text{?}\) Are they actually equal? If we use the following code (adapted from here 2 ) to compare the partial sums
\begin{equation*} \sum_{k=1}^{n}\frac{2 \, {\left(6 \, \pi - \pi^{3} k^{2}\right)} \left(-1\right)^{k}}{\pi k^{3}}\sin kx \end{equation*}
of the Fourier series with \(f(x) = x^{3}\text{,}\) then it looks like the partial sums get closer and closer if we choose larger values of \(n\text{.}\)
