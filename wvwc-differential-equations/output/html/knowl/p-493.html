The power series method generally works as we utilized it in ExampleÂ 5.1.6. In general, the power series method to solve ODEs consists of the following procedure:
  1. Write \(y = \sum_{n=0}^{\infty}c_{n}x^{n}\text{.}\)
  2. Use the ODE to build a recurrence relation for the coefficients \(c_{n}\text{.}\)
  3. Find an explicit description of the coefficients.
  4. Identify \(y\) as the power series of some function.
