Example 6.5.4. Using the Convolution Theorem.

Let
\begin{equation*} F(s) = \frac{1}{s(s^{2}+4)}. \end{equation*}
Find \(f(t) = \iLaplace{F(s)}\text{.}\)
Solution.
We will use TheoremÂ 6.5.3 to express the inverse transform:
\begin{align*} f(t) &= \iLaplace{F(s)}\\ &= \iLaplace{\frac{1}{s}\cdot\frac{1}{s^{2}+4}}\\ &= 1\ast \frac{1}{2}\sin2t\\ &= \frac{1}{2}\int_{0}^{t}\sin2\tau\,d\tau\\ &= -\frac{1}{4}\left[\cos2\tau\right]_{\tau=0}^{\tau=t}\\ &= \frac{1-\cos2t}{4} \end{align*}
