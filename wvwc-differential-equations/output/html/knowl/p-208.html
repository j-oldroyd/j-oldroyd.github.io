We begin by solving the characteristic equation, which for this ODE is
\begin{equation*} r^{2} + 2r + 1 = 0. \end{equation*}
The only solution of this equation is \(r=-1\text{,}\) which is a repeated root. We can still get one solution of the ODE using this root, namely \(y_{1} = e^{-x}\text{,}\) but we need two linearly independent solutions in order to find the general solution.
