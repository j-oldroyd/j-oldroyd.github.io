Solution 3.1.6.1.

The characteristic equation is

\begin{equation*} r^{3}+4r^{2}+4r = 0\quad\text{or}\quad r(r+2)^{2}=0 \end{equation*}

so \(r=0,-2,-2\text{.}\) One solution of the ODE will be \(x_{1} = e^{0t} = 1\text{,}\) and a second solution will be \(x_{2} = e^{-2t}\text{.}\) But since \(r=-2\) is a repeated root, it does not provide a third linearly independent solution \(x_{3}\text{.}\) So we'll use the same trick we used before and multiply by \(t\) to get a third solution: \(x_{3} = te^{-2t}\text{.}\) It can be verified that \(x_{3}\) is in fact a solution of the ODE, and is also linearly independent from \(x_{1},x_{2}\text{.}\) Therefore the general solution of the ODE is

\begin{equation*} x = c_{1}+c_{2}e^{-2t}+c_{3}te^{-2t}. \end{equation*}
