Solution 1.6.4.1.

Let \(P(t)\) denote the population of the USA (in millions), where \(t\) is the number of years after 1990. Then
\begin{equation*} \dv{P}{t} = kP(M-P) \end{equation*}
and
\begin{equation*} P(t) = \frac{MP_{0}}{P_{0} + (M - P_{0})e^{-kMt}}. \end{equation*}
So we need to find \(k\) and \(M\text{.}\)
When \(t = 0\text{,}\) we have \(P^\prime = 3.1\) and \(P = 250\text{.}\) Similarly, when \(t = 22\) we have \(P^\prime = 2.3\) and \(P = 314\text{.}\) Therefore
\begin{align*} 3.1 & = 250k(M - 250) \\ 2.3 & = 314k(M - 314) \end{align*}
Solving this system gives us \(k\approx.00008\) and \(M \approx 406.4\text{.}\) Hence
\begin{equation*} P(t) = \frac{101600}{250 + 156.4e^{-.03t}}. \end{equation*}
This model estimates the population in 2017 to be
\begin{equation*} P(27) = 317.9, \end{equation*}
which is about a \(2\%\) error. Note also that this model predicts the carrying capacity of the USA to be \(406.4\text{.}\)
