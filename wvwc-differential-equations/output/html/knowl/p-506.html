To find the solution, we return to the original equation and substitute \(y = \sum_{n=0}^{\infty}c_{n}x^{n}\) to get
\begin{equation*} \sum_{n\geq0}n(n-1)c_{n}x^{n-2} + \sum_{n\geq0}\brackets{-n(n-1) - 2n + 2}c_{n}x^{n} = 0 \end{equation*}
which becomes
\begin{equation*} \sum_{n\geq0}\brackets{(n + 2)(n+1)c_{n+2} - n(n-1)c_{n} - 2nc_{n} + 2c_{n}}x^{n} = 0\text{.} \end{equation*}
After a little algebra, we get the recurrence relation
\begin{equation*} c_{n+2} = \frac{n-1}{n+1}c_{n}\text{.} \end{equation*}
This recurrence is valid for \(n\geq0\text{.}\)
