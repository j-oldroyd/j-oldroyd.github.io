Example 5.1.7. Solving a first-order ODE with variable coefficients.

Use power series to solve the ODE \(y^\prime+2xy=0\text{.}\)
Solution.
We will solve this using the steps listed above. First, assume \(y = \sum_{k=0}^{\infty}c_{k}x^{k}\text{.}\) Now plug this guess for \(y\) into the ODE to get
\begin{equation*} \sum_{k=1}^{\infty}kc_{k}x^{k-1} = -\sum_{k=0}^{\infty}2c_{k}x^{k+1}\text{.} \end{equation*}
As in Example 5.1.6 we want to equate coefficients to build a recurrence relation, so we need to rewrite these sums so that the same power of \(x\) appears on both sides. We do this by shifting the sums, but we need to remember to shift the limits of each sum as well:
So we get
\begin{equation} \sum_{n=0}^{\infty}(n+1)c_{n+1}x^{n} = \sum_{n=1}^{\infty}(-2)c_{n-1}x^{n}\text{.}\tag{5.8} \end{equation}
Hence a recurrence relation for \(c_{n}\) is
\begin{equation*} c_{n+1} = -\frac{2}{n+1}c_{n-1} \end{equation*}
which is valid for \(n\geq1\text{.}\)
Since this two-step recurrence relation is only valid for \(n\geq1\text{,}\) it places no restrictions on \(c_0\) or \(c_1\text{.}\) However, the original ODE was first-order! So we should only expect to have one arbitrary constant in our solution, which suggests that one of \(c_0\) or \(c_1\) must be zero. To determine which constant must vanish, we write out the first couple terms of the sums in (5.8) and equate coefficients:
\begin{equation*} c_{1}+2c_{2}x+\dots = -2c_{0}x -2c_{1}x^{2}+\cdots\text{.} \end{equation*}
This tells us that \(c_{1} = 0\text{.}\) Again, we can’t get this information from the recurrence relation!
Now we try to find an explicit formula for \(c_{n}\text{.}\) Because this is a two-step recurrence, we will write out the coefficients in two columns, one for odd \(n\) and one for even \(n\text{:}\)
\begin{align*} c_{2} \amp= -\frac{2}{2}c_{0} \amp c_{3} \amp= -\frac{2}{3}c_{1} = 0\\ c_{4} \amp= -\frac{2}{4}c_{2} = \frac{1}{2}\frac{1}{1}c_{0} \amp c_{5} \amp= -\frac{2}{5}c_{3} = 0\\ c_{6} \amp= -\frac{2}{6}c_{4} = -\frac{1}{3}\frac{1}{2}\frac{1}{1}c_{0} \amp c_{7} \amp= -\frac{2}{7}c_{5} = 0\text{.} \end{align*}
So it appears that
\begin{equation*} c_{2k} = \frac{(-1)^{k}}{k!}c_{0}\qq{and}c_{2k+1} = 0 \end{equation*}
for every \(k\text{.}\)
Now we plug this into our power series for \(y\) to get
\begin{align*} y \amp= \sum_{k=0}^{\infty}c_{k}x^{k}\\ \amp= \sum_{k=0}^{\infty}c_{2k}x^{2k}\\ \amp= \sum_{k=0}^{\infty}c_{0}\frac{(-1)^{k}}{k!}x^{2k}\\ \amp= c_{0}\sum_{k=0}^{\infty}\frac{(-x^{2})^{k}}{k!}\\ \amp= c_{0}e^{-x^{2}}\text{.} \end{align*}
