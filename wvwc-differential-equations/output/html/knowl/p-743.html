Now we move on to solving (8.5), but we still need to keep the condition \(k=-(\frac{n\pi}{L})^{2}\) for \(n=\pm1,\pm2,\ldots\text{.}\) If we do so, then (8.5) becomes \(T''+(c\frac{n\pi}{L})^{2}T=0\text{,}\) which has solutions given by
\begin{equation*} T_{n} = b_{n}\cos\lambda_{n}t+b^{*}_{n}\sin\lambda_{n}t, \end{equation*}
where \(\lambda_{n} = c\frac{n\pi}{L}\text{.}\)
