Example 6.4.5. Laplace Transform of the Dirac Delta Function.

Compute \(\Laplace{\delta(t-a)}\text{,}\) where \(a\geq0\text{.}\)
Solution.
We use the definition of the Laplace transform:
\begin{align*} \Laplace{\delta(t-a)} &= \int_{0}^{\infty}\delta(t-a)e^{-st}\,dt\\ &= e^{-as}\text{.} \end{align*}
In particular, \(\Laplace{\delta(t)} = 1\text{.}\)
