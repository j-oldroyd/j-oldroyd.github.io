Solution 6.4.7.1.

This time, the IVP we must solve is
\begin{equation*} x''+9x = 3\delta(t-4)\qq{where}x(0)=x^{\prime}(0)=0. \end{equation*}
If we take Laplace transforms and solve for \(X(s)\) we get
\begin{equation*} X(s) = \frac{3e^{-4s}}{s^{2}+9} \end{equation*}
and so
\begin{equation*} x(t) = u(t-4)\sin[3(t-4)]\text{.} \end{equation*}
