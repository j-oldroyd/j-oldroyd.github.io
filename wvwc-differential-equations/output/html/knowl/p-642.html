To see how, let \(f(x)\) denote a function of period \(2L\text{.}\) Then we want to find a constant \(c\) so that \(f(cx)\) has period \(2\pi\text{,}\) that is, so that
\begin{equation*} f(c(x+2\pi)) = f(cx). \end{equation*}
For this to be true, we need \(2\pi c = 2L\text{,}\) or in other words \(c = \frac{L}{\pi}\text{.}\) So if \(f(x)\) has period \(2L,\) then \(f(\frac{L}{\pi}x)\) has period \(2\pi\text{,}\) and furthermore \(f(\frac{L}{\pi}x)\) has Fourier coefficients given by
\begin{align*} a_{0} \amp= \frac{1}{2\pi}\int_{-\pi}^{\pi}f\left(\frac{L}{\pi}x\right)\,dx\\ a_{k} \amp= \frac{1}{\pi}\int_{-\pi}^{\pi}f\left(\frac{L}{\pi}x\right)\cos kx\,dx\text{ for $k\geq1$.}\\ b_{k} \amp= \frac{1}{\pi}\int_{-\pi}^{\pi}f\left(\frac{L}{\pi}x\right)\sin kx\,dx\text{ for $k\geq1$.} \end{align*}
