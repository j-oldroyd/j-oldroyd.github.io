Theorem 7.4.2. Square Error Formula.

Let \(f(x)\) be a function of period \(p = 2L\) with Fourier coefficients \(a_{k}\) and \(b_{k}\text{,}\) and let
\begin{equation*} F(x) = A_{0} + \sum_{k=1}^{N}[A_{k}\cos\frac{k\pi}{L}x + B_{k}\sin\frac{k\pi}{L}x] \end{equation*}
be a degree \(N\) trigonometric polynomial. Then
\begin{equation*} E \geq \int_{-L}^{L}f^{2}\,dx - L\left[2a_{0}^{2} + \sum_{k=1}^{N}(a_{k}^{2} + b_{k}^{2})\right]\text{.} \end{equation*}
The error \(E\) takes this minimum value if \(A_{k} = a_{k}, B_{k} = b_{k}\text{.}\)
