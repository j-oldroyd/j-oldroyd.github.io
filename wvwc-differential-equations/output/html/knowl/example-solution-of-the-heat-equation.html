Example 8.1.3. Solution of the heat equation.

Let \(u(x,t) = e^{-9t}\sin3x\text{.}\) Show that this is a solution of the boundary value problem
\begin{equation*} u_{t} = u_{xx},\quad u(0,t) = u(\pi,t) = 0\quad\text{and}\quad u(x,0) = \sin3x. \end{equation*}
Solution.
To do so, we need to compute the partial derivatives of \(u(x,t):\)
\begin{align*} u_{t}(x,t) \amp = \frac{\partial u}{\partial t} = -9e^{-9t}\sin3x \\ u_{x}(x,t) \amp = \frac{\partial u}{\partial x} = 3e^{-9t}\cos3x \\ u_{xx}(x,t) \amp = \frac{\partial}{\partial x}u_{x} = 9e^{-9t}\sin3x \text{.} \end{align*}
So we see that \(u_{t} = u_{xx}\text{,}\) which means that \(u(x,t) = e^{-9t}\sin3x\) is a solution of \(u_{t} = u_{xx}\text{.}\) Now it remains to show that \(u(x,t)\) satisfies the boundary conditions, which we can do without too much trouble.
