Example 8.3.4. Boundary value problem with sinusoidal deflection and initial velocity.

Consider a string of length \(L=1\text{,}\) initial deflection \(f(x) = \sin\pi x\) and initial velocity \(\frac{1}{4}\sin2\pi x\text{.}\) Assume that \(c = 2\text{.}\) Find \(u(x,t)\text{,}\) the vertical displacement at \((x,t)\text{.}\)
Solution.
The vertical displacement \(u(x,t)\) of the string is given by
\begin{align*} u(x,t) \amp = \frac{1}{2}[f^{*}(x-ct)+f^{*}(x+ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct}g(s)\,ds \\ \amp = \frac{1}{2}[\sin(\pi(x-2t))+\sin(\pi(x+2t))] + \frac{1}{4}\int_{x-2t}^{x+2t}\frac{1}{4}\sin2\pi s\,ds \\ \amp = \frac{1}{2}[\sin(\pi(x-2t))+\sin(\pi(x+2t))] - \frac{1}{32\pi}\cos2\pi s\Bigg]_{x-2t}^{x+2t} \\ \amp = \frac{1}{2}[\sin(\pi(x-2t))+\sin(\pi(x+2t))] - \frac{\cos(2\pi(x+2t)) - \cos(2\pi(x-2t))}{32\pi} \text{.} \end{align*}
