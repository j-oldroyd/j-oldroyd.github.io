We now have the tools we need to begin solving matrix ODEs. Recall that if \(A\) is an \(n\times n\) matrix with constant entries, and if \(\vec{x}_{1},\vec{x}_{2},\ldots,\vec{x}_{n}\) are \(n\) linearly independent solutions of the matrix ODE \(\vec{x}^\prime=A\vec{x}\text{,}\) then the general solution of the matrix ODE is
\begin{equation*} \vec{x} = \sum_{k=1}^{n}c_{k}\vec{x}_{k}\text{.} \end{equation*}
Furthermore, if \(\lambda\) is an eigenvalue of \(A\) with eigenvector \(\vec{v}\text{,}\) then \(e^{\lambda t}\vec{v}\) is a solution of \(\vec{x}^\prime=A\vec{x}\text{.}\) So solving the matrix ODE \(\vec{x}^\prime=A\vec{x}\) requires finding enough eigenvectors and eigenvalues.
