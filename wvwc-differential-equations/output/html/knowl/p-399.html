This will go to \(\vec{0}\) as \(t\to\infty\) if either \(c_{1}=c_{2}=0\) or if each exponential goes to \(0\) as \(t\to\infty\text{.}\) Since we assume \(\vec{y}\neq\vec{0}\text{,}\) this means we need \(e^{\lambda_{i}t}\to0\) for \(i=1,2\) as \(t\to\infty\text{.}\) This means that the real part of each eigenvalue must be negative. This is because the real part of each eigenvalue determines the growth of \(e^{\lambda_{i}t}\text{:}\) if \(\lambda_{i} = a+bi\text{,}\) then
\begin{equation*} e^{\lambda_{i}t} = e^{at}[A\cos bt+B\sin bt]\text{.} \end{equation*}
So \(\vec{0}\) is asymptotically stable if the real parts of both eigenvalues are negative.
