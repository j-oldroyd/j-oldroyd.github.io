We can now find our solution just as we did in Section 5.1, we assume the ODE has a series solution of the form \(y = \sum_{k=0}^{\infty}c_{k}x^{k}\) (which is justified by the above!). We want to return to the original form of the ODE to solve it; if we didn’t do so, we would need to expand \(P(x)\) and \(Q(x)\) using their own power series, and this would greatly complicate the algebra. So we will plug \(y = \sum_{k=0}^{\infty}c_kx^k\) into
\begin{equation*} (x^2+2)y'' + 4xy^\prime + 2y = 0 \end{equation*}
