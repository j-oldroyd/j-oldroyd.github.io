At this point we can make a simple but useful observation. In Example 1.5.8, the dimension of the null space was directly tied to the number of free variables in the system \(A\vb{x} = \vb{0}\text{.}\) The number of basic variables is likewise equal to the number of pivot columns of \(A\text{.}\) Noting that the number of basic variables plus the number of free variables must be the total number of columns of \(A\text{,}\) together with Theorem 1.4.3, we get the Rank-Nullity Theorem.
