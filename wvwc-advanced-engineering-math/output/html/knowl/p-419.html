Given a continuous vector field \(\FF = \mqty[P \amp Q]\) in \(\RR^2\text{,}\) we defined its circulation along a piecewise smooth curve \(C\) as
\begin{equation*} \int_C \FF\cdot\TT\dd{s} = \int_{C}\FF\cdot\dd{\rr}\text{.} \end{equation*}
If we write \(\rr = \mqty[x \amp y]\text{,}\) \(\dd{x} = x^\prime\dd{t}\) and \(\dd{y} = y^\prime\dd{t}\text{,}\) then this formula becomes
\begin{equation*} \int_C\FF\cdot\TT\dd{s} = \int_C P\dd{x} + Q\dd{y}\text{.} \end{equation*}
If \(C\) is also a closed curve, then we often replace \(\int_C\) with \(\oint_C\) to indicate such. Now we can state Green's Theorem.
