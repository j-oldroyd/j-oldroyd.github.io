Theorem 1.6.8. Row Operations and the Determinant.

Let \(A\) and \(B\) denote square matrices of the same size and suppose that \(B\) is obtained from \(A\) by performing a single row operation.
  1. If the row operation was row replacement, then \(\det(A) = \det(B)\text{.}\)
  2. If the row operation was row scaling by a factor of \(k\text{,}\) then \(\det(B) = k\det(A)\text{.}\)
  3. If the row operation was row interchange, then \(\det(B) = -\det(A)\text{.}\)
