Theorem 1.1.3. Properties of Matrix Addition and Scalar Multiplication.

Let \(A, B\) and \(C\) denote matrices of the same size and let \(r,s\in\RR\text{.}\) Then

  1. \(A+B = B+A\text{.}\)

  2. \((A+B)+C = A+(B+C)\text{.}\)

  3. \(A+\mathbf{0} = A\) where \(\mathbf{0}\) denotes the zero matrix.

  4. \(r(A+B) = rA + rB\text{.}\)

  5. \((r+s)A = rA + sA\text{.}\)

  6. \(r(sA) = (rs)A\text{.}\)

  7. \(1A = A\text{.}\)

  8. \(0A = \mathbf{0}\text{.}\)
