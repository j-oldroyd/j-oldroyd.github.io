Section 4.1 Line Integrals

An integral is fundamentally determined by two objects: the integrand and the domain of integration. In this section, we study integrals that involve one-dimensional domains of integration. Such integrals are known as line integrals. We will have a different type of line integral depending on whether our integrand is a scalar field or a vector field.

Subsection Scalar Line Integrals

Consider the problem of finding the mass of a wire that follows a path \(C\) in either \(\RR^2\) or \(\RR^3\) given a density function \(\rho\text{.}\) As seen in calculus, the mass can be found by integrating the density. This leads to integrals of the form \(\int_C \rho\dd{s}\) which are known as (scalar) line integrals.

Definition 4.1.1. Scalar Line Integrals.

Let \(C\) be a smooth curve in \(\RR^2\) (so \(C\) has no cusps, or equivalently, can be parameterized by a smooth vector function). Let \(f(x, y)\) denote a function that is continuous on some open set containing \(C\text{.}\) The scalar line integral of \(f\) over \(C\) is given by the limit

\begin{equation*} \int_C f(x, y)\dd{s} = \lim_{\Delta s\to0}\sum_{k=1}^{n}f(x_k, y_k)\Delta s \end{equation*}

where the points \((x_k, y_k)\) are lie on \(C\text{,}\) and \(\Delta s\) gives the length of the line segment from one such point to the next.

Geometrically, we view the line integral \(\int_C f(x, y)\dd{s}\) as giving the area between the curve \(C\) in the \(xy\)-plane and the surface \(z = f(x, y)\text{.}\) Although Definition 4.1.1 assumes that \(C\) lies in \(\RR^2\text{,}\) the definition can be extended to higher dimensions.

The definition given in Definition 4.1.1, while important, will not typically be used for computing line integrals. Instead, we will take another approach using arc length. The differential \(\dd{s}\) in the line integral formula corresponds to the arc length function along \(C\text{.}\) Therefore we may write

\begin{equation*} \dd{s} = \norm{\rrp(t)}\dd{t} \end{equation*}

and

\begin{equation} \int_C f(x, y)\dd{s} = \int_a^b f(\rr(t))\norm{\rrp(t)}\dd{t}\tag{1} \end{equation}

for some suitable smooth parameterization \(\rr(t), a\leq t\leq b\) of \(C\text{.}\)

Example 4.1.2. Computing a Scalar Line Integral.

Compute \(\int_C (xy)^{\frac{1}{3}}\dd{s}\) where \(C\) is the parabolic segment \(y = x^2\) from \(x = 0\) to \(x = 1\text{.}\)

Solution.

First, we need to parameterize \(C\text{.}\) This can be done using \(\rr(t) = \mqty[t \amp t^2]\) for \(0\leq t\leq 1\text{.}\) (1) then gives

\begin{align*} \int_C (xy)^{\frac{1}{3}}\dd{s} \amp = \int_0^1 t\sqrt{1 + 4t^2}\dd{t} \\ \amp = \int_1^5\frac{1}{8}\sqrt{u}\dd{u} \\ \amp = \frac{5^{\frac{3}{2}} - 1}{12} \text{.} \end{align*}

Recall that Definition 4.1.1 defines scalar line integrals over smooth curves only. If a curve \(C\) is piecewise smooth, which means that it can be written as the union of finitely many smooth curves \(C_1\) through \(C_n\text{,}\) then we can extend Definition 4.1.1 to \(C\) as well. All we need to do is compute the line integral along the smooth components \(C_i\) and add them up:

\begin{equation*} \int_C f(x, y)\dd{s} = \sum_{i=1}^n \int_{C_i} f(x, y)\dd{s}\text{.} \end{equation*}

Example 4.1.3. Computing a Line Integral Along a Piecewise Smooth Curve.

Let \(C\) denote the curve consisting of the line segment from \((-1, 0)\) to \((0, 1)\) followed by the line segment from \((0, 1)\) to \((1,0)\text{.}\) Find \(\int_C (2x - 3y)\dd{s}\text{.}\)

Solution.

We need to decompose \(C\) into smooth components. The natural choice is to pick \(C_1\) to be the first segment and \(C_2\) to be the second segment. These curves are smooth and have the following smooth parameterizations:

\begin{align*} C_1 : \rr_1(t) \amp = \mqty[t - 1 \amp t], 0\leq t\leq 1 \\ C_2 : \rr_2(t) \amp = \mqty[t \amp 1-t], 0\leq t\leq 1 \text{.} \end{align*}

Now we can compute the resulting line integrals:

\begin{align*} \int_{C_1} (2x - 3y)\dd{s} \amp = \int_0^1 [2(t-1) - 3t]\sqrt{2}\dd{t} \\ \amp = \int_0^1 (-2 - t)\sqrt{2}\dd{t} \\ \amp = -\frac{5}{2}\sqrt{2} \text{.} \end{align*}

Likewise,

\begin{equation*} \int_{C_1} (2x - 3y)\dd{s} = \int_0^1 (5t - 3)\sqrt{2}\dd{t} = -\frac{1}{2}\sqrt{2}\text{.} \end{equation*}

Adding these integrals together, we get

\begin{equation*} \int_C (2x - 3y)\dd{s} = -3\sqrt{2}\text{.} \end{equation*}
