To find an eigenvector, we just need to find a nontrivial solution of \((Q+2I)\vb{x} = \vb{0}\)---equivalently, a nonzero vector in \(\null(Q+2I)\)---so we'll continue row reducing:

\begin{align*} \mqty[-1 \amp 2 \amp -1 \amp 0 \\ 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0] \amp\rowop{-2R_{2}+R_{1}}\mqty[-1 \amp 0 \amp 1 \amp 0 \\ 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0]\\ \amp\sim\mqty[1 \amp 0 \amp -1 \amp 0 \\ 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0] \end{align*}

So

\begin{equation*} \null(Q+2I) = \span{\mqty[1 \\ 1 \\ 1]} \end{equation*}

and a single eigenvector of \(Q\) is given by

\begin{equation*} \mqty[1 \\ 1 \\ 1]. \end{equation*}
