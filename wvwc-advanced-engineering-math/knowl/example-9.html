Example 1.7.9.

The matrices

\begin{equation*} E_{1} = \mqty[2 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1],E_{2} = \mqty[1 \amp 5 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1]\text{ and } E_{3} = \mqty[0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0] \end{equation*}

are elementary matrices. The first corresponds to scaling the first row by \(2\text{;}\) the second corresponds to adding five times the second row to the first row; and the third corresponds to switching rows one and three.
