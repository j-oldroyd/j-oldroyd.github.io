Here's what this is saying: if \(\vb{v} = \mqty[\xmat*{v}{1}{3}]^{T}\) is an eigenvector of \(A\) corresponding to \(\lambda=\frac{1}{2}\text{,}\) or in other words a nontrivial solution of \((A-\frac{1}{2}I)\vb{v} = \vb{0}\text{,}\) then we must have \(v_{1} = v_{3} = 0\) and \(v_{2}\) free. So one nonzero eigenvector corresponding to \(\lambda=\frac{1}{2}\) is given by

\begin{equation*} \vb{v}_{1} = \mqty[0\\1\\0]. \end{equation*}

To find an eigenvector corresponding to \(\lambda = 3\text{,}\) we'll now row reduce \(\mqty[A - 3I \amp \vb{0}]\text{:}\)

\begin{equation*} \mqty[-1 \amp 0 \amp -1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 1 \amp 0 \amp 1 \amp 0]\sim\mqty[0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 1 \amp 0 \amp 1 \amp 0]. \end{equation*}

So if \(\vb{v}\) is an eigenvector corresponding to the eigenvalue \(\lambda=3\text{,}\) then we need \(v_{1} = -v_{3}\text{,}\) and \(v_{2}\) and \(v_{3}\) are both free. Which means that

\begin{equation*} \vb{v} = \mqty[\xmat*{v}{3}{1}] = \mqty[-v_{3} \\ v_{2} \\ v_{3}] = v_{2}\mqty[0\\1\\0]+v_{3}\mqty[-1\\0\\1]. \end{equation*}

In this case, we can actually find two linearly independent eigenvectors corresponding to \(\lambda=3\text{:}\)

\begin{equation*} \vb{v}_{2} = \mqty[0\\1\\0]\qq{and}\vb{v}_{3} = \mqty[-1\\0\\1]. \end{equation*}

Any (nonzero!) linear combination of these eigenvectors will provide another eigenvector corresponding to \(\lambda=3\text{,}\) but every other eigenvector must be a linear combination of \(\vb{v}_{2}\) and \(\vb{v}_{3}\text{.}\) In other words, \(\vb{v}_{2}\) and \(\vb{v}_{3}\) form a basis for the eigenspace corresponding to \(\lambda=3\text{.}\) The same is also true of \(\vb{v}_{1}\) and \(\lambda=\frac{1}{2}\) above.
